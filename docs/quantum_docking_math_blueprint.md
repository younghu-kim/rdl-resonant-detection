## 현재 보유 수학

```
구조군:     U(1) ≅ SO(2)
위상 공간:  φ ∈ S¹ (단위원)
게이지 ODE: dψ/dt = -(1/τ)ψ + v_t,  dφ/dt = ψ
판별식:     F₂ = Im{e^{-iφ}(L̂ - ψ̇)} = 0
양자화:     P(φ;L) = cos²(Lφ/2) → 0  (Maslov 보정)
검증:       ξ-함수 영점 60/60 탐지
```

## 부족한 수학 1: 다체 게이지 이론 — U(1) → SO(3)

### 문제

단백질은 3D 공간. 각 원자의 로컬 좌표계는 3D 회전으로 연결된다.
U(1)의 위상 φ ∈ S¹ 하나로는 3D 방향 정보를 담을 수 없다.

### 필요한 수학

**구조군 확장**:
```
현재:  G = U(1),  Lie 대수 g = iℝ  (1차원)
필요:  G = SO(3), Lie 대수 g = so(3) ≅ ℝ³  (3차원, 반대칭 행렬)
```

**주다발 (Principal Bundle)**:
```
P → M  (M = 분자 그래프, 구조군 G = SO(3))
각 노드 i에 프레임 e_i ∈ SO(3)
게이지 변환: e_i ↦ e_i · g_i,  g_i ∈ SO(3) (노드별 독립)
```

**연결 (Connection) = 평행이동**:
```
현재:  R(ρ_ij) = [[cos ρ, -sin ρ], [sin ρ, cos ρ]] ∈ SO(2)
필요:  R_ij ∈ SO(3),  3×3 회전 행렬
       R_ij = exp(ρ_ij × ·)  여기서 ρ_ij ∈ so(3) (축-각 표현)
```

**특성 벡터 확장** (Wigner-D 표현):
```
현재:  신호 f_i ∈ ℂ  (스칼라 + 위상)
필요:  f_i ∈ V = ⊕_{l=0}^{l_max} V_l,  V_l = ℝ^{2l+1}

       l=0: 스칼라 (1차원) — 전하, 소수성
       l=1: 벡터 (3차원) — 쌍극자 모멘트, 힘 방향
       l=2: 텐서 (5차원) — 분극률, 응력
```

**등변 메시지 패싱**:
```
m_{j→i} = Σ_l D^l(R_ij) · W_l · f_j^l

D^l(R) = Wigner-D 행렬 ∈ ℝ^{(2l+1)×(2l+1)}
W_l = 학습 가능 Schur 가중치 (l별 독립)
```

**게이지 ODE 확장** (SO(3) 위의 미분방정식):
```
현재:  dφ/dt = ψ           (S¹ 위의 스칼라 ODE)
필요:  dR/dt = Ω̂ · R       (SO(3) 위의 행렬 ODE)

       여기서 Ω̂ ∈ so(3)는 각속도의 반대칭 행렬:
       Ω̂ = [[0, -ω₃, ω₂], [ω₃, 0, -ω₁], [-ω₂, ω₁, 0]]

       Lie 대수 형식: dξ/dt = [Ω, ξ] + 감쇠항
       여기서 ξ ∈ so(3), [·,·] = Lie bracket
```

**PQO 확장** (SO(3) 위의 양자화):
```
현재:  P(φ;L) = cos²(Lφ/2),  φ = U(1)의 각도
필요:  P(R;L) = 1 - |χ_l(R^L)| / (2l+1)

       χ_l(R) = Tr(D^l(R)) = sin((l+½)θ) / sin(θ/2)  (SO(3)의 지표)
       θ = R의 회전각

       특수 경우 l=1:
       P(R;L) = 1 - |1 + 2cos(Lθ)| / 3

       영점: R^L = I (항등원) ⟺ Lθ = 2πn
       → U(1)의 sin²(Lφ/2) = 0 조건과 동형!
```

**F₂ 판별식 확장**:
```
현재:  F₂ = Im{e^{-iφ}(L̂ - ψ̇)} = 0
필요:  F₂^{so(3)} = ||Ad(R^{-1})(L̂ - Ω̇)||_{so(3)} = 0

       Ad(R): so(3) → so(3)  (수반 표현, adjoint representation)
       ||·||_{so(3)} = Killing form에 의한 노름
```

### 기존 수학 참조
- Clebsch-Gordan 계수 (표현론)
- e3nn 라이브러리 (Thomas, Smidt 2018)
- Gauge Equivariant Mesh CNN (de Haan et al. 2021)

## 부족한 수학 2: 다중 스케일 결합

### 문제

분자는 최소 4개 스케일이 동시에 작동한다:
```
전자   (~1Å)   — 양자역학, 궤도함수
원자   (~3Å)   — 힘의 장, 결합 각도
잔기   (~8Å)   — 2차 구조, 소수성 패치
도메인 (~30Å)  — 접힘, 포켓 형상
```

각 스케일의 대칭군과 동역학이 다르고, 스케일 간 정보 전달의 수학이 없다.

### 필요한 수학

**스케일 계층 (Renormalization Group 유사)**:
```
도메인의 탑:  Ω₁ ⊃ Ω₂ ⊃ Ω₃ ⊃ Ω₄
              (전자) (원자) (잔기) (도메인)

대칭군의 탑:  G₁ → G₂ → G₃ → G₄
              SO(3)  SO(3)  S_n   Iso(M)

조립화 사상:  C_k: X(Ω_k) → X(Ω_{k+1})  (= LocalPooling)
복원 사상:    L_k: X(Ω_{k+1}) → X(Ω_k)   (= Upsampling / Lifting)
```

**스케일 간 등변성 조건**:
```
C_k ∘ ρ_k(g) = ρ_{k+1}(π(g)) ∘ C_k

여기서 π: G_k → G_{k+1}은 대칭 축소 사상
예: 원자 SO(3) → 잔기 수준에서는 주쇄 방향의 SO(2)로 축소
```

**단열 분리 (Born-Oppenheimer 유사)**:
```
전자 위상 φ_e(t): 빠른 자유도 (fs 스케일)
구조 위상 φ_c(t): 느린 자유도 (ns 스케일)

단열 조건: τ_e ≪ τ_c

유효 위상: φ_eff(t) = φ_c(t) + ⟨φ_e⟩_{φ_c}
           (느린 구조가 고정된 상태에서 빠른 전자 위상의 평균)

게이지 ODE에서:
dψ_c/dt = -(1/τ_c)ψ_c + v_c + F_eff(φ_c)
F_eff(φ_c) = -∂⟨H_e⟩/∂φ_c   (유효 힘 = 전자 에너지의 구조 미분)
```

**다중 스케일 게이지 연결**:
```
전체 연결: A = A₁ + A₂ + A₃ + A₄

스케일 k에서의 유효 연결:
A_eff^{(k)} = A_k + Σ_{j≠k} ⟨A_j⟩_k  (다른 스케일은 평균장)

곡률 (= 힘):
F_k = dA_eff^{(k)} + A_eff^{(k)} ∧ A_eff^{(k)}
```

### 기존 수학 참조
- Renormalization Group (Wilson, 1971)
- Born-Oppenheimer 근사 (양자화학)
- Mori-Zwanzig 형식론 (비평형 통계역학)
- Multiscale GNN (Lim et al. 2022)

## 부족한 수학 3: 열역학 다리 — 위상 → 자유 에너지

### 문제

F₂ = 0은 "결합한다"를 알려주지만, "얼마나 세게" (ΔG, Kd)를 안 준다.
약물 설계에는 정량적 결합 자유 에너지가 필수.

### 필요한 수학

**정상 위상 근사 (Stationary Phase Approximation)**:
```
분배 함수: Z = ∫ exp(-βH(q)) dq

위상 공간에서:
Z = ∫ exp(-βH(φ)) · |det(∂²H/∂φ²)|^{-1/2} · δ(F₂(φ)) dφ

정상 위상 조건: F₂(φ*) = 0  (← RDL이 찾는 점!)

새들포인트 전개:
Z ≈ exp(-βH(φ*)) · (2π/β)^{n/2} · |det(∂²H/∂φ²)|^{-1/2}_{φ*}
```

**자유 에너지 다리 방정식**:
```
ΔG = -kT ln(Z_bound / Z_free)

RDL 언어로:
ΔG ≈ ΔH(φ*) + (kT/2) ln det(H_φφ) - TΔS_config

여기서:
ΔH(φ*) = H_bound(φ*_bound) - H_free(φ*_free)
          (F₂=0 점에서의 에너지 차이)

H_φφ = ∂²F₂/∂φ²|_{φ*}  (판별식의 Hessian)
        = 결합의 "강성" — 위상이 흔들리기 어려울수록 강한 결합

ΔS_config = 배치 엔트로피 (결합 시 자유도 감소)
```

**PQO와 진동 엔트로피**:
```
cos²(Lφ/2) ≈ (L²/4)(φ - φ*)² + O(φ⁴)  (극소 근방 전개)

PQO의 곡률: κ_PQO = L²/4

진동 주파수: ω_i² = κ_PQO · (∂²H/∂q_i²)

진동 엔트로피:
ΔS_vib = -k Σ_i ln(ℏω_i / kT)
        = -k Σ_i ln(L · √(∂²H/∂q_i²) / (2kT))

→ L 파라미터가 엔트로피 기여를 결정!
  큰 L (= 미세 양자화) → 더 정밀한 위상 잠금 → 더 낮은 진동 엔트로피
```

**결합 상수 예측**:
```
Kd = exp(ΔG / kT)

RDL에서:
Kd ≈ exp(ΔH(φ*)/kT) · ∏_i (ℏω_i/kT) · V_free/V_bound

여기서 V_free/V_bound = 배치 공간 부피 비
(결합 전 리간드의 자유 공간 / 결합 후 구속 공간)
```

### 기존 수학 참조
- 정상 위상 근사 (반고전 양자역학)
- MM-PBSA / MM-GBSA (분자역학 자유 에너지)
- 알케미컬 자유 에너지 섭동 (FEP)
- Zwanzig 공식: ΔG = -kT ln⟨exp(-βΔH)⟩₀

## 부족한 수학 4: 용매 효과

### 문제

물은 배경이 아니라 능동적 참여자.
소수성 효과 (엔트로피 구동)는 약물 결합의 ~50%를 설명.
현재 위상 수학에 용매가 없다.

### 필요한 수학

**용매 평균장 게이지 연결**:
```
진공에서의 연결:   A_vac(ij)
용매에서의 유효 연결: A_eff(ij) = A_vac(ij) · ε(r_ij)^{-1}

ε(r) = 위치 의존 유전 함수:
  ε = 1    (단백질 내부, 소수성)
  ε = 4    (단백질 표면)
  ε = 80   (벌크 물)
```

**Poisson-Boltzmann을 게이지 이론으로**:
```
고전: ∇·[ε(r)∇Φ(r)] = -4πρ(r) - 4πΣ c_i z_i exp(-z_i eΦ/kT)

게이지 재해석:
ε(r) = 메트릭의 변형 (용매가 "공간을 휘게" 함)
Φ(r) = U(1) 게이지 퍼텐셜

유효 평행이동 (용매 차폐):
R_eff(ρ_ij) = R_vac(ρ_ij) · exp(-κ · d_ij)

κ = Debye 차폐 길이의 역수 = √(8πe²I/εkT)
I = 이온 강도
d_ij = 노드 간 거리
```

**소수성 효과의 위상 기술**:
```
물 분자 = 작은 게이지 진동자 (수소결합 네트워크)
소수성 표면 근처: 물의 수소결합이 강제 정렬 → 엔트로피 감소

위상 언어:
⟨cos(φ_water - φ_neighbor)⟩ → 1  (소수성 표면에서 강제 동기화)

결합 시 소수성 표면 매몰:
ΔS_hydrophobic = k · Σ_{매몰된 물} ln(자유 배향 수 / 강제 배향 수)

GDL에서:
ΔS_hydrophobic ∝ ΔASA · σ

ΔASA = 용매 접근 가능 표면적 변화 (4G Geodesic 계산량)
σ = 표면 장력 계수 (~25 cal/mol/Å²)
```

**수소결합의 양자 보정**:
```
고전: 수소결합 = 정전기 + van der Waals
양자: 양성자 터널링이 기여 (특히 짧은 수소결합 ~2.5Å)

Maslov 보정 적용:
단거리 수소결합: cos² 체제 (양자 터널링 = 전환점 통과)
장거리 수소결합: sin² 체제 (고전적)

전환 기준: d_HB < d_critical ≈ 2.6Å이면 cos², 아니면 sin²
→ Maslov 지수 μ가 수소결합 거리의 함수!
```

### 기존 수학 참조
- Implicit solvent models (GBSA, PB)
- 3D-RISM (분자 용매 이론)
- Marcus theory (전자 전달, 용매 재배열)
- 양성자 터널링 (Klinman, Scrutton)

## 부족한 수학 5: 동역학 — 자유 에너지 풍경 위의 위상 역학

### 문제

단백질은 정적이지 않다. 포켓은 열린다/닫힌다.
일시적 포켓(cryptic site) 예측이 약물 발견의 성배.
현재 Time 도메인은 단순 게이트 RNN — 분자 동역학의 수학이 아님.

### 필요한 수학

**Langevin 방정식 on 게이지 다양체**:
```
현재 게이지 ODE:
dψ/dt = -(1/τ)ψ + v_t

필요 (열적 요동 추가):
dψ/dt = -(1/τ)ψ + v_t - ∂V(φ)/∂φ + √(2kT/τ) · η(t)

V(φ) = 자유 에너지 풍경 (위상 공간에서의 포텐셜)
η(t) = 가우시안 백색 잡음, ⟨η(t)η(t')⟩ = δ(t-t')

이것은 정확히 Kramers 문제:
- V(φ)의 극소 = 안정 구조 (접힌 상태, 닫힌 포켓)
- V(φ)의 안장점 = 전이 상태 (포켓 열리는 순간)
- 장벽 넘는 속도: k = (ω_0 ω_‡ / 2πγ) exp(-ΔV‡/kT)
```

**포켓 열림 = Kuramoto 탈동기화**:
```
N개 잔기의 결합 위상 진동자:
dφ_i/dt = ω_i + (K/N) Σ_j A_ij sin(φ_j - φ_i) + η_i(t)

A_ij = 접촉 행렬 (3G Graph의 인접 행렬)
K = 결합 강도
ω_i = 고유 진동수 (잔기별)

질서 매개변수:
r(t) = (1/N)|Σ_j exp(iφ_j(t))|

r ≈ 1: 포켓 닫힘 (동기화)
r 급감: 포켓 열림 (탈동기화)

임계 결합: K_c = 2 / (πg(0))
g(ω) = 고유 진동수 분포

→ K < K_c이면 포켓이 자발적으로 열린다!
→ 온도, 리간드 결합, pH 변화가 K를 바꿔서 포켓 전이를 유도
```

**알로스테릭 신호 전파 = 게이지파 (Gauge Wave)**:
```
부위 A에서 교란: δφ_A

전파 방정식 (감쇠파):
∂²(δφ)/∂t² + γ ∂(δφ)/∂t = c² Δ(δφ)

Δ = 그래프 라플라시안 (3G Graph)
c = 위상파 속도 (신호 전달 속도)
γ = 감쇠율 (구조적 마찰)

부위 B에 도달하는 신호:
δφ_B = G(A→B) · δφ_A

G(A→B) = 감쇠 게이지 전파자 (Green 함수):
G(A→B) = Σ_n (ψ_n(A) ψ_n(B)) / (ω_n² - ω² + iγω)

ψ_n = 라플라시안의 n번째 고유벡터
ω_n = n번째 고유진동수

→ 전파 효율 = |G(A→B)|
→ 알로스테릭 경로 = |G|가 최대인 노드 체인
```

**경로 적분으로 전이 확률**:
```
포켓이 시간 T 안에 열릴 확률:

P(open, T) = ∫ D[φ(t)] exp(-S[φ]/kT)

S[φ] = ∫₀ᵀ [(τ/2)|dφ/dt|² + V(φ)] dt  (Onsager-Machlup 작용)

이 경로 적분은 RDL의 훈련 루프와 구조적으로 동일:
- φ(t) = 게이지 ODE의 궤적
- S[φ] = 훈련 손실 함수의 시간 적분
- exp(-S/kT) = Boltzmann 가중치

→ RDL 훈련 = 자유 에너지 풍경의 최적 경로 탐색
```

### 기존 수학 참조
- Kramers rate theory (화학반응론)
- Kuramoto model (비선형 동역학)
- Markov State Models (MSM, 단백질 동역학)
- Transition Path Theory (E, Vanden-Eijnden 2006)
- Metadynamics (Laio, Parrinello 2002)

## 통합 방정식 — 양자 도킹 스코어 함수 (최종 목표)

모든 수학을 합치면 하나의 스코어 함수:

```
ΔG_quantum(ligand, receptor) =

    ΔH_phase(φ*)                          ← [수학 1] SO(3) 위상 정렬 에너지
  + (kT/2) ln det(H_φφ(φ*))              ← [수학 3] 진동 엔트로피
  - T · ΔS_config                         ← [수학 3] 배치 엔트로피
  + Σ_k λ_k · ΔE_scale^{(k)}             ← [수학 2] 다중 스케일 기여
  - T · σ · ΔASA                          ← [수학 4] 소수성/용매 효과
  + Σ_{HB} V_HB(d) · P_μ(φ_HB; L)       ← [수학 4] 수소결합 (Maslov 선택)
  - kT · ln ∫ exp(-V_pocket(φ)/kT) dφ    ← [수학 5] 포켓 유연성 보정

조건: F₂^{so(3)}(φ*) = 0  (SO(3) 위상 판별식 = 0에서 평가)
```

여기서:
- φ* = F₂ = 0을 만족하는 최적 위상 배열 (RDL이 찾는 점)
- H_φφ = 판별식의 Hessian (결합 강성)
- ΔASA = 4G Geodesic 표면적 변화
- P_μ = Maslov 선택적 PQO (거리에 따라 sin²/cos²)
- V_pocket = 포켓 자유 에너지 풍경 (5번 수학)

## 구현 순서 (현실적)

```
[보유]  U(1) 위상, F₂ 판별식, cos² Maslov
  ↓
[1단계] SO(3) 게이지 확장 — e3nn 참조, 기존 수학 조합
        검증: 작은 분자 (벤젠 이합체)의 배향 예측
  ↓
[2단계] 열역학 다리 — 정상 위상 근사로 ΔG 추정
        검증: PDBbind 데이터셋 (실험 Kd와 비교)
  ↓
[3단계] 용매 — GBSA 스타일 암묵 용매를 게이지에 통합
        검증: 소수성/친수성 리간드 ΔG 차이 재현
  ↓
[4단계] 다중 스케일 — 전자/원자/잔기 계층 결합
        검증: 알로스테릭 약물의 원거리 효과 예측
  ↓
[5단계] 동역학 — Langevin + Kuramoto 포켓 전이
        검증: cryptic site 예측 (CryptoSite 벤치마크)
  ↓
[최종]  양자 도킹 시스템 통합
```

각 단계는 독립 논문 1편 분량.
