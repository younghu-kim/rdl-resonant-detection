파트 A: 해석적 도출 — Hadamard 분해에서 A-gap 음상관

A.1: H₁의 NN 하한

  정의: H₁ = Σ_{k≠0} 1/(γ₀-γₖ)² + (Gamma 보정항)
  관찰: 모든 항 1/(γ₀-γₖ)² > 0 (양정치)
  따라서: H₁ ≥ 1/Δ_L² + 1/Δ_R²  (NN 두 항만으로 하한)
  여기서 Δ_L = γ₀ - γ_{-1}, Δ_R = γ₁ - γ₀

  g = gap_min = min(Δ_L, Δ_R) 이면:
  H₁ ≥ 1/Δ_L² + 1/Δ_R² ≥ 1/g² + 1/(Δ_L+Δ_R-g)²
  특히 H₁ ≥ 1/g²

  결론: g → 0 이면 H₁ → ∞. H₁은 gap_min의 역수 제곱에 하한.

A.2: S₁의 NN 지배

  S₁ = -1/Δ_L + 1/Δ_R + Σ_{|k|≥2} 1/(γ₀-γₖ) + Γ-terms
     = (Δ_L - Δ_R)/(Δ_L·Δ_R) + B(γ₀)
  여기서 B(γ₀) = 원거리 영점 기여 + digamma 보정 (smooth background)

  NN 지배 조건: |1/Δ_L - 1/Δ_R| >> |B|
  이 조건은 min(Δ_L, Δ_R) << mean spacing 일 때 성립.
  (밀집 영점 쌍에서 자동 만족)

  S₁² ≈ (Δ_L - Δ_R)²/(Δ_L·Δ_R)²
  특수 경우 Δ_L = Δ_R = g: S₁ → B(γ₀) (상쇄!)
  일반: |Δ_L - Δ_R| > 0 이면 S₁² > 0

A.3: A-gap 음상관 증명 (Prop 12)

  A(γ₀) = S₁² + 2H₁ ≥ 2H₁ ≥ 2/g²

  정리 (Prop 12, pair correlation conjecture 가정):
  gap_min(γ₀) = g 일 때, E[A | gap_min = g] 는 g에 대해 단조 감소.

  증명 (골격):
  (i)  A ≥ 2/g² → 하한이 g 감소에 따라 급증 (∝ g⁻²)
  (ii) E[H₁ | gap = g] 분해:
       H₁ = H₁^{NN} + H₁^{tail}
       H₁^{NN} = 1/Δ_L² + 1/Δ_R² ≥ 2/g²  (결정적)
       H₁^{tail} = Σ_{|k|≥2} 1/(γ₀-γₖ)² + Γ'-terms
       R₂(x) = 1-(sin πx/πx)² 가정하에:
       E[H₁^{tail}] = ∫₋∞^∞ R₂(x)/x² dx - (NN 제외) + Gamma 보정
       이 값은 g에 약의존 (GUE repulsion으로 두 번째 인접도 반발)

  (iii) 따라서 E[A|gap=g] ≈ E[S₁²|g] + 2·E[H₁^{NN}|g] + 2·E[H₁^{tail}]
        ≥ 2/g² + (g-독립 상수)
        → g 감소 시 단조 증가

  (iv) 부호: ρ(A, gap_min) < 0  ⬜ QED (부호)

A.4: E[H₁^{tail}]의 pair correlation 적분

  GUE pair correlation: R₂(x) = 1 - (sin πx / πx)²
  E[H₁^{tail}] = ∫₁^∞ R₂(x)/x² dx + ∫₋∞^{-1} R₂(x)/x² dx
               = 2 ∫₁^∞ [1 - (sin πx / πx)²] / x² dx

A.5: ρ ≈ -0.57 보편성 분석

  GUE 가설: ρ(A, gap_min)의 값은 GUE spacing 분포에서 결정.
  A ~ 2/g² + const, g ~ Wigner surmise
  ρ(2/g², g) = ρ of (f(g), g) where f = 2/g² is strictly decreasing
  → ρ < 0 (명백)

  Wigner surmise에서 ρ(1/s², s) = -1.0000  (이론 하한)
  참고: 엄밀히 ρ(1/s², s) = -1 (단조 감소 변환)
  실측 ρ ≈ -0.57은 noise 기여 반영:
  A = H₁^{NN} + (H₁^{tail} + S₁²/2) = 결정적 + 확률적
  noise가 signal의 ~50% → |ρ| ≈ 0.57

  결론: ρ의 부호(음)는 H₁ ≥ 2/g²에서 결정적으로 보장.
  |ρ| ≈ 0.57의 보편성은 H₁^{{tail}}/H₁^{{NN}} 비율이
  GUE pair correlation에 의해 보편적으로 정해지기 때문.
