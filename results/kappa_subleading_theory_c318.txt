[6] 해석적 도출 요약
===========================================================================

  ◆ [1단계] G(s) 기울기 — Stirling 유도
    G(s) = 1/s + 1/(s-1) - logπ/2 + ψ(s/2)/2
    Stirling: ψ(z) = log(z) - 1/(2z) - 1/(12z²) + ...
    ψ(s/2) ≈ log(s/2) - 1/s  (|s|→∞)
    Re(G) ≈ (1/2)log(γ/(2π)) + O(δ²/γ², 1/γ)
    → a_G = 1/2 (해석적)
    → 수치: a_G = 0.499997 (오차 0.000003)

  ◆ [2단계] Hadamard 상쇄 메커니즘
    ξ'/ξ = Σ_ρ [1/(s-ρ) + 1/ρ]  (Hadamard 곱)
    R = ξ'/ξ - 1/δ = Σ_{ρ≠ρ_n} [1/(s-ρ) + 1/ρ] + 1/ρ_n
    → G(s) 항이 완전히 상쇄!
    → Re(R) = C₀ + Σ_{k≠n} Re[1/(s-ρ_k)] + mirror
    → Re(R)의 기울기 a_R는 G(s)에 독립
    → 동시에 Re(S_rest) = Re(R) - Re(G) → a_S = a_R - a_G ≈ a_R - 0.5

  ◆ [3단계] 잔여 기울기 a_R의 기원
    Σ_{k≠n} δ/(δ²+(γ_n-γ_k)²) → Poisson kernel × 영점 분포
    d̄(t) = log(t/(2π))/(2π) 증가 → 유효 이웃 수 증가 → 합 증가
    GUE R₂(v) = 1 - sinc²(πv) → 수준 반발 → 소간격 억제
    수치: a_R = 0.036771
    GUE 적분 예측 기울기: 0.038660

  ◆ [4단계] Δκ 기울기 합성
    Δκ = 2Re(R)/δ + |R|²
    a_Δκ = 2·a_R/δ + a_{|R|²}
         = 2×0.036771/0.03 + 0.568141
         = 3.0195
    측정: 3.0195
    기존(N=20): 1.5945
