[Step 5] 이론 예측: κ_near = 1/δ² + C(t₀)/δ + O(1)
========================================================================

  이론: ξ(s) 근방에서 conn = ξ'/ξ ≈ 1/(s-s₀) + 2γ'(s₀)/γ(s₀) + L''(s₀)/L'(s₀)
        s = s₀ + δ (δ 실수 오프셋)
        → conn ≈ 1/δ + (2 Re[γ'/γ + L''/L'])
        → κ = |conn|² ≈ 1/δ² + 2C/δ + C²

  C 추정 (데이터 기반):
    δ=0.01: C_est = 0.0121
    δ=0.02: C_est = 0.0241
    δ=0.03: C_est = 0.0362
    δ=0.05: C_est = 0.0603
    δ=0.10: C_est = 0.1206

  참고: ζ(s)의 감마 인자 γ(s) = π^{-s/2}Γ(s/2)
        γ'/γ(s) = -log(π)/2 + (1/2)ψ(s/2)  (ψ = digamma)

  γ'/γ(s₀) 실부분 at t₀=14.134725: 0.4053
  → C 이론 예측: 2 × γ'/γ(s₀) = 0.8105
     (L''/L' 항도 있어 정확한 C와 차이 발생 가능)
