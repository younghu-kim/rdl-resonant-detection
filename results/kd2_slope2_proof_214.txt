THEOREM (κδ² Slope Universality)
========================================================================

Statement:
  Let Λ(s) be a completed L-function satisfying:
    (FE)  Λ(s) = ε·Λ(1-s)    (functional equation)
    (RC)  Λ(s̄) = Λ̄(s)        (reality condition, self-dual)
  Let ρ = ½+iγ be a simple zero on the critical line.
  Define κ(δ) := |Λ'/Λ(½+δ+iγ)|² for real δ>0.
  Then κ(δ)·δ² = 1 + A·δ² + O(δ³), so
    log(κδ²-1) = log A + 2·log δ + O(δ)
  and the log-log slope is exactly 2.

Proof:
  Laurent expansion at ρ: Λ'/Λ(s) = 1/(s-ρ) + c₀ + c₁(s-ρ) + ...
  At s = ½+δ+iγ (so s-ρ = δ ∈ ℝ):
    Λ'/Λ = 1/δ + c₀ + c₁δ + ...                    (★)

  From (FE): Λ'/Λ(s) = -Λ'/Λ(1-s)
  At 1-s = ½-δ-iγ = conj(½-δ+iγ):
  From (RC): Λ'/Λ(conj(z)) = conj(Λ'/Λ(z))
    So Λ'/Λ(½-δ-iγ) = conj(Λ'/Λ(½-δ+iγ))         (†)

  Laurent at ½-δ+iγ (i.e. s-ρ = -δ):
    Λ'/Λ(½-δ+iγ) = 1/(-δ) + c₀ + c₁(-δ) + ...
                   = -1/δ + c₀ - c₁δ + ...          (‡)

  Combining (FE) at s = ½+δ+iγ:
    Λ'/Λ(½+δ+iγ) = -Λ'/Λ(½-δ-iγ) = -conj(Λ'/Λ(½-δ+iγ))  [by (†)]
    From (★): 1/δ + c₀ + c₁δ = -conj(-1/δ + c₀ - c₁δ)
                               = 1/δ - c̄₀ + c̄₁δ

  Equating coefficients:
    δ⁰: c₀ = -c̄₀  ⟹  Re(c₀) = 0   (c₀ purely imaginary)  ■
    δ¹: c₁ = c̄₁   ⟹  Im(c₁) = 0   (c₁ real)              ■

  Therefore:
    κδ² = |1 + c₀δ + c₁δ²|² = 1 + 2Re(c₀)δ + (|c₀|²+2Re(c₁))δ² + O(δ³)
         = 1 + 0 + A·δ²      (since Re(c₀)=0)
    where A = Im(c₀)² + 2c₁.                              □

Remark: A depends on the local zero density (via c₀) and
  Γ-factor structure (via digamma contributions to c₁).
  This explains the observed A(t₀) ∝ d² + log N variation.
