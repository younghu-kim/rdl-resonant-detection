  [Phase 5] S1 내부의 공유 g 기여 분석
======================================================================

  S1(n) = (-1/g + 1/δ_L) + Σ_{j≥2} tail terms
  S1(n+1) = (-1/δ_R + 1/g) + Σ_{j≥2} tail terms

  → S1(n)에서 g의 직접 기여: -1/g
  → S1(n+1)에서 g의 직접 기여: +1/g

  S1(n)² = (1/g)² - 2(1/g)·(1/δ_L + tail) + (1/δ_L + tail)²
  → Cov(1/g², S1(n)²) = Cov(1/g², (1/g)²) + 교차항
  → Cov(1/g², (1/g)²) = Var(1/g²) 의 비선형 버전 → 거의 확실 양

  핵심 구조:
    S1(n)² ⊃ 1/g² → S1²와 1/g²가 공통 인수를 공유
    → Cov(1/g², S1²) > 0 은 1/g² 자기상관의 귀결
