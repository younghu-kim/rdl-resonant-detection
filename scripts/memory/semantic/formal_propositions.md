# ξ-다발 프레임워크: 공식 명제 레지스트리
# 최종 갱신: 2026-04-13

---

## Definitions

### Definition 1 (ξ-다발)
임계대 S = {s = σ+it ∈ ℂ : 0 < σ < 1} 위의 주 U(1)-다발.
- 단면: ψ(s) = ξ(s)/|ξ(s)| = e^{iθ(s)} (Z = {ξ=0} 밖에서)
- 접속: A(s) = L(s)ds = (ξ'/ξ)(s)ds
- 곡률 밀도: κ(s) = |L(s)|²

### Definition 2 (모노드로미)
경로 γ: [0,1] → S\Z에 대해 μ(γ) = ∮_γ Im(L(s))ds = Δarg(ξ).

### Definition 3 (횡방향 곡률)
임계선 위에서 F₂ ≈ v_σ · cos(φ), v_σ = Im(L)|_{σ=1/2}.

## Theorems (해석적 증명)

### Theorem 1 (접속 반대칭)
L(1-s) = -L(s). 임계선 위에서 Re(L) = 0 (유니터리 게이지).
**증명**: ξ(s)=ξ(1-s) 미분 → ξ'(s)=-ξ'(1-s) → L(1-s)=-L(s). σ=1/2에서 s와 1-s̄의 관계로 Re(L)=0.
**검증**: |Re(L)|/|Im(L)| < 10⁻³⁵, 20개 비영점 + 6개 근방점. `bundle_gauge_symmetry.txt`
**비평가**: 생존.

### Theorem 2 (위상적 영점 세기)
폐곡선 C가 영역 R을 둘러쌀 때, N(R) = (1/2π) ∮_C d(arg ξ).
**증명**: 논증 원리(Cauchy)의 다발 기하학적 표현.
**검증**: N=1,3,5 정확 일치, N(50)=10. `bundle_chern_number.txt`
**비평가**: 생존. 단, 기존 수학의 재서술임을 명시.

### Theorem 3 (곡률 소멸 — 유니터리 게이지 귀결)
임계선 위 일반점(cos φ ≠ 0)에서 L = 0. 영점에서만 L → ∞.
**증명**: Thm 1에서 Re(L)=0. Klein 대칭 F₂=F₂^R 요구 → 2·Im(L)·cos(φ)=0 → Im(L)=0.
**검증**: 비영점 대비 영점에서 |L| 34배 집중. `bundle_curvature_2d.txt`
**비평가**: 생존. **[NEW]** 곡률 집중이 대칭에서 강제된다는 메커니즘은 새로운 기여.

## Propositions (수치 검증)

### Proposition 1 (Klein 4-군 게이지 대칭)
ξ-다발은 G = Z₂×Z₂ = {Id, R₁:s→1-s, R₂:s→s̄, R₃:s→1-s̄} 대칭을 가진다.
Fix(R₃) = {σ=1/2} (임계선).
**검증**: |ξ(s)-ξ(R_k(s))|/|ξ(s)| < 10⁻⁵¹, 3개 시험점. `bundle_gauge_symmetry.txt`
**비평가**: 생존. 함수방정식+켤레대칭의 재서술.

### Proposition 2 (모노드로미 양자화 = π)
ξ의 단순 영점 t_k에서 임계선 방향 모노드로미 = ±π.
전체 평면 감김수 = 1 (2π). 임계선 기여 = 정확히 1/2.
**검증**: 5/5 영점, ε→0 수렴, 오차 < 10⁻⁴⁰. `bundle_geometry_verification.txt`
**비평가**: 생존. 반-뒤틀림(half-twist) 해석은 **[NEW]**.

### Proposition 3 (cos² = k²=-1 격자) **[NEW]**
cos²(φ) 영점 = {π/2, 3π/2} = arg(k²=-1의 근 {i,-i}).
sin²(φ) 영점 = {0, π} = arg(k²=+1의 근 {1,-1}).
모노드로미 π가 k²=-1 격자를 따르므로 cos² PQO만 정합.
**검증**: 정확 일치. `bundle_geometry_verification.txt`
**비평가**: 생존. 기존 문헌에 없는 새 설명.

### Proposition 4 (PQO 호환성 정리, L≤2) **[NEW]**
cos²(Lθ/2)의 영점 간격 = 2π/L. 영점 간 θ 증가 ≈ π.
구간당 PQO 영점 수 ≈ L/2. 유령 영점 수 = L/2-1.
L=2: 0 유령 (호환). L≥4: 유령 ≥1 (비호환).
Nyquist 해석: 영점 신호 주기 π, 검출기 간격 ≥ π 필요 → L ≤ 2.
**검증**: 9개 연속 영점 쌍에서 유령 수 확인. `bundle_L4_explanation.txt`
**비평가**: 생존. 실험적 L=4 실패의 최초 수학적 설명.

### Proposition 5 (F₂ = 횡방향 곡률) **[NEW]**
수렴 시 F₂ ≈ v_σ·cos(φ) = Im(ξ'/ξ)|_{σ=1/2} · cos(arg ξ).
F₂는 "임계선에서 벗어나면 위상이 얼마나 변하는가"를 측정.
**검증**: 해석적 도출 + 영점에서 v_σ 발산 확인. `bundle_gauge_symmetry.txt`
**비평가**: 생존. 판별식의 기하학적 의미 최초 부여.

## Conjectures (증거 있으나 증명 미완)

### Conjecture 1 (RH의 다발 기하학적 재서술)
다음은 동치이다:
(i) Z ⊂ Fix(R₃) = {σ=1/2} (리만가설)
(ii) Klein 4-군의 영점 궤도가 크기 4를 갖지 않음
(iii) 임계선 밖 감김수 = 0
(iv) κ(σ,t) < ∞ for all σ ≠ 1/2
**검증**: (iii) σ∈[0.1,0.49]와 [0.51,0.9], t∈[10,50]에서 감김수=0. `bundle_chern_number.txt`
**비평가**: 생존 (동치 관계 자체는 자명, 가치는 기하학적 관점 제공).
