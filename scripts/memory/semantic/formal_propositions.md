## Definitions

### Definition 1 (ξ-다발)
임계대 S = {s = σ+it ∈ ℂ : 0 < σ < 1} 위의 주 U(1)-다발.
- 단면: ψ(s) = ξ(s)/|ξ(s)| = e^{iθ(s)} (Z = {ξ=0} 밖에서)
- 접속: A(s) = L(s)ds = (ξ'/ξ)(s)ds
- 곡률 밀도: κ(s) = |L(s)|²

### Definition 2 (모노드로미)
경로 γ: [0,1] → S\Z에 대해 μ(γ) = ∮_γ Im(L(s))ds = Δarg(ξ).

### Definition 3 (횡방향 곡률)
임계선 위에서 F₂ ≈ v_σ · cos(φ), v_σ = Im(L)|_{σ=1/2}.

## Theorems (해석적 증명)

### Theorem 1 (접속 반대칭)
L(1-s) = -L(s). 임계선 위에서 Re(L) = 0 (유니터리 게이지).
**증명**: ξ(s)=ξ(1-s) 미분 → ξ'(s)=-ξ'(1-s) → L(1-s)=-L(s). σ=1/2에서 s와 1-s̄의 관계로 Re(L)=0.
**검증**: |Re(L)|/|Im(L)| < 10⁻³⁵, 20개 비영점 + 6개 근방점. `bundle_gauge_symmetry.txt`
**비평가**: 생존.

### Theorem 2 (위상적 영점 세기)
폐곡선 C가 영역 R을 둘러쌀 때, N(R) = (1/2π) ∮_C d(arg ξ).
**증명**: 논증 원리(Cauchy)의 다발 기하학적 표현.
**검증**: N=1,3,5 정확 일치, N(50)=10. `bundle_chern_number.txt`
**비평가**: 생존. 단, 기존 수학의 재서술임을 명시.

### Theorem 3 (곡률 소멸 — 유니터리 게이지 귀결)
임계선 위 일반점(cos φ ≠ 0)에서 L = 0. 영점에서만 L → ∞.
**증명**: Thm 1에서 Re(L)=0. Klein 대칭 F₂=F₂^R 요구 → 2·Im(L)·cos(φ)=0 → Im(L)=0.
**검증**: 비영점 대비 영점에서 |L| 34배 집중. `bundle_curvature_2d.txt`
**비평가**: 생존. **[NEW]** 곡률 집중이 대칭에서 강제된다는 메커니즘은 새로운 기여.

### Theorem 4 (κδ² 임계선 감별) **[NEW]**
Λ(s) = ε·Λ̄(1-s̄), ε=±1인 완비 L-함수의 단순 영점 ρ에서:
(a) Re(ρ) = 1/2 ⟹ κδ² = 1 + O(δ²).
(b) Re(ρ) ≠ 1/2 ⟹ κδ² = 1 + Re(Λ''(ρ)/Λ'(ρ))·δ + O(δ²), 일반적으로 Re(Λ''/Λ')≠0.

**증명 (a)**: FE 미분:
  Λ'(s) = -ε·Λ̄'(1-s̄) → ρ=1/2+it₀에서 1-ρ̄=ρ이므로 Λ'(ρ)=-ε·Λ̄'(ρ) → e^{2iα}=-ε.
  Λ''(s) = ε·Λ̄''(1-s̄) → Λ''(ρ)=ε·Λ̄''(ρ) → e^{2iβ}=ε.
  ε=+1: α=π/2, β=0. ε=-1: α=0, β=π/2. 양쪽 모두 cos(β-α)=0.
  Re(Λ''/Λ') = |Λ''|/|Λ'| · cos(β-α) = 0. □

**증명 (b)**: Re(ρ)≠1/2이면 1-ρ̄≠ρ. FE는 Λ'(ρ)와 Λ'(1-ρ̄)을 연결 (다른 점).
  arg(Λ''(ρ)/Λ'(ρ))에 자기 구속 없음 → Re(Λ''/Λ') 일반적으로 ≠0. □

**검증**: DH 함수 (#115). On-critical κδ²=1.0017±0.0008 (O(δ²)=O(10⁻³) 일치).
  Off-critical κδ²=1.2095±0.0992. min(off)=1.114 > max(on)=1.004.
  `dh_off_critical_115.txt`
**δ-sweep 검증 (#116)**: δ∈{0.005,...,0.1} 8값에서 log|κδ²-1| vs log(δ) 기울기 추출.
  On-critical: 기울기 = 1.999 ± 0.003 (이론: 2.0), R²=1.0000, 10/10 영점. **O(δ²) 확인.**
  Off-critical: 기울기 = 0.963 ± 0.039 (이론: 1.0), R²≥0.999, 3/3 영점. **O(δ) 확인.**
  모든 δ에서 min(off) > max(on) → 완전 분리.
  c₁·|σ-0.5| ≈ {1.16, 1.04, 1.00} (3점, 관측).
  `dh_delta_sweep_116.txt`
**비평가**: P3 감별 **생존**. δ-sweep O(δ)/O(δ²) 스케일링 **생존**. c₁ 스케일링 **취약** (N=3→4).
**상태**: 해석적 증명 + δ-sweep 수치 검증 완료.

### Theorem 5 (Laurent 계수 패리티) **[NEW]**
FE Λ_π(s)=ε·Λ_{π̃}(1-s)와 CC Λ_{π̃}(s̄)=Λ̄_π(s)를 만족하는 L-함수의
임계선 위 단순 영점 ρ=σ_c+iγ에서:
  Λ'/Λ(ρ+u) = 1/u + Σ_{n≥0} cₙuⁿ
이면
  cₙ = (-1)^{n+1} c̄ₙ  for all n ≥ 0.
따라서:
  - n 짝수: cₙ 순허수 (Re(cₙ)=0)
  - n 홀수: cₙ 순실수 (Im(cₙ)=0)

**조건 일반화 (C-241, 수학자 판정 2026-04-22)**:
  원래 조건 SR → CC로 약화. CC: Λ(s̄, χ̄) = conj(Λ(s, χ)).
  모든 automorphic L-함수는 CC 자동 충족 (Dirichlet, GL(n) 포함).
  SR = 자기쌍대 특수 경우. CC가 더 일반적이고 충분.
  경계: CC 없는 L-함수 (Epstein ζ 등)에서 sharpness 미검증 (B-39→B-39').

**증명**:
  (1) FE 미분: Λ'/Λ(s) = -Λ'/Λ(1-s). s=ρ+u에서 1-s=ρ'-u (ρ'=1-ρ).
      Laurent 계수 비교: cₙ(ρ) = (-1)^{n+1} cₙ(ρ')  [u^n 차수별].
  (2) CC: Λ'/Λ(s̄, χ̄) = conj(Λ'/Λ(s, χ)) → cₙ(ρ̄, χ̄) = c̄ₙ(ρ, χ).
      자기쌍대(χ=χ̄): 직접 cₙ(ρ̄) = c̄ₙ(ρ).
      비자기쌍대: χ̄ 영점 = {ρ̄ : ρ∈Z(χ)}, 동일 패리티 구조.
  (3) ρ가 임계선 위: ρ' = 1-ρ = ρ̄. ∴ cₙ(ρ') = cₙ(ρ̄) = c̄ₙ(ρ).
  (4) 합산: cₙ = (-1)^{n+1} c̄ₙ. □

**따름정리**: Re(c₀)=0 (기존 Thm slopeuniv), Im(c₁)=0, Re(c₂)=0, Im(c₃)=0, ...
**검증 (C-240)**: ζ(s) 5영점, L(s,11a1) 3영점, L(s,χ₋₃) 3영점. 11/11 PASS.
  |Re(c₀)|/|c₀|: max 3.2e-12
  |Im(c₁)|/|c₁|: max 0
  |Re(c₂)|/|c₂|: max 3.2e-7  ← **최초 검증**
  |Im(c₃)|/|c₃|: max 0       ← **최초 검증**
  `laurent_parity_c240.txt`
**C-241 확장 검증**: 비자기쌍대 χ₅ (mod 5), χ₇ (mod 7), 대조군 χ₋₃. 9/9 INTACT.
  비자기쌍대에서도 패리티 유지 — 이유: FE + CC (Λ(s̄,χ̄)=conj(Λ(s,χ)))가 SR 없이 충분.
  정리의 진정한 조건: FE + CC (모든 Dirichlet에 자동 성립). SR/자기쌍대 불필요.
  Sharpness (B-39): CC 없는 L-함수(Epstein ζ, 비산술적)에서 검증 필요.
  `nonselfdual_parity_c241.txt`
**비평가**: **생존**. 증명은 FE+CC의 직접 귀결 (SR보다 약한 조건 충분). degree 무관. c₂, c₃는 최초 수치 검증.
**상태**: ★★★★ 확립. 해석적 증명 + 수치 검증 완료. C-241 FE+CC 일반화 수학자 확인 (2026-04-22).

### Theorem 5b (Off-Critical 홀수차 패리티 보호) **[NEW]**
FE+CC를 만족하는 L-함수에서 off-critical 단순 영점 ρ=(1/2+δ)+it₀ (δ≠0)의
Laurent 계수 cₙ (Λ'/Λ(ρ+u) = 1/u + Σcₙuⁿ)에 대해:

**(a) 홀수차 보호**: |Im(c_{2k+1})| / |Re(c_{2k+1})| = O(|δ|^{2k+3}) as δ→0
**(b) 짝수차 증폭**: |Re(c_{2k})| / |Im(c_{2k})| = O(|δ|^{-(2k+1)}) as δ→0

**증명**:
  FE+CC에 의해 off-critical 영점 ρ에는 파트너 1-ρ̄=(1/2-δ)+it₀가 존재.
  (1) **파트너 지배**: Hadamard 곱에서 파트너 기여 = (-1)^n/(2δ)^{n+1} (순수 실수).
      - n 홀수: Re(cₙ)에 O(1/δ^{n+1}) 기여 (패리티 보존 성분 증폭)
      - n 짝수: Re(cₙ)에 O(1/δ^{n+1}) 기여 (패리티 위반 성분 증폭)
  (2) **위반 성분**: on-critical 영점들의 δ-섭동에서 유래.
      j=1 항: O(δ)·Σ 1/Δt^{n+2}. n 홀수일 때 Δt의 홀수 거듭제곱 합이므로
      상하 대칭 영점 분포에 의해 상쇄 → 잔차 O(δ·ε/d^{n+3}).
  (3) **비율**:
      r(c_{2k+1}) = O(δ) / O(1/δ^{2k+2}) = O(δ^{2k+3}). □

**수치 검증 (C-243)**: DH 함수, 5 on-critical + 4 off-critical 영점, c₀-c₇ (72개 값).

### Corollary 4.2 (진폭 공식 보편성) **[NEW]**

  A = Im(c₀)² + 2Re(c₁)
이 공식은 degree 1-4의 모든 자기 표현(automorphic representation)에서 성립.

**증명**: Laurent 전개 Λ'/Λ(s) = 1/(s-ρ) + c₀ + c₁(s-ρ) + ... 에서
  |Λ'/Λ|² = 1/δ² + (Im(c₀)²+2Re(c₁)) + O(δ)  [∵ Re(c₀)=0, Im(c₁)=0]
  ∴ |Λ'/Λ|²δ² - 1 = (Im(c₀)²+2Re(c₁))δ² + O(δ³). □

### Corollary 4.3 (σ-프로파일 전개) **[NEW]**
On-critical 단순 영점 ρ=1/2+iγ에서 ε=σ-1/2≠0에 대해:
  κ(1/2+ε, γ) = |ξ'/ξ(1/2+ε+iγ)|² = 1/ε² + A(γ) + B(γ)ε² + O(ε⁴)
여기서:
  A(γ) = Im(c₀)² + 2c₁  (Cor 4.2와 동일)
  B(γ) = 2c₃ + 2Im(c₀)Im(c₂) + c₁²
**모든 홀수 차수의 ε가 소거됨** — κ는 ε²의 짝수 함수.

**증명**:
  (1) Laurent 전개 L = ξ'/ξ = 1/ε + c₀ + c₁ε + c₂ε² + ...
  (2) |L|² 전개에서 εᵏ (k 홀수)의 계수 = Σ_{m+n-2=k} Re(cₘc̄ₙ).
      k 홀수 → m+n 홀수 → m,n의 패리티가 다름.
      Thm 5: cₘ(짝수)=순허수, cₙ(홀수)=순실수 → cₘc̄ₙ = 순허수 → Re=0. □
  (3) ε⁰: 2Re(c₀)/ε × ε + |c₀|²+2Re(c₁) = 0 + Im(c₀)²+2c₁ = A ✓
  (4) ε¹: 2Re(c₂+c₀c̄₁) = 2Re(iIm(c₂)+iIm(c₀)c₁) = 0 ✓

**검증 (C-254 + C-255)**: ζ(s) 20영점.
  - α = -1.9946 ± 0.0025 (이론: -2), R²=1.0000, 20/20 algebraic 우세
  - FE 대칭 κ(σ)=κ(1-σ): max|Δ|/κ½ = 0 (기계 정밀도)
  - σ-scan c(γ) vs Laurent A(γ) 교차검증: ρ=0.99999744, max|Δ|/|A|=0.19%
  - O(ε²) 피팅: a_fit vs A(Laurent) 차이 ~ 3.5×10⁻⁶ (6자리 일치)
  `sigma_localization_profile_c254.txt`, `sigma_A_crosscheck_c255.txt`
**비평가**: **생존**. Thm 5의 대수적 귀결. 독립 계산법 2가지(σ-scan, Cauchy 적분)의 교차검증.
**상태**: ★★★★ 해석적 증명 + 수치 교차검증 완료.

### Corollary 4.1 (c₁ 거울쌍 점근 법칙) **[NEW]**
FE Λ(s)=ε·Λ̄(1-s̄)인 L-함수의 단순 off-critical 영점 ρ=σ₀+it₀ (σ₀>1/2)에서:
  c₁ = Re(Λ''(ρ)/Λ'(ρ)) = 1/(σ₀ - 1/2) + O(1)
따라서 c₁·|σ₀-1/2| → 1 as |σ₀-1/2| → 0.

**증명**:
  (1) Λ(s) = (s-ρ)·A(s), A(ρ)=Λ'(ρ)≠0 → Λ''/Λ' = 2A'/A
  (2) 하다마르 곱 분해: A'/A = 1/(ρ-(1-ρ̄)) + Σ_{ρ'≠ρ,(1-ρ̄)} 1/(ρ-ρ') + (정칙항)
  (3) ρ-(1-ρ̄) = 2(σ₀-1/2) (실수) → 거울쌍 기여 = 1/(2(σ₀-1/2))
  (4) c₁ = Re(2·[1/(2(σ₀-1/2)) + O(1)]) = 1/(σ₀-1/2) + O(1) □

**검증 (#117)**: DH 4 off-critical 영점에서:
  c₁_fit/c₁_analytic = 0.9982 ± 0.0019 (max rel_err 0.50%)
  c₁·|σ-½| = {1.160, 1.127, 1.043, 1.007} → mean=1.084±0.062
  거울쌍 분해: 배경/거울쌍 비율 → 1 as |σ-½| → 0
  `dh_c1_law_117.txt`
**비평가**: c₁=Re(Λ''/Λ') **생존**. c₁·|σ-½|→1 **생존** (점근 관계로서). N=4 **취약** (추가 L-함수 필요).

### Proposition 3 (cos² = k²=-1 격자) **[NEW]**
cos²(φ) 영점 = {π/2, 3π/2} = arg(k²=-1의 근 {i,-i}).

### Proposition 4 (PQO 호환성 정리, L≤2) **[NEW]**

**비평가**: 생존. 실험적 L=4 실패의 최초 수학적 설명.

### Proposition 5 (F₂ = 횡방향 곡률) **[NEW]**

### Observation 1 (중간점 NN 상쇄) [C-246, 2026-04-24]

### Proposition 11 (A(t₀) Hadamard 분해) **[NEW]**
**서술**: Selberg class degree-d, conductor N인 L-함수의 단순 영점 ρ₀ = σ_c + it₀에서
부선도 진폭 A(t₀) ≡ κ(σ_c+δ, t₀) - 1/δ²는 다음으로 분해된다:

  A(t₀) = Im(R(ρ₀))² + 2Re(R'(ρ₀))

여기서 R(s) = (d/ds)log[Λ(s)/(s-ρ₀)]|_{s=ρ₀} (ρ₀ 제외 Hadamard 잔차).

**증명 (Theorem 4 귀결)**:
  1. Λ(s) = (s-ρ₀)·A(s), A(ρ₀)≠0. Λ'/Λ = 1/(s-ρ₀) + R(s), R = A'/A.
  2. s = σ_c + δ + it₀ (δ 실수, 작음) 대입:
     Λ'/Λ = 1/δ + R(ρ₀) + R'(ρ₀)·δ + O(δ²)
  3. κ = |Λ'/Λ|² = 1/δ² + 2Re(R)/δ + |R|² + 2Re(R') + O(δ)
  4. Theorem 4: on-critical 영점에서 Re(R(ρ₀)) = 0 (FE 강제)
     → 1/δ 항 소거, κ = 1/δ² + Im(R)² + 2Re(R') + O(δ)
  5. A(t₀) = Im(R(ρ₀))² + 2Re(R'(ρ₀)). □

**물리적 해석**:
- Im(R(ρ₀)) = -Σ_{k≠0} 1/(t₀-tₖ) + Im(Γ-terms) = "영점 간 간섭" (S₁)
- 2Re(R'(ρ₀)) = 2Σ_{k≠0} 1/(t₀-tₖ)² + Re(Γ''-terms) = "인접 영점 밀도 효과" (2H₁)
- A ~ d²·(log N_eff)² (경험적, R²=0.983, 6 degree, 52 영점)
- 동일 가족(sym^n) 내: A ∝ d² (logN = n·log|Δ_E| ∝ d)

**검증**: #212 (6 degree, 52 영점). Out-of-sample:
- Artin S₅ (d=4, N=2869): pred 23.9, actual 24.9 (4%)
- RS(11a1×37a1) (d=4, N=407): pred 19.6, actual 30.7 (high variance)
- Artin S₃ (d=2, N=23): pred 8.6, actual 5.0 (70% — 소-N 영역 부정확)
