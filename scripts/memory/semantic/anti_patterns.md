## 방법론 안티패턴

### AP-05: 사후 필터로 precision 교정
- **설명**: 검출 후 사후 필터링으로 precision을 높이려는 시도 — 실패
- **이유**: F1=0.430 한계, |F₂| 풍경 구조의 근본적 한계
- **교훈**: precision 개선은 모델 자체에서 해결해야 함

### AP-06: 3시드로 "확립" 주장
- **설명**: 3시드 결과로 "확립됨"이라 표현 금지
- **올바른 표현**: "3시드 기준 양성 신호"
- **최소**: 5시드 이상에서 "확립" 사용 가능

### AP-07: 위상 측정 시 branch cut 미보정
- **설명**: arg() 차분으로 위상 점프를 셀 때 branch cut 보정 없이 `abs(phase - prev_phase) > threshold` 사용 금지
- **이유**: arg()는 (-π, +π] 범위. ±π 경계에서 wrap → diff≈2π → 거짓 점프 다수 카운트
- **올바른 방법**: delta를 (-π, π] 범위로 보정 후 `abs(delta) > π/2` 판정 (bundle_prediction_2_offcritical.py 패턴)
- **발생 횟수**: 1회 (dirichlet_high_conductor.py → 타σ=8 아티팩트 생성, 사이클 21에서 규명)
- **영향**: 결과 #13이 3사이클간 "미해명"으로 체류

### AP-08: κ 프로파일 해상도 부족
- **설명**: 상관 분석 시 κ 프로파일의 FWHM 측정에서 격자 step > 피크폭/3이면 FWHM=0 반환
- **이유**: 피크폭 ≈ 2δ (오프셋). δ=0.03이면 FWHM≈0.06. step=0.033에서 2점만 피크 내 → 해상 불가
- **올바른 방법**: step ≤ δ/3 확보 (예: half_width=0.3, n_points=200 → step=0.003)
- **발생 횟수**: 1회 (사이클 22 lehmer_curvature_analysis.py 4단계)
- **영향**: g_norm vs FWHM 상관 분석 실패 (78개 영점 데이터 손실)

### AP-09: Hadamard B 상수 단일점 추정
- **설명**: |F₂| 잔차 계산 시 보정 상수 B를 단일 보정점에서만 추정
- **이유**: B 불확실성이 모든 잔차에 체계적 바이어스 생성 가능
- **올바른 방법**: 최소 3개 보정점에서 B 추정 후 표준편차 보고

### AP-10: bundle_utils.py h=10^{-20} 고정 수치미분
- **설명**: connection_zeta()의 h가 10^{-20}으로 고정. t>100에서 ξ(s)가 10^{30}~10^{40} 크기 → 차분 유효자릿수 소진
- **이유**: dps=50에서 유효자릿수 ≈ 50 - log₁₀|ξ| - 20 → t>100이면 음수
- **올바른 방법**: h = 10^{-(dps//2)} 또는 dps ≥ 100으로 상향
- **발생 횟수**: 1회 (사이클 25, curvature_residual_nonlocal.py, 234쌍 중 192개 cap)
- **영향**: 82% 데이터 무효화, 통계 분석 전면 INVALID

### AP-11: 중간점에서 κ_nn = Σ 1/d² 정의
- **설명**: Hadamard 전개에서 중간점 m=(γₙ+γₙ₊₁)/2일 때, 두 최근접 영점의 복소 기여가 정확히 상쇄 (허수부 반대 부호). κ_nn = 1/d_n² + 1/d_np1² 는 교차항을 무시한 상한이며, 실제 기여 |기여합|² = 0.
- **이유**: 중간점에서 1/(s-ρₙ) + 1/(s-ρₙ₊₁) = 0 (대칭 취소)
- **올바른 방법**: (1) 중간점 κ_total 자체를 "비국소 기여"로 해석, 또는 (2) 중간점 대신 δ-오프셋 점에서 측정
- **발생 횟수**: 1회 (사이클 25)
- **영향**: κ_nn/κ_total 평균 = 3.672 (>1), 잔차 83% 음수 → 분석 프레임워크 전면 무효

### AP-12: 상관 분석에서 편상관 미보고
- **설명**: κ vs gap_next 상관 보고 시, gap과 gap_next 사이 자기상관(영점 반발로 ρ≈-0.15)을 통제한 편상관 미계산
- **이유**: 자기상관에 의한 자명한 효과를 배제하지 못하면 비국소성 주장의 근거 불충분
- **올바른 방법**: ρ(X,Z|Y) = (ρ_XZ - ρ_XY·ρ_YZ) / √((1-ρ_XY²)(1-ρ_YZ²)) 로 편상관 계산. 스크립트에 표준 포함.
- **발생 횟수**: 1회 (사이클 26, curvature_midpoint_nonlocal_v2b.py)
- **영향**: 이번 사례에서는 ρ(κ,gap)≈0이라 편상관≈원래 상관. 결과 무효화 아님. 그러나 일반적으로 필수.

### AP-13: mpmath.diff(xi_func, s) — t>300 실패
- **설명**: mpmath.diff로 ξ(s)의 수치미분 시도 시, t>300에서 ξ ~ exp(-πt/4)로 underflow
- **이유**: ξ 절대값이 dps 표현 범위를 벗어남
- **올바른 방법**: log(ξ)' 해석적 공식 = 1/s + 1/(s-1) - log(π)/2 + ψ(s/2)/2 + ζ'/ζ 사용. ξ 자체 계산 불필요.
- **발생 횟수**: 1회 (사이클 26, v2 → v2b로 수정)
- **영향**: v2에서 47.9% cap → v2b에서 0% cap으로 영구 해결

### AP-14: Dirichlet bare L'/L 사용 시 t-추세 혼입
- **설명**: κ = |L'(s,χ)/L(s,χ)|² 로 계산하면 ψ((s+a)/2) ~ log(t) 항이 포함되어 κ가 t와 단조 증가
- **이유**: L'(s,χ)/L(s,χ) = Λ'/Λ - (1/2)log(q/π) - (1/2)ψ((s+a)/2). ψ 항이 log(t) 배경 생성 → ρ(κ, density) = 0.93~0.97
- **올바른 방법**: Λ'/Λ = (1/2)log(q/π) + (1/2)ψ((s+a)/2) + L'/L 해석적 공식 사용. Λ 자체 계산 불필요.
- **발생 횟수**: 1회 (사이클 27, dirichlet_midpoint_nonlocal.py v1 → v2 수정)
- **영향**: v1에서 ρ(κ, density)=0.97 (무효) → v2에서 ρ(κ, density)≈0 (정상)

### AP-15: GUE 비교 시 unfolding 없이 raw gap 사용
- **설명**: GUE 고유값의 간격은 semicircle 밀도에 따라 위치별로 다름. raw gap을 사용하면 edge 효과로 가짜 상관 발생
- **올바른 방법**: 국소 평균 간격(sliding window ±25 이상)으로 정규화, 또는 semicircle 밀도로 나눔. Bulk만 사용 (중앙 80%).
- **발생 횟수**: 0회 (사이클 28에서 올바르게 구현됨. 예방적 기록)

### AP-16: Poisson 점 과정에서 Hadamard NN 상쇄 기대
- **설명**: Poisson(독립 랜덤 점)은 Hadamard 급수 구조가 없으므로, 중간점에서 NN 기여가 자동 상쇄되지 않음
- **이유**: NN 상쇄는 Hadamard 전개의 1/(s-ρ) 허수부 구조에서 비롯. 실수 resolvent에서도 1/(m-x_n) + 1/(m-x_{n+1}) = 0 (중간점이면). 그러나 κ = R(m)² 계산 시 NN 제외를 명시적으로 해야 L-함수와 공정 비교 가능.
- **올바른 방법**: κ 계산 시 NN 인덱스를 마스킹하여 제외 (GUE/Poisson 모두)
- **발생 횟수**: 0회 (사이클 28에서 올바르게 구현됨. 예방적 기록)

### AP-19: 대칭 비율 측정 전 FE 자명성 검증 필수
- **설명**: |Λ(c+δ+it)| / |Λ(c-δ+it)| = 1 항등 (FE: Λ(s)=ε·Λ(w+1-s)에서 자동). 이를 "독립 성질" P3으로 측정하는 것은 P1과 중복.
- **이유**: FE가 성립하면 임계선 중심 대칭 비율은 반드시 1. 이를 간과하고 실험 설계 → 자명한 결과를 "발견"으로 오인.
- **올바른 방법**: κ_near를 비대칭 측정 (|Λ(σ₁+it)| vs |Λ(σ₂+it)| where σ₁+σ₂≠2c) 또는 2차미분 기반으로 재정의.
- **발생 횟수**: 1회 (#108에서 확인, #107 κδ²≠1은 lfuninit 아티팩트)
- **영향**: P3 성질 전면 재검토 필요. B-26 해소.

### AP-18: ε=-1 자기쌍대 L-함수에 Re(Λ) 부호변환 적용 금지
- **설명**: root number ε=-1인 자기쌍대 L-함수에서 Re(Λ(center+it)) 부호변환을 측정하면 S=0이 나오지만, 이는 모노드로미 부재가 아님
- **이유**: ε=-1 + 자기쌍대 → Λ(center+it) = -Λ̄(center+it) → Re(Λ)≡0 (항등적). 자명한 산술적 귀결.
- **올바른 방법**: ε=-1일 때 **Im(Λ(center+it))** 의 부호변환을 P4 모노드로미 지표로 사용. σ-sweep도 Im 기반.
- **발생 횟수**: 1회 (#107, rank 1 37a1 + rank 3 5077a1 → S=0, Red Team "치명적")
- **영향**: B-08 rank-dependent 실험의 절반(ε=-1 곡선) 결과 무효화

### AP-17: Dirichlet Λ(s,χ) 직접 계산 → monodromy arg 실패
- **설명**: completed_L(s,χ) = (q/π)^{s/2} · Γ((s+a)/2) · L(s,χ)에서 t>190 (mod 7)일 때 |Γ| ~ 10^{-65}로 Λ가 underflow. arg(Λ) 반환값이 포화/0.
- **이유**: Γ((s+a)/2)의 지수감쇠 e^{-πt/4}. dps=80에서도 t≈193부터 파손.
- **올바른 방법**: arg(Λ) = Im[log(Λ)] = Im[(s/2)log(q/π) + loggamma((s+a)/2) + log(L)]. 각 항 log-space 개별 계산.
- **발생 횟수**: 1회 (사이클 58, #40 → #40b에서 수정)
- **영향**: 136영점 중 7개 mono/π = 0.000 오판정 → log 수정 후 136/136 = 2.000 완벽
