# RDL Anti-Patterns — 하지 말아야 할 것

## 코드 안티패턴

### AP-01: Z_out.abs() 사용 금지
- **설명**: eval_F2에서 Z_out.abs() 대신 phi/psi/L_G 기반 잔차 사용
- **이유**: Z_out.abs()는 위상 정보를 잃어 영점 검출 불가
- **발생 횟수**: 2회

### AP-02: get_or_build_cache 인자 순서
- **설명**: 첫 인자는 cache_path (string), in_features는 별도 키워드 인자
- **이유**: 순서 뒤바뀌면 파일명으로 정수가 전달됨
- **발생 횟수**: 1회

### AP-03: is_near_zero at t>300
- **설명**: t>300에서 xi 값이 언더플로 → is_near_zero 마스크 사용 불가
- **대안**: 극소값 탐색 사용
- **발생 횟수**: 1회

### AP-04: 병렬 실험 실행
- **설명**: CPU 12코어에서 2개 이상 실험 동시 실행 금지
- **이유**: 경합으로 전체가 느려짐 (실측 확인)
- **발생 횟수**: 1회

## 방법론 안티패턴

### AP-05: 사후 필터로 precision 교정
- **설명**: 검출 후 사후 필터링으로 precision을 높이려는 시도 — 실패
- **이유**: F1=0.430 한계, |F₂| 풍경 구조의 근본적 한계
- **교훈**: precision 개선은 모델 자체에서 해결해야 함

### AP-06: 3시드로 "확립" 주장
- **설명**: 3시드 결과로 "확립됨"이라 표현 금지
- **올바른 표현**: "3시드 기준 양성 신호"
- **최소**: 5시드 이상에서 "확립" 사용 가능

### AP-07: 위상 측정 시 branch cut 미보정
- **설명**: arg() 차분으로 위상 점프를 셀 때 branch cut 보정 없이 `abs(phase - prev_phase) > threshold` 사용 금지
- **이유**: arg()는 (-π, +π] 범위. ±π 경계에서 wrap → diff≈2π → 거짓 점프 다수 카운트
- **올바른 방법**: delta를 (-π, π] 범위로 보정 후 `abs(delta) > π/2` 판정 (bundle_prediction_2_offcritical.py 패턴)
- **발생 횟수**: 1회 (dirichlet_high_conductor.py → 타σ=8 아티팩트 생성, 사이클 21에서 규명)
- **영향**: 결과 #13이 3사이클간 "미해명"으로 체류

### AP-08: κ 프로파일 해상도 부족
- **설명**: 상관 분석 시 κ 프로파일의 FWHM 측정에서 격자 step > 피크폭/3이면 FWHM=0 반환
- **이유**: 피크폭 ≈ 2δ (오프셋). δ=0.03이면 FWHM≈0.06. step=0.033에서 2점만 피크 내 → 해상 불가
- **올바른 방법**: step ≤ δ/3 확보 (예: half_width=0.3, n_points=200 → step=0.003)
- **발생 횟수**: 1회 (사이클 22 lehmer_curvature_analysis.py 4단계)
- **영향**: g_norm vs FWHM 상관 분석 실패 (78개 영점 데이터 손실)

### AP-09: Hadamard B 상수 단일점 추정
- **설명**: |F₂| 잔차 계산 시 보정 상수 B를 단일 보정점에서만 추정
- **이유**: B 불확실성이 모든 잔차에 체계적 바이어스 생성 가능
- **올바른 방법**: 최소 3개 보정점에서 B 추정 후 표준편차 보고

### AP-10: bundle_utils.py h=10^{-20} 고정 수치미분
- **설명**: connection_zeta()의 h가 10^{-20}으로 고정. t>100에서 ξ(s)가 10^{30}~10^{40} 크기 → 차분 유효자릿수 소진
- **이유**: dps=50에서 유효자릿수 ≈ 50 - log₁₀|ξ| - 20 → t>100이면 음수
- **올바른 방법**: h = 10^{-(dps//2)} 또는 dps ≥ 100으로 상향
- **발생 횟수**: 1회 (사이클 25, curvature_residual_nonlocal.py, 234쌍 중 192개 cap)
- **영향**: 82% 데이터 무효화, 통계 분석 전면 INVALID

### AP-11: 중간점에서 κ_nn = Σ 1/d² 정의
- **설명**: Hadamard 전개에서 중간점 m=(γₙ+γₙ₊₁)/2일 때, 두 최근접 영점의 복소 기여가 정확히 상쇄 (허수부 반대 부호). κ_nn = 1/d_n² + 1/d_np1² 는 교차항을 무시한 상한이며, 실제 기여 |기여합|² = 0.
- **이유**: 중간점에서 1/(s-ρₙ) + 1/(s-ρₙ₊₁) = 0 (대칭 취소)
- **올바른 방법**: (1) 중간점 κ_total 자체를 "비국소 기여"로 해석, 또는 (2) 중간점 대신 δ-오프셋 점에서 측정
- **발생 횟수**: 1회 (사이클 25)
- **영향**: κ_nn/κ_total 평균 = 3.672 (>1), 잔차 83% 음수 → 분석 프레임워크 전면 무효
