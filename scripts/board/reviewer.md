#### 5. ★ 스크립트 비교 버그 확인 (설계자 이미 식별)

결과 파일의 "Δκ/κ=799%" 비교는 **무의미**:
- ζ의 κ_near=1112.32는 **δ=0.03 기준** (1/0.03² + 1.21 = 1112.32)
- χ의 κ_near는 **δ=0.01 기준** (1/0.01² + A = 10002+A)
- 서로 다른 δ에서 κ_near를 비교하는 것은 1/δ² 차이를 비교하는 것일 뿐

**올바른 비교** (검토자 독립 계산, 동일 δ=0.03):

| L-함수 | κ_near(δ=0.03) | Δκ/κ(ζ) |
|--------|--------------|----------|
| ζ(s) | 1112.32 | 기준 |
| χ₃ | ~1113.36 | **+0.093%** |
| χ₄ | ~1113.88 | **+0.140%** |
| χ₅ | ~1114.25 | **+0.174%** |

→ **모든 차이 < 0.5% → 가설 A (degree-only) 지지** (κ_near 수준에서)
→ **단, A(t₀) 자체는 conductor-의존** (72~142% 차이) → 가설 B 부분 지지

#### 6. ★★ 실험 설계 결함: 짝수 character 대조군 누락

**수학자 원래 지시 (표)**:
```
| 1 | L(s,χ₃) | ??? | Γ((s+1)/2) (χ₃ odd) |
| 1 | L(s,χ₄) | ??? | Γ(s/2) (χ₄ even) |   ← 핵심 대조군!
```

수학자는 χ₄(even, μ=0)을 ζ와 동일한 감마인자 대조군으로 요청했으나:
- **mod 4 비자명 character는 홀수(odd, a=1)뿐** — 짝수 비자명 character가 존재하지 않음
- bundle_utils의 세 character 모두 a=1 (홀수): χ₃(odd), χ₄(odd), χ₅(odd)
- **μ=0 (even) 대조군이 전혀 테스트되지 않음**

**결과**: μ=0 vs μ=1 효과를 분리할 수 없음. 관측된 A(t₀) 차이가:
  (a) μ 차이 때문인지 (ζ: μ=0 vs χ: μ=1)
  (b) conductor q 차이 때문인지 (q=π vs q=3,4,5)
  (c) 둘 다인지

**부분 해결**: 같은 μ=1인 χ₃/χ₄/χ₅ 간 A(t₀) 차이(2.25→2.77→3.14)는 순수 conductor 효과 ★

**권고**: mod 5에는 짝수 비자명 character (Legendre symbol (n/5), a=0)가 존재. 이를 추가 테스트하면 μ 효과 격리 가능. bundle_utils에 추가 필요.

#### 7. Devil's Advocate — 반례 탐색

1. **A(χ₅) ≈ π는 우연인가?** A(χ₅)=3.1407, π=3.14159, 차이 0.03%. 24영점 mean의 우연 일치 가능성 높음. median=2.7812로 π와 무관. **과대 해석 경고**.

2. **A(t₀)의 conductor 의존성은 자명한가?** Λ'/Λ = log(q/π)/2 + ψ((s+a)/2)/2 + L'/L(s). log(q/π)/2가 A에 기여하므로, q 증가 → A 증가는 **해석적으로 예측 가능**. 참신성 제한.

3. **영점 검증 없음**: χ의 영점이 진짜 영점인지 독립 확인 방법 없음 (LMFDB 데이터 부재). 하지만 20-24개 영점의 A(t₀) 일관성(CV < 1%)이 간접 증거.

4. **t 범위 편향**: ζ는 t∈[10,60] 13영점, χ₃/₄/₅는 t∈[14,59] 20-24영점. 범위 약간 다르나 실질 영향 미미.

#### 8. 수학자 판단 시 권고 사항

1. **가설 판정**: "가설 A/B"는 이분법이 아님. κ_near 수준에서는 A(degree-only), A(t₀) 수준에서는 B(conductor+μ 의존). **정밀한 기술 필요**.

2. **짝수 character 추가 실험**: B-09 정밀 검증을 완결하려면 mod 5 even character (a=0) 테스트 필수. 현재 데이터만으로는 μ 효과 미격리.

3. **논문 반영 시 주의**: 
   - "799%" 비교는 절대 인용하지 말 것 (버그)
   - A(t₀)의 conductor 의존성 = log(q/π)/2 기여 → 자명한 부분 명시
   - A(χ₅)≈π 언급 자제 (우연 일치 가능성 높음)
