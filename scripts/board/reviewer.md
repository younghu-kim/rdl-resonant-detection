#### 4. Devil's Advocate — 반례 탐색

1. **A_Γ = Im(H₀^Γ)²는 H₁ 기여 무시**: A = Im(H₀)² + 2Re(H₁)의 전체 공식에서 H₁^Γ 기여가 A_Γ에 포함되지 않음. 따라서 A_L = A_measured - Im(H₀^Γ)²는 순수 "산술적 기여"가 아니라 H₁^Γ를 포함하는 "잔차"이다.
   - **영향**: A_L의 해석을 "산술적 기여"에서 "Γ 1차 근사를 뺀 나머지"로 수정 필요
   - 논문 기술 시 "A_L contains both arithmetic (L'/L) and higher-order Γ-contributions (H₁^Γ)" 명시 권고

2. **개별 L-함수 내에서 A_Γ 비중은 무시 불가**: ζ에서 A_Γ/A = 42%, χ₅²에서 20%. *변화량* ΔA에서만 5.2%. 이 차이 주의 — 논문에 "A_Γ contributes only 5.2%" 기술 시 "of the *change* ΔA" 한정 필요.

3. **H₀^L 직접 검증 불가**: Im(H₀^L) 수치 값이 5e8 수준으로 발산하여 A_L의 독립 검증 경로 부재. A_L = A - A_Γ라는 단일 경로에만 의존. 그러나 A_measured(#76)와 A_Γ(해석적) 모두 신뢰도 높으므로 실질적 우려 아님.

4. **H₀^Γ 공식 차이**: ζ는 ξ'/ξ에서 1/s + 1/(s-1) 항을 포함하나 χ₅²는 미포함. 이는 정당 — ζ(s)는 s=0,1에 극이 있고 L(s,χ)는 없음. 하지만 이 구조 차이가 "Γ 기여"에 s=0,1 극 효과를 혼입시킨다. 엄밀히 H₀^Γ라 하기 어려울 수 있음 — "H₀^{Γ+poles}"가 정확.
