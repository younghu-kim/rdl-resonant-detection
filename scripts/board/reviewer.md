#### 1-2. Red Team: 해석적 자명성 비판

**핵심 비판**: α=-2는 **해석적으로 자명한** 결과.

1. ξ(s)의 단순 영점 ρ = ½+it_n에서, L = ξ'/ξ는 단순극 (잔류 1)
2. κ = |L|² = |ξ'/ξ|² ~ 1/|s-ρ|² (ρ 근방)
3. σ방향 접근 시: |s-ρ|² = (σ-½)² → κ ~ 1/(σ-½)²
4. **따라서 α=-2는 ξ의 단순영점이면 자동**. 새 정보량은 없음.

**설계자도 이 점을 인지**: "α=-2는 단순 영점의 직접 귀결 — 해석적으로 설명 가능"

**비자명한 부분**: 배경 보정항 κ(σ,t_n) - 1/(σ-½)²의 구조.
- 영점 #1: 배경 ~0.019 (σ=0.3)

- 설계자 해석 "왜 σ=1/2에서만 발산하는가?" → "왜 ξ의 영점이 σ=1/2에만 있는가?"로 환원" — **정확하지만 순환적**. RH 자체를 전제한 실험이므로.
