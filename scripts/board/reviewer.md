#### 4. Red Team: 반례/우려 탐색

1. **4점 감쇄율**: n=1,3,5,7 → 4개 데이터포인트. R²=0.9975로 높지만, 4점 선형 피팅은 통계적으로 2 자유도. **"지수적 감쇄"라고 단언하기보다 "4점에서의 일관된 감쇄 관측"으로 표현 권고.**
2. **n=6 on-critical 노이즈 ~0.05**: off-critical n=6 짝수차 r은 10¹~10⁶으로 S/N > 10⁷ → 안전. off-critical n=7 홀수차 r은 10⁻⁵~10⁻¹⁰ → on-critical 짝수 n=6 노이즈(0.05)보다 3~8자릿수 아래 → **홀수차 신호는 진짜일 가능성 높음.** ✓
3. **OFF#3 극적 비대칭**: r(c₆)=10⁶ vs r(c₇)=10⁻¹⁰ → 16자릿수 차이. 이 영점이 임계선에 가장 가까워(|σ-½|=0.074) 효과가 가장 극적 — 물리적으로 합리적. ✓
4. **DH 단일 사례**: 여전히 DH만 off-critical 영점 보유. 다른 FE 만족 함수에서 동일 현상 확인 불가. **한계이나, 현재 가용 자원으로 최선.** ✓
5. **power law β**: R²=0.96은 양호하나, 4점에서 벗어나는 패턴(OFF#3의 비선형성)이 있을 수 있음. 조심스러운 해석 필요.
