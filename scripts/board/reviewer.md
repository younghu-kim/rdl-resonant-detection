#### 방법론 검증

| 항목 | 구현 | 판정 |
|------|------|------|
| GUE 생성 | H=(A+A†)/(2√N), A_{ij}~CN(0,1) | ✅ 표준 GUE |
| 고유값 | `eigvalsh` (에르미트 전용) | ✅ |
| Unfolding | 반원 CDF: (arcsin(x/2)+(x/2)√(1-x²/4))/π+0.5 | ✅ 정확 |

| 내부 선택 | 양 끝 10% 제외 (edge Tracy-Widom 방지) | ✅ |

| 시드 관리 | base_seed + ens×7919 + N (비중복) | ✅ |
| ⚠️ 상관계수 | **Pearson** 사용 (L-함수는 Spearman) | ⚠️ 주의 |

#### ⚠️ Pearson vs Spearman 차이

L-함수 실험들은 Spearman rank correlation을 사용. C-276은 Pearson을 사용.
- **정성적 결론에 영향 없음**: Pearson도 모든 N에서 음의 상관 + p≈0
- **정량 비교 시 주의**: GUE ρ_Pearson = -0.23~-0.33 vs L-함수 ρ_Spearman = -0.42~-0.59
  - Pearson이 이상치에 민감 → GUE의 |ρ|가 실제보다 약화될 수 있음
  - 정밀 비교 필요시 Spearman으로 재측정 권장

#### 수치 검증

| N | 2H₁/A_bare | ρ(A,gap) | n | 일관성 |
|---|------------|----------|---|--------|
| 50 | 0.7743 | -0.1240 | 4,000 | N×100×0.8=4,000 ✅ |
| 100 | 0.7781 | -0.3208 | 8,000 | ✅ |
| 200 | 0.7786 | -0.3302 | 16,000 | ✅ |
| 500 | 0.7787 | -0.2259 | 40,000 | ✅ |

- **2H₁/A 수렴**: 0.7743→0.7787 (Δ=0.004). N=200→500에서 Δ=0.001. 수렴 확인 ✅
- **2H₁/A ≠ 2/3**: |0.779 - 0.667| = 0.112. 명확한 차이 ✅
- **A-gap**: 모든 N에서 ρ<0, p≈0. N=500에서 약화(0.33→0.23)는 주목 — finite-size 또는 N→∞ 소실 여부 추가 조사 가치
- **H₁^{NN}/H₁ → 2/3**: 0.6851→0.6692 (N=50→500). 수렴 방향 2/3 ✅

#### Red Team

1. **N=500에서 ρ(A,gap) 감소**: N=200: -0.330 → N=500: -0.226. 감소 추세. N→∞에서 A-gap이 소실될 가능성? 
   - 반론: n=40,000으로 p≈0이므로 통계적으로 유의. 크기 변화는 bulk 밀도 변화에 따른 정상적 스케일링일 수 있음.
   - **다음 실험에서 N=1000, 2000 추가하면 추세 확정 가능.**

2. **Pearson vs Spearman**: 위 항목 참조. Spearman 재측정 권장하나, 정성적 결론 불변.

3. **2H₁/A ≈ 0.78 해석**: GUE에서 2H₁/A = 0.78, L-함수에서 0.66. 차이 0.12는 "L-함수의 산술적 구조가 S₁²(= B²) 성분을 증폭시킨다"는 해석이 자연스러움. 이것은 과대 해석이 아님 — B² 성분이 커지면 2H₁/A가 줄어드는 것은 정의에서 직접 따름.

4. **과대 해석 점검**: 설계자가 "★★★★★ A-gap은 GUE 보편 현상"이라고 판정했으나, N=500에서 ρ 감소 추세를 고려하면 "GUE에서도 A-gap 신호가 관측된다" 정도가 적절. N→∞ 추세 확인 전까지 "확립"은 유보.
