#### 1-3. Red Team: 핵심 비판

**비판 1: A_bg 분산 기여율 해석 주의**

보고된 A_bg 분산 기여율 50.8%는 `var(A_bg)/var(A)` — 그러나 A_NN, A_bg, A_cross는 독립이 아니므로 분산의 합이 100%가 되지 않는다. 교차항 A_cross(10.9%)가 있으므로 분산 기여율은 참고치일 뿐 엄밀한 분해가 아님.
→ **논문 반영 시 "분산 기여율"보다 Spearman 상관 비교(ρ(A, 2c₁)=+0.88 >> ρ(A, A_NN)=+0.62)가 더 적절한 표현.**

**비판 2: "B-42 부분 해결" 주장의 한계**

- ρ(κ_mid, A_avg | gap) = +0.391 → gap 통제 후에도 유의미. 이는 "A가 gap의 대리변수가 아님"을 보여줌. ✅
- 그러나 편상관 0.39는 moderate. "부분 해결"이 적절한 표현.
- "어떤 메커니즘으로 κ_mid와 A가 연결되는가"는 미해명. ← **완전 해결 아님**

**비판 3: A_NN 통제 후 강화 현상의 해석**

- ρ(κ_mid, A_avg | A_NN) = +0.436 > 비조건부 +0.351 → "A_NN 통제 시 강화"
- 이는 A_NN이 κ_mid-A 관계에 노이즈로 작용함을 시사. 
- 단, suppressor variable 효과일 수 있음 — A_NN이 A_avg의 "불필요한 분산"을 설명하여 κ_mid와의 순수 상관이 노출될 수 있음. 인과 해석 불가.
→ **적절한 표현: "A_bg가 κ_mid 연결의 주된 담지체"**

**비판 4: 짝 패턴 강화 (ρ: 0.40 → 0.58)**

- ρ(A_bg_n, A_bg_n+1) = +0.58 > ρ(A_n, A_n+1) = +0.40 → NN 제거 시 강화
- 이는 NN 항이 인접 쌍 A 상관에 노이즈를 추가함을 시사 (단, 0.58-0.40 = 0.18 차이는 suppressor로도 설명 가능)
- **핵심 발견**: 배경항(c₁ 기반)에 비자명한 인접 영점 상관이 존재. 이는 GUE 통계와 연결될 가능성 ← 추후 연구 가치 높음

**비판 5: ρ(κ_mid, gap | A_avg) = +0.185 약한 유의 해석**

- p=0.009 → 유의하지만 약함. A를 통제해도 κ_mid-gap 약한 잔류 상관.
- "A가 완전한 매개변수는 아님" → B-42의 κ_mid-gap 관계는 A 외의 경로도 존재 가능
