#### 4. ★★★ 과대 해석 점검 (Red Team 핵심)

**κ ≈ 1/δ² 자체는 수학적 자명 결과 (tautology)**:

모든 해석함수 f(z)가 z₀에서 단순 영점을 가질 때:
```
f'/f(z₀+δ) = 1/δ + a₀ + a₁δ + ...  (Laurent 전개)
|f'/f|² = 1/δ² + 2Re(a₀)/δ + |a₀|² + O(δ)
```
- **1/δ² 지배항**: 단순 영점의 수학적 귀결. 어떤 L-함수에서든 성립.
- **Re(a₀)=0**: 함수방정식 ξ(s)=ξ(1-s)의 σ=1/2 대칭에서 자동 성립.
- **이것은 "발견"이 아니라 "확인"**

**논문에 쓸 때 주의사항**:
- ❌ "We discover a universal scaling law κ ~ 1/δ²"
- ✅ "We verify numerically that κ exhibits the expected 1/δ² scaling from the simple-pole structure of ξ'/ξ, with O(1) correction A(t₀) = |Im(a₀)|² encoding zero-specific information"
- 핵심 가치: degree별 κ_near 차이가 A(t₀)의 γ-인자 의존에서 기인함을 **정량적으로 확인**
- B-12 단조증가 위반의 이론적 해명: 1/δ²가 보편 지배항이고, 비단조 부분은 A(t₀)에만 있음

#### 5. 진짜 가치 있는 결과 (논문 반영 시 강조할 것)

1. **O(1/δ) 계수 = 0**: 함수방정식 대칭의 수치적 확인
2. **A(t₀) 구조**: t₀ 증가 → A 증가, digamma 기여로 설명 가능
3. **B-12 재해석**: κ_near(d) 비단조 = A(t₀, μ) 의존, 1/δ²는 보편적
