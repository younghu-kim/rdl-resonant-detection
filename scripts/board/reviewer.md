#### 1. ★ 핵심 발견: FWHM 41× 주장은 측정 아티팩트

**사이클 42 주장**: OFF #1 (Spira zero) FWHM = 0.615 (on-critical 대비 41×) — "σ-국소화 붕괴 명확"

**불일치 원인 확인 (코드 직접 검증)**:

**결론**: "41× FWHM 확대"는 윈도우 아티팩트. 실제 FWHM 확대는 **~2×** 수준. 사이클 42의 핵심 주장이 대폭 약화됨.

#### 5. Devil's Advocate — 결과 #30 전체 재평가

1. **41× FWHM은 아티팩트**: 핵심 주장 붕괴. 실제 효과는 2× (OFF #1만).
2. **N=4 중 양성 1개**: 4개 clean 영점 중 peak_σ→σ₀ 패턴은 OFF #1만. 25% 성공률.
3. **OFF #2-4의 반대 이동**: peak_σ < 0.5는 프레임워크 예측과 모순. 고 t에서 DH 함수의 κ 구조가 예상과 다르게 작동.
4. **DH ≠ L-함수**: 오일러 곱 없음. DH에서의 결과가 진짜 L-함수로 전이되는지 불명.

**반론**:
1. FWHM 2×도 on-critical과 구분됨. 작지만 존재하는 차이.
2. OFF #1 (Spira)은 가장 유명하고 검증된 off-critical 영점. 이것만으로도 가치.
3. OFF #2-4의 실패는 "DH 함수의 고 t 구조" 문제일 수 있음 — 프레임워크 자체의 결함이 아닌 DH 특수성.
4. 맞음. 이것은 한계로 명시해야.
