#### 1-2. Red Team: 핵심 비판

**비판 1: t-confounding (가장 중요한 우려)**

A는 t와 양상관 (R²=0.25), gap은 t와 음상관 (평균 간격 ~ 2π/log(t)). 따라서 A와 gap의 음상관이 단순히 공유된 t-추세에서 발생할 수 있다.

**검증**: t-추세 제거 후 잔차 상관:
- 20점 부분집합: ρ(A_resid, gap_resid) = **-0.7624** (p=9.3e-5)
- GUE 정규화 후 (200점): ρ(A, gap_min_GUE) = **-0.6209** (p=1.7e-22)

→ **t-추세 제거 후에도 상관 유의미하게 지속**. 단, 원시 상관 -0.85에서 -0.62~-0.76으로 약화 → 원시 상관의 ~25%는 t-confounding. **논문 반영 시 GUE 정규화 상관(-0.62)을 주 수치로 보고** 권고.

**비판 2: left/right 비대칭의 해석**

- ρ(A, gap_right) = -0.81 vs ρ(A, gap_left) = -0.18 → 4.4배 비대칭
- GUE 정규화 후: gap_right_GUE = -0.59 vs gap_left_GUE = +0.09(비유의)

Hadamard 곱에서 c₀의 지배적 기여는 최근접 영점. 좌우 대칭이 기대되므로 이 비대칭은 **비자명**. 그러나:
- 가능한 설명: gap_right와 A의 상관이 같은 "영점 클러스터"의 국소 밀도를 인코딩. gap_left는 이전 클러스터에 속할 수 있음.
- 대안 설명: A(γ_n)은 ξ'(ρ_n)에서 유도 — ξ'의 값은 Hadamard 곱의 비대칭적 부분합에 의존 (미래 영점의 기여가 과거와 다를 수 있음)
- **이 비대칭의 메커니즘은 미해명** → 관측적 보고만 적절

**비판 3: B-42 연결의 강도**

- ρ(κ_mid, A_avg) = +0.36 — moderate. 인과가 아닌 상관.
- ρ(κ_mid, gap) = -0.03 — 비유의. 직접 경로 없음.
- "κ_mid → A → gap 인과 체인" 주장은 **과대 해석** 위험. 매개변수 분석(mediation analysis) 없이 인과 주장 불가.
- 적절한 표현: "A(γ)가 κ_mid와 gap 사이의 통계적 매개 역할을 할 가능성을 시사"

**비판 4: 200점 통계 충분성**

p-값들이 매우 작아(10⁻⁵⁷) 통계적 유의성은 의심 여지 없음. 200점은 Spearman 상관에 충분. ✅ 생존.

**FE 대칭**: κ(σ) = κ(1-σ) 모든 180쌍에서 |Δ|/κ½ = 0.00e+00 (float 정밀도까지 완벽) ✅

#### 1-2. Red Team: 해석적 자명성 비판

**핵심 비판**: α=-2는 **해석적으로 자명한** 결과.

1. ξ(s)의 단순 영점 ρ = ½+it_n에서, L = ξ'/ξ는 단순극 (잔류 1)
2. κ = |L|² = |ξ'/ξ|² ~ 1/|s-ρ|² (ρ 근방)
3. σ방향 접근 시: |s-ρ|² = (σ-½)² → κ ~ 1/(σ-½)²
4. **따라서 α=-2는 ξ의 단순영점이면 자동**. 새 정보량은 없음.
