#### 1. 수치 재현 확인 ✅

독립 재계산 수행:

- **EM 보정(t₁)**: B_tail=0.01250, H₁_tail=0.00088, A_corrected=0.471332 ✓ (0.001% 오차)

#### 2. 수학적 증명 검증 ✅

- **D=0 논증**: ξ(s)=ξ(1-s) → ξ'/ξ(1/2)=0 → 대칭 쌍 합산 시 D=0. ✓ 표준 논증.
- **Laurent 전개**: g(ρ₀) = ξ'/ξ(ρ₀+δ) - 1/δ에서 c₀ = iB(순허수), c₁ = H₁(실수). ✓
  - Re(c₀)=0: 각 항 1/(i(t₀-tₙ))+1/(i(t₀+tₙ))은 순허수. ✓
  - c₁ 실수: 각 항 1/(t₀-tₙ)²+1/(t₀+tₙ)²은 실수. ✓
- **A 전개**: |1/δ + iB + H₁δ|² = 1/δ² + B² + 2H₁ + O(δ²). ✓
  - 교차항 2Re(iB/δ) = 0 (B 실수, 1/δ 실수). ✓
  - O(δ) 항 2Re(iB·H₁·δ) = 0 (순허수). ✓ → 정확히 O(δ²).

#### 3. 반례 탐색 / Devil's Advocate

1. **N=2000 부분합이 <1% 미달 (0/13)**: 성공 기준은 "N=5000 < 1%"였으나 캐시가 N=2000뿐. EM 보정으로 우회.
   - **판단**: EM 보정은 zero density ρ(t)=log(t/2π)/2π를 사용한 적분 근사로, 수학적으로 정당. 다만 논문에 "N=2000 partial sum + EM tail correction"임을 명시해야 함. "N=5000에서 수렴" claim은 ✗.
   - **권고**: 논문 반영 시 "partial sums with N=2000 zeros supplemented by Euler–Maclaurin tail estimates" 표현 필수.

2. **Richardson 외삽 A_rich ≠ A_direct**: t₁에서 A_rich=0.431 vs A_direct=0.471 (8.5% 차이).
   - **원인**: Richardson은 ε=1.25e-4에서 c₁ 추출 — O(ε²) 항 소거 불충분.
   - **판단**: Richardson은 검증 보조일 뿐, 핵심 결과에 영향 없음.

3. **조건부 수렴 처리**: B 급수는 개별 항이 ~1/tₙ로 발산하지만, (tₙ, -tₙ) 쌍으로 합산하면 ~2t₀/tₙ²로 절대수렴.
   - **판단**: 스크립트가 올바르게 (t₀-tₙ)⁻¹+(t₀+tₙ)⁻¹ 형태로 쌍합산. ✓

4. **B-20 해석의 강도**: "t₉의 높은 A는 근접 영점 H₁ 기여 0.406이 66%를 차지"
   - **검증**: H₁_near(t₉)=0.406, A(t₉)=2.458 → 0.406/2.458 = 16.5% (H₁의 66%가 아니라 A의 16.5%).
   - **수정 필요**: 결과 파일 remark에서 "66% of A(t₉)" → 부정확. 정확히는: 근접 영점 H₁=0.406 중 t₁₀(Δt=1.77) 기여가 0.320으로 H₁_near의 79%. 전체 A에서의 비중은 2×0.406/2.458 = 33% (2H₁_near/A).
   - **권고**: 논문 반영 시 정확한 비율 사용. "근접 영점이 H₁을 지배" 정도가 적절.
