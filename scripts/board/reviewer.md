#### 2. sym³(Δ) Richardson: ❌ 완전 실패

**핵심 문제**: δ=0.0001 데이터 오염
- κδ²(0.0001) = 0.993 → [0.99, 1.01] 범위 내이나 한계
- Z'(t₀) ≈ 2.8×10⁻³⁰ → float64 변환 시 손실 우려
- 2-point Richardson(δ=0.0001, 0.001) → A_true = 57263~84797 (비물리적)

**모델 부적합**:
- 4-point fit 잔차: 6649%~3028482% → A = A_true + c/δ 모델 자체가 sym³(Δ)에 맞지 않음
- A(δ) 패턴이 단순 1/δ가 아닌 고차항(1/δ², log δ 등) 포함 가능

#### 4. 구조적 발견 (유효)

- **Hardy Z κ에서 1/δ 보정항(c₁) ≠ 0**: 이는 Z의 테일러 전개 κ ≈ 1/δ² - Z''/Z' · 1/δ + ...에서 자연스럽게 발생
- **c₁ = -Z''(t₀)/Z'(t₀)**: L-함수 의존적 (영점 곡률에 의존)
- **#73 'c₁=0 정리'와 정합**: ξ-bundle κ(σ-방향)와 Hardy Z κ(t-방향)는 다른 양 → 모순 없음
- **c값은 L-함수 의존적**: c(sym³Δ) ≫ c(sym³(11a1)) (비율 ~224×)
