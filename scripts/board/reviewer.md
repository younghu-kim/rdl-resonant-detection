**과대 해석 경고**: 설계자가 "P3(κδ²) → RH 감별기 ★★★"라고 결론했으나, 이것은 **유한-δ 효과**이다.

이론적 분석:
- 단순 영점 s₀ 근방: Λ(s) ≈ c(s-s₀) + c₂(s-s₀)² + ...
- Λ'/Λ at s₀+δ = 1/δ + a₁ + a₂δ + ... (여기서 a₁ = Λ''/(2Λ'))
- κδ² = 1 + 2Re(a₁)·δ + O(δ²)
- **δ→0에서 κδ²→1은 on/off 무관하게 모든 단순 영점에서 성립**

δ=0.03에서의 차이(0.002 vs 0.21)는 **a₁ = Λ''/(2Λ')의 실수부 차이**를 반영:
- On-critical: Re(a₁) ≈ 0.03 (FE 대칭에 의한 부분 상쇄)
- Off-critical: Re(a₁) ≈ 3.5 (상쇄 없음, Γ-인수 기여 현저)

**결론**: κδ²는 δ→0 극한에서 감별하지 못한다. 유한 δ에서 차이가 나는 것은 local zero geometry의 차이를 반영하지만 "RH 감별기"라는 표현은 과대. **"유한-δ에서 on/off-critical zero의 국소 기하 차이를 감지"**가 정확한 서술.

설계자의 공식 검증:

- OFF#3 (t=166.5): 예측 0.133, 실측 0.373 (**불일치, 추가 항 존재**)
→ 부분적으로만 설명 가능. 완전한 이론적 예측 공식 부재.

#### 오류의 영향 분석

이 오류들은 **개별 측정값에 영향을 주지 않는다** — PARI는 실제 곡선으로부터 올바르게 계산한다.

**수정된 곡선 분류**:
- rank 0: 11a1 (N=11), "197a1" (실제 N=3099) — **2곡선만** (3곡선이 아님)

- rank 3: 5077a1 — 1곡선

**conductor 통제**: rank 1에서 N=37,43,53,79 (4곡선) → 여전히 유효.
rank 0에서 N=11,3099 (2곡선) → 가용하지만 N 차이가 크다 (11 vs 3099).
