#### Red Team 분석

1. **Sym²(11a1) 2H₁/A_L = 0.748 — 이상값**: d=1~6 중 가장 높음 (다른 모두 ~0.66).
   - Sym²(37a1)은 0.678 (정상 범위) → GL(3) 전체가 아닌 **11a1 특이적**
   - 가능 원인: N=121 (상대적 소 conductor) → smooth part B_smooth 효과 약함 → A_L이 A_bare에 가까움
   - mean_A_L(11a1) = 152 vs mean_A_L(37a1) = 46.5: 스케일 차이 3×
   - **결론**: 계통적 오류 아닌 물리적 특이점. 논문에 이미 명시됨 (Obs 5.3).

2. **Sym²(37a1) n=48 소표본**: p=3.8e-04로 유의하지만, 다른 L-함수(n=90~337)에 비해 작음.
   - 소표본 보정 없이도 충분히 유의 (p < 0.001)
   - Bootstrap 교차검증은 불필요 (Spearman은 분포-무관)

3. **damping 값**: 0.365 (11a1), 0.383 (37a1)
   - d=2: 0.43~0.53, d=4: 0.34, d=5: 0.37, d=6: 0.43
   - d=3이 d=2보다 낮고 d=4와 비슷 → **단조감소가 아닌 비선형 패턴**
   - B-48 경계에 새 데이터 포인트 제공. 수학자 해석 필요.

4. **gap_right 상관**: 양 곡선 모두 A_bare × gap_right 비유의 (p>0.4) → **gap_min이 canonical** 패턴 d=3에서도 확인.

5. **precision 다운그레이드 (100→38)**: 11a1 Sym²에서 precision=100이 PARI 스택 오버플로우 유발.
   - 해결: precision=38 (C-264와 동일)
   - **위험**: 저정밀도로 인한 수치 편향?
   - **판단**: mpmath digamma는 독립 정밀도 제어. PARI precision은 영점 위치에만 영향. 영점은 충분히 정확 (37a1과 일관된 결과 패턴). 위험 낮음.
