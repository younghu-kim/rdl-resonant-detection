### ★★★ 치명적 오류: Dirichlet 계수 공식이 잘못됨

**스크립트가 계산하는 것**: L(s,Δ)² = (Σ τ(n)/n^s)²
- 계수: a_n = Σ_{d|n} τ(d)·τ(n/d) (Dirichlet convolution = τ*τ)
- a(2) = 2τ(2) = **-48** (스크립트 출력 일치)

**계산해야 하는 것**: L(s, Δ⊗Δ) = Rankin-Selberg tensor product
- 계수: 소수에서 a_p = τ(p)² (Satake parameter 텐서곱)
- a(2) = τ(2)² = (-24)² = **576**

**검증**:
```
L(s,Δ)²:     a(2) = -48,   a(3) = 504,    a(5) = 9660
L(s,Δ⊗Δ):   a(2) = 576,   a(3) = 63504,  a(5) = 23328900
                    ↑ 완전히 다른 함수 ↑
```

### 결과적 파급

1. **함수 정체**: 스크립트는 L(s,Δ)² (degree 2의 제곱)을 잘못된 감마 인자와 결합한 **프랑켄슈타인 함수**를 계산
   - L(s,Δ)²의 영점 = L(s,Δ)의 영점 (중복도 2) — 새로운 degree 4 정보 없음
   - 감마 인자 (11,11,12,12)도 L(s,Δ)²용이 아님 → 함수 전체가 부정확

2. **57개 "영점"**: 진짜 L-함수 영점이 아닌 인공물
   - L(s,Δ)의 Weyl 추정: [2,50]에서 ~17개 영점
   - 감마 위상 변화: t∈[5,50]에서 Im[log γ] 변화 ~133.6 → ~21회 추가 진동
   - 57 ≈ 17(진짜) + 40(감마 위상 인공물) 정도의 구성

3. **κ_near(d=4) = 1125.69**: 무의미
   - 진짜 degree 4 L-함수의 영점이 아님
   - B-12 해결에 사용 불가
   - κ_near(d) 4점 비교표 무효

4. **κ ratio = 3.7×**: 밀도 포화로 의미 없음 (설계자도 지적)

5. **σ-유일성 "PASS"**: σ=0.3,0.5,0.7 모두 49회로 동률 — 실질적으로 판별 불가

### 추가 우려: FE 검증 자명성

설계자가 정확히 지적: AFE 구조에서 ε=1일 때 Λ(s) = Λ(1-s)가 공식상 자동 성립.
→ FE rel=0은 L-함수 성질이 아닌 AFE 구성의 결과.
→ Dirichlet 계수/감마 인자 오류를 **감지할 수 없음**.
