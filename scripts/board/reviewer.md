#### 2. ξ-bundle A(t₀) ❌ 기술적 실패 — 원인 분석

**증상**: κ ≈ 13~21 (δ에 무관). 정상값은 κ ≈ 1/δ² (예: δ=0.001 → κ≈10⁶).

**원인 진단**:
- `lfuninit(L3, [0, 12])` 사용 → `lfunlambda(L3i, s)` 계산
- s = 2.0 + δ + i·t₀ 에서 σ=2.001~2.1 (center 우측)
- κ가 δ-독립 → `lfunlambda`가 Λ(s)를 정확히 계산하지 못하고 있음
- **가능 원인**: conductor N=1369에서 lfuninit 범위/정밀도 부족
  - GL(5) N=14641에서는 정상 작동 → N 자체보다 gamma factor 조합 문제 가능
  - gammaV=[-1,0,1]의 motivic center=2에서 σ=2+δ → 감마 인자 영역 특수성?

**반례 탐색**: sym²(11a1)의 ξ-bundle (#74)은 mpmath로 계산, PARI lfunlambda가 아님. 따라서 두 방법의 정합성이 검증되지 않았을 가능성.

**결론**: ξ-bundle A 값(-175482)은 무효. 논문에 사용 불가. conductor 비교도 불가.

- **원인**: ξ-bundle κ 계산 자체가 잘못되어 σ-유일성도 무효
- κ가 영점 근방의 발산 (1/δ²)을 포착하지 못하므로, "σ=center에서 최대" 판정도 무의미

#### 4. 과대 해석 점검

- ✅ "d=3 보편성 통계적 강화": 4성질 기준으로는 타당. sym²(11a1)과 sym²(37a1) 두 독립 L-함수에서 모두 P1-P4 통과.

- ❌ "conductor 효과 1372132%": ξ-bundle 계산 오류에 의한 허위 수치. 논문 기재 불가.
