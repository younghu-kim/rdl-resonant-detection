- 근거: "realprecision=100에서 FE=1.0 반환 (오작동), 38에서 FE=−51 정상"

**4. κδ² slope 개별 일관성**

- **이것은 degree-4에서 처음이며, 역대 최고 수준과 동률.**

**5. σ-유일성 FAIL 분석**
- L(1/2) ≈ 0 → root number = −1 (스크립트에서 확인)
- root number = −1이면 L(s)가 center에서 강제 영점 → σ=center 근방에서 부호변환 감소 예측
- σ=0.50에서 24회, σ=0.40에서 30회: center보다 약간 벗어난 σ에서 최대
- **B-01 구조적 패턴 + root number 효과. GL(2) 사례와 일관.**
- 📌 논문에 "root number ε=−1 induces a forced zero at center, reducing sign changes" 명시 권장.

**6. sym³ Euler 계수 검증**
- 정규화: t_p = τ(p)/p^(11/2)
- e1 = tp³ − 2tp, e2 = (tp²−2)(tp²−1): 표준 sym³ Euler 다항식 재귀
- 4-term recurrence: c(p,k) = e1·c(p,k-1) − e2·c(p,k-2) + e1·c(p,k-3) − c(p,k-4)
- FE=−51 통과가 계수 정확성의 강력한 간접 증거.
- τ(2)=−24, τ(3)=252, τ(5)=4830: 표준값 ✅

**7. 잠재적 반례/약점**
- 단일 GL(4) L-함수(sym³(Δ))만 테스트. conductor=1로 최소 난이도.
- sym³ 구조 → 이미 GL(2)에서 파생. 진정한 "독립" GL(4)는 아닌가?
  → 반론: sym³(Δ)는 Selberg class에서 자기동형(automorphic) degree-4 L-함수로 인정됨. LMFDB에서도 degree 4로 분류.
- conductor 독립성 미확인 (N=1만). 향후 sym³(11a1) 등 N>1 테스트 가치 있음.
