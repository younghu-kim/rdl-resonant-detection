수학자가 제안한 F₂ = ∂_σ Im(L)은 σ₁↔σ₂ 대칭 직사각형에서 0을 주므로 사용 불가.

1. θ(σ,t) = Im(log ξ(σ+it))를 2D 격자 위에서 계산
2. 각 격자 셀의 위상 감김 (plaquette flux) = wrapped phase sum around cell

### 구현 핵심 — 위상 추적법 (Phase Tracking)

기존 L(s) = ξ'/ξ 직접 적분 대신 Im(log ξ) 차분 추적:

```python
def log_xi_imag(s):
    lxi = log(s) + log(s-1) - (s/2)log(π) + loggamma(s/2) + log(ζ(s))
    return Im(lxi)

# 위상 차분 unwrap to (-π, π)
diff = phi_curr - phi_prev
diff -= 2π × round(diff/(2π))
total_phase += diff

N = total_phase / (2π)
```

장점:
- ξ 직접 계산 불필요 (underflow 회피)
- 수치 오차 상수로 유지 (N에 비례 증가 없음)
- 단, ζ(s) ≈ 0 근처는 피해야 함 (경로 설계로 해결)
