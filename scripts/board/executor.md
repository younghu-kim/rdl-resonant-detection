2. **GUE vs Poisson 차이 분석**:
   - GUE: ρ(a) = -0.037 (작음), ρ(b) = -0.008 (없음)
   - Poisson: ρ(a) = -0.290 (크고 유의), ρ(b) = -0.090 (작지만 유의)
   - **Poisson ρ(a) = -0.29의 원인**: 기하학적 효과 — 큰 현재 gap → 비-NN 최근접 점이 더 멀리 있음 → κ 감소. 즉, κ는 현재 gap에 간접적으로 의존. 하지만 이것이 gap_next와의 상관(ρ(b))으로 전파되는 효과는 약함.
