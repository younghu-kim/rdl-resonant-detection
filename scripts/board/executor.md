### 수학자에게: GL(1) 4/5 FAIL 원인 분석

**χ₃만 PASS하고 나머지 GL(1)이 FAIL한 이유**:

1. **ζ (N_crit=3)**: t ∈ [1, 30]에서 영점이 3개뿐. Off-critical에서도 이 3개 영점이 "bleeding through" → N_off=3=N_crit. **t 범위 확장 시 PASS 예상** (T=1000에서 N_crit~230 >> N_off~86 예상).

2. **χ₄, χ₅, χ₇**: Re(Λ) 사용. 이 경우 off-critical에서 σ 근처 영점이 Re(Λ)에도 부호 변화를 일으켜 N_off ≈ N_crit. **측정 방법(Re vs Im) 의존성**.

3. **χ₃만 PASS하는 이유**: ε=-1이므로 Im(Λ) 사용. Im(Λ(1/2+it))는 영점에서만 정확히 0이고, off-critical Im(Λ(0.6+it))는 영점의 Im 기여가 약해짐. **Stirling R=0.37과 정확히 일치 (8/21=0.38)**.

4. **결론**: 이 측정 방법은 Im(Λ) ε=-1 경우에서 Stirling 예측을 정확히 확인. 전체 GL(1)에 대한 통일적 검증을 원한다면 **Im(Λ) vs |Λ|의 로컬 최솟값** 방법이 더 적합.

### Stirling 공식 적용 시 주의사항

- GL(1) Stirling zero density formula `(1/2π)log(t/2π)`는 ζ에 특화. 디리클레 χ_q의 경우 `(1/π)log(qt/2π)` 사용 필요
- χ₃의 경우: `(1/π)log(3t/2π)` → t=15에서 zero density ≈ 0.70/unit → [1,30]에서 ~18-25 영점 (경험 21개와 일치)

- → GL(1) Dirichlet는 도체가 클수록 R이 작아져 σ-유일성 PASS가 더 명확
