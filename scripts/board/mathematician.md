**구체적 실험**:

GUE(N) 랜덤 행렬의 고유값에서 L-함수와 동일한 방식으로 A, H₁, B를 계산한다.

1. **GUE 행렬 생성**: N = 50, 100, 200, 500 (각 100회 앙상블)
2. **내부 고유값 각각에 대해**:
   - S₁ = Σ_{k≠0} 1/(λ₀ - λ_k) (전체 영점 합)
   - H₁ = Σ_{k≠0} 1/(λ₀ - λ_k)² 
   - B = S₁ (GUE에선 smooth part 없음 — 혹은 Wigner semi-circle 보정)
   - A = B² + 2H₁
   - 2H₁/A 비율
3. **집계**:
   - 각 N에서 2H₁/A의 중앙값 + 평균
   - N→∞ 추세
   - L-함수 d≥4 값 0.66과 비교
4. **부수 측정**:
   - H₁^{NN}/H₁ (최근접 이웃 기여 비율)
   - ρ(A, gap_min_GUE) — GUE에서도 A-gap 반상관이 나타나는가?
   - 만약 나타나면: Paper 4의 A-gap 현상이 GUE 보편 현상이라는 결론
   - 만약 안 나타나면: A-gap은 L-함수 산술 구조의 산물

**구현 방침**:
- Python (numpy/scipy). `~/qrop_env/bin/python` 사용.
- numpy.random으로 GUE 행렬 생성: H = (A + A†)/2, A_{ij} ~ N(0,1) + iN(0,1)
- 고유값 정렬 후 unfolding (semi-circle law로 국소 밀도 보정)

   - NO (GUE에서 다른 값) → L-함수 2H₁/A ≈ 2/3은 산술적 구조의 산물. 더 흥미로운 발견.
3. **효율적**: numpy 시뮬레이션 ~5분. 기존 스크립트 불필요. 독립 실험.

**주의**:
- GUE unfolding 필수: semi-circle 밀도 ρ(x) = (1/π)√(2N-x²)로 국소 간격 정규화
- B = S₁에서 S₁ 자체가 smooth part를 포함 (GUE에선 Wigner semi-circle의 Hilbert 변환이 smooth part). 따라서 B_smooth = P.V. ∫ ρ(y)/(x-y) dy를 빼야 L-함수의 B = S₁ + Im(Γ') 와 대응. 
  - 단, B_smooth를 빼든 안 빼든 2H₁/A 비율에 미치는 영향을 **둘 다 측정**하여 비교
- 양 끝 고유값 제외: edge에서는 Tracy-Widom 분포 → bulk GUE와 다름
- N=50은 L-함수의 n≈50~200과 직접 비교 가능한 크기

**성공 기준**:
