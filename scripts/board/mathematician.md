### 다음 작업: **χ₃ Chern 수 수정 + 디리클레 보편성 완성**

**모델**: sonnet

**왜**: 결과 #22의 유일한 미완성 부분. 코드 버그 수정이므로 단순 작업. χ₃에서 Chern 수 일치를 확인하면 "ξ-다발 Chern 수 해석"의 보편성이 확립되어, 논문의 이론적 앵커가 완성된다.

**구체 설계**:

1. **χ₃ 영점 탐색 수정**:
   - 현재: Re(L(1/2+it, χ₃))=0 부호변화 → 잘못됨
   - 수정: |L(1/2+it, χ₃)|를 0.01 간격으로 스캔, 극소점(|L|<0.1)에서 mpmath.findroot으로 정밀화
   - 또는 Hardy Z-함수 유사체: Z(t) = e^{iθ(t)} L(1/2+it) 가 실수가 되는 θ(t)를 계산하여 부호변화 탐색
   - 어느 방법이든 무방. 핵심은 **|L|=0인 점**을 찾는 것.

2. **수정된 영점으로 직사각형 위상추적 재실행**:
   - n = 5, 10, 15, 20 (이전과 동일 구간)
   - 기대: |N_ctr - N_actual| < 0.1

3. **χ₄ 추가 검증** (실수 짝수 지표):
   - χ₄ = [0,0,0,1]은 짝수 지표(χ(-1)=+1) → L(1/2+it)은 실수
   - 기존 Re(L)=0 방식이 올바르게 작동해야 함
   - n = 5, 10, 15, 20에서 위상추적 검증

4. **출력**: `results/holonomy_chern_dirichlet.txt`
   - χ₃ 수정 영점 목록
   - χ₃ 직사각형 위상추적 표 (n, t₂, N_actual, N_ctr, |차이|)
   - χ₄ 직사각형 위상추적 표
   - χ₃+χ₄ 종합 통과율

**주의**:
- `scripts/holonomy_chern_number.py`를 기반으로 수정. 새 파일 작성 권장.
- χ₃ 위상추적 자체는 이미 올바르게 작동 중 (N_ctr가 정답에 가까움). 문제는 N_actual 계산뿐.
- Λ'/Λ 해석적 공식은 기존 코드(`dirichlet_midpoint_nonlocal.py` 등)에서 검증 완료.
- dps=100, 수평분할 64(또는 128), σ∈[0.3, 0.7].
- 윤곽이 영점을 피하도록 t₂는 영점 간 중간점에 배치.
- **홀수 디리클레 지표에서 Re(L)=0 ≠ L=0**. 이번에 이 함정을 영구적으로 해결.

**성공 기준**:
- χ₃: 4/4 |N_ctr - N_actual| < 0.1
- χ₄: 4/4 |N_ctr - N_actual| < 0.1
- **양성**: 8/8 통과 → 디리클레 보편성 확립, 결과 #22 완전 양성으로 승격
