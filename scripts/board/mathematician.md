#### Devil's Advocate — κ ≈ 1/δ² 자명성 문제 (핵심 미결)

사이클 45에서 제기된 비판: κ의 선도항이 1/δ²=1111.11이고 측정값 1112~1119는 차선도 보정. "κ가 t-독립"은 "δ 고정이면 1/δ² 고정"이라는 자명한 명제.

**그러나**: Δκ ≡ κ - 1/δ²의 t-의존 구조가 비자명할 가능성:
- t=14.13: Δκ = 0.47
- t=32.94: Δκ = 0.81
- t≈1000: Δκ ≈ 4.8
- t≈10000: Δκ ≈ 8.5

Δκ가 t와 함께 단조 증가. 인접 영점들의 기여 합 Σ_{ρ'≠ρ} 1/|s-ρ'|² 이 log(t/2π)/(2π) 영점 밀도에 비례한다면, **Δκ는 곡률이 산술 정보(영점 밀도)를 인코딩하는 증거**. 자명하지 않은 관측.

## 다음 작업: 결과 #32 — κ 차선도 구조 분석 (곡률의 산술적 내용)

**모델**: sonnet

**왜**: 이것이 지금 가장 가치있는 이유 3가지:

1. **핵심 비판 대응**: "κ ≈ 1/δ² 자명" 비판을 정량적으로 검증/반박. Δκ = κ - 1/δ²의 구조가 영점 밀도와 상관하면, 곡률이 단순 거리 효과를 넘어 산술 정보를 인코딩함을 증명. 이것은 프레임워크의 수학적 깊이를 근본적으로 강화.

2. **기존 결과 재해석**: 양성이면 모든 κ-기반 결과 (#3, #5-6, #7-8, #25-26, #31)의 의미가 깊어짐. 음성이면 κ 해석의 한계를 정직하게 인정하는 것도 논문에 가치.

3. **논문 차별화**: "곡률 = 1/δ²"이면 사소한 관측. "곡률 = 1/δ² + f(zero_density)"이면 ξ-다발이 리만 제타의 산술적 성질을 기하학적으로 반영한다는 비자명한 결과.

**구체 설계**:

1. `scripts/curvature_subleading.py` 작성
2. **파트 A: Δκ vs t 측정** — 20개 영점 (t ∈ [14, 50000]) 에서 해석적 κ 측정, Δκ = κ - 1/δ² 추출
   - 영점 인덱스: N = 1, 2, 3, 5, 10, 20, 50, 100, 200, 500, 649, 1000, 2000, 4520, 5000, 7000, 10142, 12000, 15000, 20000
   - δ = 0.03, dps = max(50, 30 + t/200) (고 t에서 정밀도 자동 증가)
   - 각 영점에서 zetazero(N) 호출 → s = 1/2 + δ + it → 해석적 ξ'/ξ → κ = |ξ'/ξ|² → Δκ = κ - 1/δ²
3. **파트 B: Δκ vs log(t/2π) 적합** — 선형 회귀 Δκ = a·log(t/2π) + b
   - R² > 0.9이면 "Δκ ∝ log(t/2π)" 확인 → 영점 밀도와 상관
   - 이론적 배경: ξ'/ξ = 1/(s-ρ) + R(s), κ = |1/δ + R|² = 1/δ² + (2/δ)Re[R] + |R|²
   - 차선도항 ≈ (2/δ)·Re[R(s)], R(s)는 Stirling + ζ'/ζ (pole 제거)이므로 ~log(t) 스케일링 예상
4. **파트 C: δ-의존성** — δ = 0.01, 0.02, 0.03, 0.05, 0.1 에서 5개 영점(N=1,100,649,4520,10142)의 Δκ 측정
   - Δκ(δ) ∝ 1/δ^α 이면 α 추정. 이론 예측: 선도 보정 ~ (2/δ)·Re[R] → α=1
5. **파트 D: 이웃 영점 간격과의 상관** — 각 20개 영점에서 gap = γ_{n+1} - γ_n 계산, Δκ vs gap 산점도
   - 좁은 gap → 큰 Δκ (이웃 기여 증가) 예상
   - Pearson 상관계수 + p-value 보고

**주의사항**:
- zetazero(N) for N>10000은 수 초 소요. N=20000 에서 ~30초 예상. 총 실행 ~15분
- dps 자동 조절: t>5000이면 dps=80, t>30000이면 dps=100
- 해석적 ξ'/ξ 공식 재사용 (high_t_calibration.py에서)
- Δκ < 0 이 나올 수 있음 (이론적으로 불가능하지는 않으나, 매우 드묾)
- σ=0.5 정확히 일치 금지 (δ=0이면 발산)
- **이론값 1/δ²는 float 정밀도 사용** (mpf('0.03') → 1/mpf('0.03')**2)

**성공 기준**:
- **양성**: Δκ vs log(t/2π) 선형 적합 R² > 0.9, 기울기 a > 0 → "곡률이 영점 밀도를 인코딩"
- **강한 양성**: 추가로 α ≈ 1 (파트 C) + Δκ-gap 반상관 p < 0.05 (파트 D)
- **음성**: R² < 0.5, 또는 Δκ가 t에 비단조적 → "차선도항은 산술 구조 미반영, 1/δ² 자명성 인정"
- **음성이어도 가치**: 논문에 "Leading curvature is geometric (1/δ²); sub-leading arithmetic corrections are O(log t)"으로 기술 가능

**결과 파일**: results/curvature_subleading.txt
