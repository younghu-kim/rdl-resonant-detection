**핵심 결론**:
1. GUE에서도 국소 합 > 전체 합 (Δ=0.054). **보편 현상** — 원거리 합이 잡음 역할.

3. N 의존성: N≥1000에서 Δ 수렴 (0.052→0.054). 안정적.

**Devil's Advocate** (양성이므로 반론 3가지):
1. trim 방식 차이 (ζ: 높이 T 기반 vs GUE: 인덱스 기반) — 둘 다 edge 제거 목적이므로 비교 유효.
2. ±300은 ζ(s) T=2000에서 ~46%, GUE N=2000에서 ~15%. 비율 차이 — C-283 N 의존성이 보정.
3. n 차이 (GUE 24000 vs ζ ~400) — 결론이 "ζ ≈ GUE"이므로 차이의 유의성은 부차적.
→ 모두 방향을 뒤집지 못함. 판정 유지.
