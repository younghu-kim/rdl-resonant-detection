### 판정 A: κ-conductor 스케일링 — [음성]

- Spearman r_s = -0.31 (p = 0.54) — 유의하지 않음
- 모델 비교: 상수(SS=42204) ≈ 로그(SS=41933) ≈ 멱법칙(SS=42880) — 구별 불가

**Devil's Advocate**:
1. 53a1의 κ_far=2.41이 이상치 → FP 선정이 영점 근방이었을 수 있음 → ratio 467× 과대평가

→ **음성 확정.** 하지만 이 음성 자체가 의미 있음 — κ ratio의 변동은 κ_far 선정에 의존하는 아티팩트.

**다음 작업**: **#69 Maass form L-함수 4성질 검증**

Maass form은 비정칙(non-holomorphic) GL(2) — 지금까지의 모든 L-함수와 근본적으로 다른 유형.

대상: **첫 번째 Maass form on SL(2,Z)**, eigenvalue R = 9.53369526135...
- LMFDB: `maass/SL2Z/R9.53...`
- conductor N=1 (가장 단순)
- Ramanujan-Petersson 추측이 **미증명**인 유일한 테스트 케이스

4성질 검증:
1. σ=1/2 유일성
2. κ-피크 (κ_near 측정 → 1125와 비교)
3. 모노드로미 (mono/π = ±2)
4. FE 잔차

핵심 구현 주의:
- Fourier 계수: LMFDB API에서 수치적으로 가져와야 함
- 감마 인자: Γ_R(s+iR) · Γ_R(s-iR) where Γ_R(s) = π^(-s/2)Γ(s/2)
- #63-#68 코드를 템플릿으로, Dirichlet 계수 생성부와 감마 인자를 교체
- DPS=50, c=2.0 유지

sym²에서 Hecke 계수 계산:
- sym²(19a1) conductor N=19²=361
- sym²(23a1): 이 실험에서는 해당 없음. Maass form 자체의 L-함수를 직접 다룸.

**모델**: opus

**왜**: 
- 새 L-함수 유형 → AFE 감마 인자가 완전히 다름. opus 수준 수학 필요.
- κ_near가 1125이면 holomorphicity/weight 무관 보편상수 → Paper 2 핵심 정리.
- 다르면 L-함수 분류론적 의미 → 역시 핵심 발견.
- "모든 자기쌍대 GL(2) L-함수"로 확장하면 논문 가치 비약적 상승.

**주의**:
- LMFDB에 Maass form 영점이 있으면 블라인드 비교 가능
- Maass Fourier 계수 a(n)은 Hecke 고유값과 다름 — 정규화 확인 필수
- even/odd 구분 (even Maass form: ε=1, odd: ε=-1). R=9.533... 은 even.
- 실패 시 Epstein zeta로 전환 가능 (비정칙이지만 해석적 공식 존재)

**성공 기준**:
- FE 통과 (rel < 10⁻⁸)
- κ_near 측정 성공 (≥6 TP)
- κ_near CV < 2%
- mono/π = 2.0 (≥80% TP)
- 블라인드: Recall ≥ 0.7 (LMFDB 영점 있을 경우)
