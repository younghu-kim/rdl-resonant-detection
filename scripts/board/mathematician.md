### 다음 작업: **비국소 도달 범위 측정 — κ_mid ↔ gap_{n+k} 상관 감쇠**

**모델**: sonnet

**왜**: 기존 curvature_midpoint_nonlocal_v2.py의 구조를 재사용, gap_next 대신 gap_{n+k} (k=1..10)에 대해 반복 측정. 알고리즘 신규 구현이 아닌 파라미터 확장이므로 sonnet 충분.

**구체 설계**:

1. **데이터**: 기존과 동일 — ζ 영점 235개 (t∈[0,450]), 중간점 κ 234개
2. **측정**: 각 중간점 m_n = (γ_n + γ_{n+1})/2 에 대해:
   - gap_{n+k} = γ_{n+k+1} - γ_{n+k}, k=0,1,2,...,10
   - k=0은 gap (NN 상쇄 확인용), k=1은 gap_next (기존 #18 재현)
   - Spearman ρ(κ_mid_n, gap_{n+k}) 각 k에 대해 계산
   - 편상관(gap 제어)도 같이 계산
3. **역방향도 측정**: gap_{n-k} (k=1..10) — 과거 방향으로도 비국소 신호 존재?
   - 대칭이면: 시간 반전 대칭 (함수방정식의 반영?)
   - 비대칭이면: 비국소 신호에 방향성 존재
4. **디리클레도 같이**: χ₃에서 동일 분석 (보편성 확인). 가능하면 χ₄, χ₅도.
5. **출력**: `results/nonlocal_reach_decay.txt`
   - 표: k | ρ(forward) | p | ρ(backward) | p | partial_corr
   - 감쇠 피팅: |ρ(k)| ~ A·exp(-k/ξ) 또는 |ρ(k)| ~ A·k^{-α} (지수 vs 멱법칙)
   - Figure 데이터 (k, ρ 쌍) CSV도 함께 저장

**주의**:
- κ 계산은 기존 해석적 공식 (log(ξ)' = 1/s + 1/(s-1) - log(π)/2 + ψ(s/2)/2 + ζ'/ζ) 사용. dps=100.
- NN 상쇄: κ = |Σ' 1/(m-ρ)|² (NN 두 항 제외). 기존 코드 그대로.
- k=10까지만 (n=234이면 유효 데이터 224개까지 확보 가능).
- Dirichlet κ는 Λ'/Λ 해석적 공식 사용. bare L'/L 금지.
- gap_{n+k} 인덱스 범위 초과 주의 (n+k+1 ≤ 234).

**성공 기준**:
- k=1에서 |ρ| > 0.5 (기존 #18 재현)
- k=2 이상에서 감쇠 패턴 관측 (단조 감소 또는 진동 감쇠)
- 감쇠 모델 피팅 R² > 0.8
- 전방/후방 비교 결과 기록
- **양성**: 특성 길이 ξ > 1 (2개 이상 이웃까지 유의미한 상관)
- **음성**: k=2에서 이미 |ρ| < 0.1 (nearest-neighbor+1 고유, 장거리 구조 없음)
- **어느 쪽이든 논문에 Figure로 삽입 가능 → 실험 가치 보장**

### 알려진 함정 (누적)

- 모노드로미: eps 차분 금지 → 폐곡선 적분(radius=0.5, 64단계)

- κ 측정: 영점 위 직접 측정 금지 → δ=0.03~0.05 오프셋
- findroot: 단일 시작점만, 튜플 전달 금지
- numpy 2.0: np.trapezoid (trapz 아님)
- Spearman 상관분석 시 해상도 확보 필수 (FWHM은 step < 피크폭/3 필요)
- Hadamard 부분합에서 |F₂| 잔차는 보정 상수 B 추정 필요
- **bundle_utils.py h=10^{-20}**: t>100에서 ξ ~10^{30-40} → 수치미분 유효자릿수 소진. dps ≥ 100 또는 해석적 공식 사용.
- **중간점 Hadamard 상쇄**: m=(γₙ+γₙ₊₁)/2에서 두 최근접 영점의 L(s) 기여 = 0.
- **κ_nn 정의**: |Σ 1/(s-ρ)|² ≠ Σ |1/(s-ρ)|². 전자가 올바른 물리적 정의.
- **ξ(s) underflow**: log(ξ)' 해석적 공식으로 영구 해결. ξ 직접 계산 불필요.
- **mpmath.diff(xi, s)**: t>300에서 실패. 사용 금지.
- **상관 분석 시 편상관 필수**: 인접 간격 자기상관(ρ≈-0.15) 존재.
- **d_second vs gap_next 구분**: κ는 d_second가 아닌 gap_next에 직접 민감.
- **Dirichlet κ 계산**: bare L'/L 사용 금지. Λ'/Λ 해석적 공식 필수.
- **Dirichlet 영점 탐색 underflow**: 적응형 tol 필수.
- **GUE κ 비교 시 unfolding 필수**: sliding window 또는 semicircle 밀도.
- **Poisson에서 NN 상쇄 미작동**: NN 명시적 제외 필수.
