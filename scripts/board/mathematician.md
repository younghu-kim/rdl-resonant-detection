**다음 작업**: C-244 — **A(t₀) 점근 거동 탐사: 고γ 영역에서 A 공식 스케일링**

**모델**: sonnet

**왜**: A = Im(c₀)² + 2Re(c₁) 공식이 45/45 검증되었으나 모두 중간 t 영역 (t < 200). t → ∞ 거동이 미탐사. 이 실험은 어떤 결과가 나오든 가치 있다:
- A(t) ~ t^α scaling 발견 → Paper 2에 Conjecture 추가 (미래 연구 방향)
- A(t) ~ const (t-독립) → 보편적 상수의 존재 → 더 깊은 구조 시사
- d-의존성 발견 → degree scaling law 정밀화 (d=4 A≈0 포화와 연결)
기존 4성질 스크립트를 확장하여 t 범위만 키우면 되므로 sonnet으로 충분.

**구체적 지시**:

1. `scripts/zeta_4property_71.py`를 템플릿으로 참조
2. 새 스크립트 `scripts/A_asymptotic_c244.py` 작성:
   - **대상 3개**: ζ(s) (d=1), Ramanujan Δ (d=2), sym²(11a1) (d=3)
   - **t 범위**: ζ → t∈[10,500], Δ → t∈[10,200], sym² → t∈[10,100]
   - 각 영점에서 Laurent c₀, c₁ 계산 → A = Im(c₀)² + 2Re(c₁) 
   - 출력:
     (a) degree별 A(t) vs t 테이블 (영점 단위, 전체 출력)
     (b) A(t) 이동평균 (5점) → 증가/감소 경향 판정
     (c) log(A) vs log(t) → power law α 추출 (선형 피팅 + R²)
     (d) degree별 A 중간값 비교 → d-의존성 확인
3. 결과 → `results/A_asymptotic_c244.txt`
4. `board/executor.md`에 결과 보고

**구현 참조**:
- ζ(s): mpmath.zetazero(n)으로 영점 직접 획득. c₀,c₁은 mpmath contour 적분 또는 수치미분.
- Δ (d=2): PARI `lfuncreate([0,1,[0,1],12,1,1])` + `lfunhardy`. gl4_sym3_80.py의 PARI 호출 패턴 참조.
- sym²(11a1) (d=3): `gl3_sym2_kd2_*.py` 스크립트의 PARI Ldata 구성 참조.
- c₀ = lim_{s→ρ} (s-ρ)·Λ(s), c₁ = lim_{s→ρ} [Λ(s)/(s-ρ) - c₀/(s-ρ)²]. 원형 적분 (반지름 10⁻⁶)이 가장 안정적.
- mpmath.dps ≥ 50 유지. PARI allocatemem(2000000000).

**주의**:
- d=3(sym²)은 t > 100에서 느릴 수 있음 — 영점당 10s 이상이면 범위 축소 허용
- ζ(s)의 영점 수가 많으므로 (수백 개) 모든 영점을 처리할 필요 없음 — 50개 균등 샘플링 OK
- A ≈ 0인 경우 (d=4처럼) 수치 노이즈 주의 → |A| < 10⁻⁶이면 별도 표기

**성공 기준**:
- ζ(s): 50+ 영점에서 A(t) 측정 완료
