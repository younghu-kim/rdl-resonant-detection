**모델**: opus

**왜**: E_diag α=1은 Hadamard 곱에서 해석적으로 자명 (각 Lorentzian의 적분 = π/δ, 합 = πN/δ).
진짜 질문은 교차항이 α=1을 보존하는지, 그리고 그 메커니즘이 GUE pair correlation인지이다.
이것이 확인되면 "수치적 관찰"이 "조건부 정리 (GUE 가정 하)"로 승격된다.
B-61은 "후속 논문 씨앗"으로 분류되었지만, 실제로는 Prop 6의 이론적 근거를 완성하는 핵심 조각이다.

**구체적 실험 설계**:

ζ(s)의 처음 ~400개 영점을 사용하여 (가장 밀도 높은 데이터):

1. **E_diag 해석적 검증**:
   - E_diag(σ) = Σ_{n=1}^{N} π/(σ-1/2) (이론값) vs 수치 적분 비교
   - A_diag/πN 비율 계산 → 1에 수렴하는지 확인 (경계 효과 보정 포함)

2. **교차항 쌍별 분해**:
   - E_cross(σ) = E_full(σ) - E_diag(σ) 계산
   - 교차항을 쌍 간격 Δ_k = |γ_i - γ_j| 별로 그룹화
   - C(Δ, σ) = Σ_{|γ_i-γ_j|≈Δ} Re[∫ dt/((s-ρ_i)(s̄-ρ̄_j))] 계산
   - C(Δ, σ)의 Δ-의존성 + σ-의존성 프로파일 추출

3. **GUE 비교**:
   - 영점 간격을 평균 간격으로 정규화: x_k = Δ_k / (2π/log(γ_k/2π))
   - 정규화된 간격의 pair correlation histogram 추출
   - R₂_empirical(x) vs R₂_GUE(x) = 1 - (sin πx / πx)² 비교
   - 교차항 기여도와 R₂ 편차의 상관 분석

4. **교차항 스케일링**:
   - E_cross(σ) ~ B/|σ-1/2|^β 에서 β 측정
   - β < 1이면: 교차항은 E_diag보다 약해서 α=1 보존
   - β = 1이면: 교차항이 α=1 유지하되 A를 수정 (A ≠ πN)
   - β > 1이면: 문제 (이 경우는 기존 수치와 모순이므로 가능성 낮음)

**출력 파일**: `results/cross_term_gue_c311.txt`

**성공 기준**:
1. E_diag = πN/δ 이론값과 수치값 차이 < 2%
2. E_cross β 값 추출 (오차 포함)
3. R₂ empirical vs GUE 비교 (KS test 또는 χ² p-value)
4. 교차항 부호 전환점 Δσ_cross와 R₂ 편차의 관계 규명

**주의**:
- Hadamard 곱 표현 사용 시 1/ρ 항 (상수 배경) 처리 주의. 큰 |ρ|에서 1/ρ는 무시 가능하지만 작은 γ에서는 기여.
- 적분 구간 [0,T]의 경계 효과: γ≈0 또는 γ≈T 근처 영점의 Lorentzian이 잘림. 경계 보정 필요.
- 기존 bundle_utils의 Hadamard 분해 함수 활용 가능 여부 확인.
- GL(4) 교차항 부호 전환(Δσ≈0.05)과 GL(2) 교차항(Δσ≈0.10)의 차이가 GUE 통계에서 설명되는지도 점검.
