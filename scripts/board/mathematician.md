### 다음 작업: **홀로노미 적분 — 영점 개수 = 제1 Chern 수**

**모델**: sonnet

**왜**: 21개 결과가 모두 ξ-다발의 개별 성질(곡률, 모노드로미, 접속)을 측정했으나, 다발의 **전역적 위상 불변량**과 영점 개수의 관계는 미검증. 이것은 논문의 이론적 앵커가 될 수 있다.

논증 원리(Argument Principle)를 다발 언어로 재서술:
- L = (log ξ)' 는 접속 1-form
- ξ의 단순 영점 ρ_n에서 L의 residue = 1
- 직사각형 윤곽 C: σ∈[σ₁,σ₂], t∈[t₁,t₂] 에서 ∮_C L(s) ds = 2πi × N(C)
- 이것이 제1 Chern 수 c₁ = N(C)

**수학적으로 자명하지만 수치 검증이 가치있는 이유**:
1. L 해석적 공식의 정합성 종합 검증 (모든 이전 실험의 기반)
2. 논문에서 "N(T) = c₁(ξ-bundle)" 명제를 수치 증거와 함께 서술 가능
3. 개별 영점 모노드로미(#7, ±π)에서 전역 불변량으로의 자연스러운 확장
4. Figure: N(T) vs (1/2πi)∮ L ds 의 계단함수 일치 → 즉시 삽입 가능

**구체 설계**:

1. **개별 영점 홀로노미** (sanity check):
   - ζ 첫 20개 영점 각각에 대해, 반경 r=0.3 원형 윤곽에서 ∮ L ds / (2πi) 계산

   - dps=100, 윤곽 64분할

2. **직사각형 윤곽 적분** (핵심):

   - t₂ = 영점 사이 중간점에서 선택, n = 10, 20, 50, 100, 150, 200
   - (1/2πi)∮_C L(s) ds 계산 → 정수 N 기대

   - 계단함수: 영점 통과 시 +1 점프

5. **출력**: `results/holonomy_chern_number.txt`
   - 개별 영점 홀로노미 표 (γ_n, ∮/(2πi), |오차|)
   - 직사각형 윤곽 표 (t₂, N_contour, N_actual, |차이|)
   - 연속 누적 CSV (t, C(t), N(t)) → Figure 데이터

**주의**:
