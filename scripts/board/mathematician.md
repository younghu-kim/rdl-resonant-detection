**다음 작업**: C-393 — Adjacent Amplitude Correlation Covariance Decomposition

Paper 4 Observation 1: ρ(A_n, A_{n+1}) ∈ [+0.28, +0.53] (13 L-함수 전부 양성).
이를 증명하려면 Cov(A_n, A_{n+1}) > 0 을 보여야 함. 핵심: Hadamard 분해에서 **어떤 항이 공분산을 지배하는가?**

### 수학적 구조

A(γ_n) = B(γ_n)² + 2H₁(γ_n), where H₁(γ_n) = Σ_{k≠n} 1/(γ_n - γ_k)²

인접 쌍 (n, n+1)에서:
- H₁(n) = **1/g²** + 1/Δ_L(n)² + H₁^{tail}(n)
- H₁(n+1) = **1/g²** + 1/Δ_R(n+1)² + H₁^{tail}(n+1)

여기서 g = γ_{n+1} - γ_n (공유 간격), **1/g²가 양쪽에 동일하게 나타남**.

### 실험 명세

ζ(s)의 T=2000 영점 (C-297 인프라 활용)에서:

**Step 1**: 각 영점 γ_n에 대해 4-way 분해 계산
```
H1_shared(n) = 1/g(n,n+1)²          # 오른쪽 NN (인접 쌍의 공유 항)
H1_left(n) = 1/Δ_L(n)²             # 왼쪽 NN (비공유)
H1_tail(n) = Σ_{|k-n|≥2} 1/(γ_n-γ_k)²  # 원거리
B_sq(n) = B(γ_n)²                    # 교대합 제곱
```
→ A(n) = B_sq(n) + 2·(H1_shared(n) + H1_left(n) + H1_tail(n))

**Step 2**: 인접 쌍별 공분산 분해
```
A'(n) = A(n) - 2/g(n,n+1)²          # 공유항 제거
ρ_full = ρ(A(n), A(n+1))            # 원래 인접 상관
ρ_residual = ρ(A'(n), A'(n+1))      # 공유항 제거 후 잔차 상관
```

**Step 3**: 잔차 상관의 원천 분해
```
ρ(H1_tail(n), H1_tail(n+1))         # 환경 공유 (nearby zeros가 양쪽에 기여)
ρ(Δ_L(n)⁻², Δ_R(n+1)⁻²)           # 왼쪽/오른쪽 NN 교차 상관
ρ(B_sq(n), B_sq(n+1))              # B항 간 상관
```

**Step 4**: 분산 기여도 분석
```
Var(2/g²) / Var(A)                   # 공유항이 설명하는 분산 비율
Var(A') / Var(A)                     # 잔차가 설명하는 분산 비율
```

**Step 5**: W 의존성 (W=100, 500에서 H1_tail 수렴 확인)

**Step 6**: 편상관(partial correlation) 교차 검증
```
ρ_partial(A_n, A_{n+1} | g)          # 논문의 +0.391 재현 확인
ρ_partial(A'(n), A'(n+1) | g)        # 이중 보정
```

**모델**: sonnet (C-297 인프라 확장, 수식 정확히 명세됨)

**왜**: 증명 전략을 결정하는 핵심 실험.
- 만약 ρ_residual ≈ 0이면: 증명은 "공유 항의 분산 기여 > 0" 만 보이면 됨 (쉬움)
- 만약 ρ_residual > 0이고 H1_tail 상관이 주범이면: pair correlation 함수 분석 필요 (중간 난이도)
- 만약 ρ_residual > 0이고 B² 상관이 주범이면: 교대합 공분산 bound 필요 (어려움)
결과가 어떻든 증명의 **정확한 난점**을 식별할 수 있음.

**주의**:
- g(n,n+1) = 0인 경우 없을 것이나 안전 가드 추가
- A_rest = A_bare - A_NN (C-297) ≠ A'(n) (여기서의 정의). 혼동 금지.
  C-297의 A_NN은 양쪽 NN(j=1) 기여를 포함하나, 여기서 H1_shared는 오른쪽 NN만.
  → **인접 쌍별로** 공유항을 정의해야 함 (각 n에 대해 n+1 방향만 공유)
- 비인접 쌍 ρ(A_n, A_{n+2}) 도 대조군으로 한 줄 추가 (공유항 없으므로 상관 약할 것)
