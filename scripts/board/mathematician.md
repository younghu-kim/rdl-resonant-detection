**다음 작업**: 결과 #37 — **곡률 값 분포와 GUE 간격 분포의 정량적 관계 검증**

**모델**: sonnet

**왜**: 
1. 결과 #34에서 Δκ-NNS 상관 R²=0.92 확립 (Lorentzian² 모델). 이는 "상관"이지 "분포 예측"이 아님.
2. 이 실험은 한 단계 더 나아가 **RMT(GUE 간격 분포)로부터 곡률 분포를 정량적으로 예측**한다.
3. κ ≈ 1/(δ²+NNS²)² 관계가 확립되었으므로, P(κ)를 P_GUE(NNS) 변수 변환으로 유도 가능.
4. 이것이 성공하면 논문의 두 기둥(곡률 분리 + RMT)을 **하나의 정량적 예측으로 통합**.
5. spacing ratio 실패 후 새 방향으로, RMT 연결을 곡률 관점에서 심화하는 자연스러운 전진.

**설계 핵심**:
1. σ=1/2+δ (δ=0.03)에서 t ∈ [1000, 10000] 구간에 N=2000 균등 샘플링
2. 각 t에서 κ(t) 계산 + 최근접 영점 γ_n 탐색 → NNS = |t - γ_n|
3. 경험적 κ 분포 P_emp(κ) 구축
4. GUE 간격 분포 P_GUE(s)로부터 NNS 분포 도출 → κ = 1/(δ²+s²)² 변수 변환 → P_pred(κ) 예측
5. KS 검정: P_emp vs P_pred

**주의**:
- ★ κ 계산은 해석적 공식 (h=10^{-20} 수치미분 금지)
- ★ δ=0.03 오프셋 필수 (영점 위 측정 금지)
- ★ NNS는 **unfolded 좌표** 사용: s_unf = NNS · (log(t/2π))/(2π)
- ★ P_GUE(s) = (32/π²)s² exp(-4s²/π) (Wigner surmise for GUE)
- ★ 변수 변환 야코비안: P_pred(κ) = P_GUE(s(κ)) · |ds/dκ|
- ★ 극값 영역(κ→∞, NNS→0) 제외: NNS_min = δ (s=0에서 κ=1/δ⁴ 발산)

**성공 기준**:
- 양성: KS p > 0.05 (P_emp과 P_pred 비기각) + Q-Q 상관 > 0.95
- 강한 양성: + AD 비기각 + 분위수 편차 전구간 < 10%
- 음성: KS p < 0.01 또는 Q-Q 상관 < 0.80
- 중립: 0.01 < p < 0.05 또는 Q-Q > 0.90이나 꼬리 편차 > 20%
