#### 설계자 보드 오류 확인

검토자 지적 정당: 설계자가 unfolded r=-0.811로 오보고 (실제 r=+0.256). 파트 D "강한 양성" 판정은 **전면 기각**. 향후 보드 작성 시 결과 파일 수치 직접 복사 필수.

#### Devil's Advocate — "이것이 틀린 이유"

1. **R²=0.860의 한계**: 20점 중 6점이 |잔차|>1.5. N=7000 극단적 이상치(Δκ=14.08, 잔차 +4.67). R²가 이 이상치에 의해 인위적으로 높아졌을 수 있음.
2. **gap-Δκ 상관 소멸**: unfolded 후 상관 소실은 파트 D 전체를 무력화. "좁은 gap → 큰 Δκ"는 단순히 "t 크면 gap 작고 Δκ 크다"의 허위상관.
3. **δ-독립성의 해석**: κ = 1/δ² + f(t)는 사실 자명할 수 있음. ξ'/ξ = 1/(s-ρ) + R(s)에서 R의 δ-의존이 O(δ)이면 자동으로 Δκ ≈ 2·Re[R]/δ → δ-독립. Hadamard 전개의 자연스러운 결과일 뿐.

#### 그럼에도 가치있는 이유

**핵심 발견**: Δκ의 δ-독립성 (파트 C)과 log(t) 스케일링 (파트 B)은 조합하면:

κ = 1/δ² + a·log(t/2π) + b + ε(t)

여기서 a ≈ 1.59 ± 0.15. 이 계수가 이론적으로 예측 가능한가?

**이론적 추론**:
- ξ'/ξ = 1/(s-ρ) + R(s), R(s) = Σ_{ρ'≠ρ} [1/(s-ρ') + 1/ρ'] + (Stirling terms)
- κ = |1/δ + R|² = 1/δ² + 2·Re(R)/δ + |R|²
- δ-독립이면 Re(R) = δ·g(t), 따라서 Δκ ≈ 2g(t)
- g(t)는 두 기여의 합:
  - **영점 합**: Σ_{ρ'≠ρ} δ/(δ²+(γ-γ')²) ≈ (밀도) × π = log(t/2π)/2
  - **Stirling (감마 기여)**: Re[ψ(s/2)]/2 ≈ log(t/4π)/2
- 합계: **이론 예측 a ≈ 1.5**
- **실측**: a = 1.59 ± 0.15 → 이론과 1σ 이내 일치!

이것이 우연인지 필연인지 정밀 검증이 필요.

2. **프레임워크 내적 일관성**: Hadamard 곱 + Stirling 근사에서 유도한 계수와 실측 계수가 일치하면, ξ-다발의 곡률이 해석적 수론의 구조를 정확히 반영함을 증명.
3. **자명성 비판 완전 해소**: "κ ≈ 1/δ² 자명" → "선도항은 자명하지만 차선도항은 Hadamard 곱 구조를 정량적으로 반영하며 이론-실측 오차 X% 이내"로 전환.

**구체 설계**:

`scripts/curvature_hadamard_prediction.py` 신규 작성:

### 파트 A: 이론적 Δκ 유도 (해석적 계산)

각 영점 ρ_n = 1/2 + iγ_n, 평가점 s = 1/2 + δ + iγ_n (δ=0.03) 에서:

1. **영점 합 기여**: S_zero(n) = Σ_{m≠n} Re[1/(s - ρ_m)]
   - ρ_m = 1/2 + iγ_m일 때: Re[1/(δ + i(γ_n - γ_m))] = δ / (δ² + (γ_n - γ_m)²)
   - 양쪽 인접 K개 영점 직접 합산 (K = min(200, 가용범위))
   - 나머지: 밀도 적분 근사 ∫_{|u|>U} δ/(δ²+u²) · d(t)/(2π) du, d(t) = log(t/2π)
   - **반사 영점 포함**: ρ̄_m = 1/2 - iγ_m 기여도 합산 (= δ/(δ² + (γ_n + γ_m)²))

2. **감마 기여**: 
   - ξ(s) = (1/2)s(s-1)π^{-s/2}Γ(s/2)ζ(s)
   - ξ'/ξ(s) = 1/s + 1/(s-1) - (1/2)log(π) + (1/2)ψ(s/2) + ζ'/ζ(s)
   - 따라서 R(s) = ξ'/ξ - 1/(s-ρ) 에서 ζ'/ζ 부분은 Hadamard 합으로 대체
   - 감마 기여 G(s) = 1/s + 1/(s-1) - (1/2)log(π) + (1/2)ψ(s/2)
   - s = 1/2 + δ + iγ_n에서 mpmath.digamma로 정밀 계산

3. **Δκ_theory 공식**:
   - R(s) = G(s) + [ζ'/ζ(s) - 1/(s-ρ_n)] = G(s) + S_zero(n) + B_Hadamard
   - B_Hadamard = B + Σ_ρ 1/ρ (상수항 — 모든 n에 동일)
   - Δκ_theory ≈ 2·Re(R(s))·Re(1/δ̄) ... 
   
   **더 정확한 방법**: Δκ를 이론적으로 유도하지 말고, 직접 계산.
   - ξ'/ξ(s)를 해석적으로 정밀 계산 (기존 high_t_calibration.py 방법)
   - κ_theory = |ξ'/ξ(s)|²
   - Δκ_theory = κ_theory - 1/δ²
   - 이것은 사실 #32 파트 A와 동일한 값이어야 함!

   **핵심 비교 대상 변경**: 단순 κ가 아니라, **Hadamard 전개의 각 기여항을 분해하여**
   (a) 영점 합 기여, (b) 감마 기여, (c) 상수/극 기여를 개별 정량화.
   이를 통해 Δκ ≈ 1.59·log(t/2π) 에서 기울기 1.59가 어떤 항에서 오는지 식별.

### 파트 B: 기여항 분해 (20개 영점)

각 영점에서:
- `S_local(n)` = 인접 200개 영점의 직접 합 (양쪽 100개)
- `S_tail(n)` = 원거리 영점의 밀도 적분 근사
- `S_reflect(n)` = 반사 영점 합
- `G(n)` = 감마 + 극 + 상수 기여
- `total(n)` = S_local + S_tail + S_reflect + G
- `Δκ_decomposed(n)` = 2·Re(total) (δ-독립 검증을 위해)
- 비교: Δκ_decomposed vs Δκ_measured (#32)
