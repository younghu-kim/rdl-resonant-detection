**모델**: opus

**왜**: 3편 논문은 완결. 제출은 사용자 결정을 기다리는 동안, 가장 가치 높은 미해결 문제인 σ-국소화의 해석적 근거를 수치적으로 탐색한다. 현재 σ=1/2 유일성은 385개 수치 검증(ratio >>10)으로 확립되었으나, **왜 σ=1/2에서만 κ가 발산하는지** 해석적 메커니즘이 불명확하다. 이것이 Paper 4의 핵심이 될 것이다.

**구체적 지시**:

1. **σ 근방 κ 프로파일 정밀 측정**: ζ(s)의 처음 20개 영점에 대해
   - σ = 0.3, 0.35, 0.40, 0.45, 0.48, 0.49, 0.495, 0.499, 0.4999, 0.5, 0.5001, 0.501, 0.505, 0.51, 0.52, 0.55, 0.60, 0.65, 0.70 (19개 σ값)
   - 각 σ에서 κ(σ, t_n) 계산 (bundle_utils 활용)
   - t_n = 각 영점의 허수부

2. **log κ vs σ 프로파일 분석**:
   - σ=1/2에서의 peak가 logarithmic인지, algebraic인지, essential singularity인지 판별
   - 구체적으로: log κ ~ α·log|σ - 1/2| + β 피팅 → α 추정 (algebraic이면 α가 유한)
   - log κ ~ γ/|σ - 1/2| 피팅 → 지수적 발산 여부
   - 20개 영점 각각의 α, γ 분포 보고

3. **좌우 비대칭 측정**: σ < 1/2 vs σ > 1/2에서 κ 프로파일이 대칭인지 확인
   - FE에 의해 이론적으로 대칭이어야 함 — 수치적 확인
   - 비대칭이 있다면 계산 정밀도 문제 vs 물리적 원인 분류

4. **결과 저장**: `results/sigma_localization_profile_c254.txt`

**주의**:
- bundle_utils의 kappa 함수가 off-critical σ에서도 작동하는지 확인 필요. 안 되면 직접 계산: κ = |ξ''/ξ - (ξ'/ξ)²| (ξ는 완비 ξ-함수)
- mpmath dps=50 이상 사용 (σ ≈ 0.5 근방 소거 오차 방지)
- σ=1/2에서의 값은 기존 385x 결과와 일치해야 함 (sanity check)

**성공 기준**:
- 19×20 = 380개 κ값 측정 완료
- α (algebraic order) 추정값이 20개 영점에서 일관적 (CV < 30%)
- FE 대칭 확인: |κ(σ) - κ(1-σ)| / κ(1/2) < 1% (σ ≠ 1/2인 10쌍)
- peak 유형 판별: algebraic / exponential / logarithmic 중 하나 선택 + 근거

## [아카이브] 판정 [2026-04-24] — 사이클 #253 (C-246 중간점 Hadamard 예측)

**C-246 판정**: ★★ 부분 양성 + 부분 음성

| 발견 | 판정 | R²/ρ |
|------|------|------|
| NN 상쇄 (R²=99.97%) | ★★★★ 확립 | ρ=+0.9997 |
| NNN 고t 공식 | ❌ 기각 | R²=6.1% |
| 10영점 Hadamard | ❌ 불충분 | R²=2.4% |
| 비국소 상관 재현 | ✅ | ρ=-0.637 (p=6e-35) |

**핵심 해석**: 
- NN 상쇄는 Hadamard 대칭의 직접 귀결 (산술적으로 자명하나 ξ-bundle에서 유의미)
- κ_mid의 96%는 전역 Hadamard 합 + smooth background (ψ(s/2), 1/s 등)이 결정
- gap_next 비국소 상관 ρ=-0.637은 GUE에서 부재 (v2b 확인) → ζ 영점 고유 현상
- **메커니즘 미해결**: local_density 무상관 (ρ=0.022), NNN 무상관 (ρ=0.008)
- NNN 공식 [1/(Δ_{-1}+Δ/2) - 1/(Δ_{+1}+Δ/2)]²은 변동의 6%만 설명 → Proposition 후보 탈락

**비평가 판정**: NN 상쇄 = 생존. NNN 공식 = 기각. 비국소 메커니즘 = 미해결.
**논문 반영**: NN 상쇄는 부록 관측으로 기록 가능. 주 정리 추가 불필요.
**다음**: 비국소 메커니즘 규명은 별도 연구 과제. 현재 논문 범위 밖.
