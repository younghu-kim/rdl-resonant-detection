#### 확립이 아닌 이유

1. **항등식 vs 예측**: 파트 B의 0.00% 오차는 수학적 항등식의 수치 검증. 인프라 정확성 벤치마크로 가치있으나, 물리적 예측이 아님.
2. **기울기 이론값 부재**: a=1.59가 왜 1.59인지는 분해로 "어디서 오는지"를 밝혔지만, a_S_all = −0.4768이 왜 −0.5가 아닌지(잔차 0.0232의 정확한 이론식)는 미도출.
3. **R²=0.860 미해결**: Im[R]² 산포가 원인이라는 것은 진단이지 해결이 아님.

#### Devil's Advocate

1. **95.4% 상쇄의 자명성**: G와 S_all은 같은 함수(ξ'/ξ)의 부분이므로 Stirling 수준에서 상쇄는 *예상*되는 것. 상쇄가 정확히 95.4%라는 수치 자체는 δ-의존적(δ가 달라지면 비율 변동).
2. **N=20 샘플**: 기울기 R²=0.854에서 σ=0.15. 95% CI는 [1.29, 1.90]. "이론 예측 1.5와 일치"는 CI가 넓어서이지 정밀도가 높아서가 아님.
3. **δ 의존성 미검증**: 모든 결과가 δ=0.03 단일 값. δ=0.01, 0.05, 0.1에서 기울기가 어떻게 변하는지 미확인.

#### 가치 요약

**#32 + #33 통합 의미**: 
- κ = 1/δ² + Δκ(t)에서 차선도항 Δκ의 구조를 완전 분해
- Δκ의 *결정론적* 부분(2Re[R]/δ)은 감마-영점 상쇄 잔차에서 기원
- *확률론적* 부분(Im[R]²)은 인접 영점 간격 통계에서 기원 → GUE 연결 가능

## 다음 작업: 결과 #34 — Δκ 잔차와 GUE 간격 통계의 상관

**모델**: sonnet

**왜**: 이것이 지금 가장 가치있는 이유:

1. **R²=0.860 문제 해결 가능**: Δκ의 산포가 인접 영점 간격(nearest-neighbor spacing, NNS)에 의해 설명된다면, log(t) + NNS 2변수 회귀에서 R² > 0.9 달성 가능.
2. **두 독립 결과 통합**: #4 GUE pair correlation과 #32-33 subleading κ를 단일 프레임으로 연결. "곡률 잔차가 GUE 통계를 반영한다"는 기하-RMT 연결.
3. **#33의 자연스러운 귀결**: 파트 E에서 Im[R]²이 stochastic이라 진단했으므로, 다음 질문은 "Im[R]의 분포가 무엇인가?"

**구체 설계**:

`scripts/curvature_gue_residual.py` 신규 작성:

### 파트 A: 확장 샘플 Δκ + NNS 수집 (N=100개 영점)

기존 20개 인덱스를 100개로 확장 (등간격 선택: N=1,5,10,15,...,20000 — np.linspace(1,20000,100).astype(int)).
각 영점 n에서:
- t_n = zetazero(n)
- s_n = 1/2 + δ + it_n (δ=0.03)
- κ_n = |ξ'/ξ(s_n)|² (해석적 — log(ξ)' 공식)
- Δκ_n = κ_n - 1/δ²
- gap_left = t_n - t_{n-1}, gap_right = t_{n+1} - t_n
- NNS_min = min(gap_left, gap_right)
- NNS_unfolded = NNS_min × log(t_n/(2π)) / (2π)

### 파트 B: Δκ 잔차 vs NNS 상관

1. 잔차 계산: res_n = Δκ_n - (a·log(t_n/2π) + b), 기울기 a,b는 1변수 OLS
2. res_n vs NNS_unfolded: Pearson r, Spearman ρ, p-value
3. res_n vs 1/NNS_min²: 이론 예측 — 가장 가까운 영점이 gap=g이면 S_local ≈ δ/(δ²+g²), 따라서 Δκ 잔차 ∝ [δ/(δ²+g²)]²
4. 산점도 데이터 출력 (NNS_unfolded, res_n) 상위/하위 10개

### 파트 C: 2변수 회귀

Δκ ~ a·log(t/2π) + b + c·f(NNS_min) 적합, f 후보 3종:
- f₁ = δ²/(δ²+NNS_min²)²  (Lorentzian² — 이론 동기)
- f₂ = 1/NNS_unfolded
- f₃ = log(NNS_unfolded)
각각 R²_2var, AIC, F-test p-value 보고.
**성공 기준**: R²_2var > 0.92 (vs R²_1var = 0.860)

### 파트 D: Im[R] 분포 vs GUE 예측

- 100개 영점의 Im[R] 수집 (해석적 공식)
- Im[R]은 주로 인접 영점에 의존: Im[R] ≈ Σ_m (γ_n-γ_m)/((δ²+(γ_n-γ_m)²))
- Im[R]²의 분포: 히스토그램 (10 빈)
- GUE Wigner surmise에서 NNS 분포 p(s) = (32/π²)s²exp(-4s²/π)
  → Im[R]²의 이론적 분포 유도 가능한지 검토
- KS-test: 경험 분포 vs 이론 분포

### 파트 E: 이상치 해부

- 100개 영점 중 |res_n| > 2σ인 이상치 식별
- 이상치의 NNS_unfolded 분포: 전부 NNS_unfolded < 0.5?
- GUE level repulsion: P(s<0.5) ≈ ∫₀^{0.5} p(s)ds 이론값과 비교

**주의사항**:
- 100개 영점 × 3 (n-1, n, n+1) = 300회 zetazero 호출. dps = max(50, 30+t/200).
- 고 t(N>15000): zetazero 느림. 타임아웃 30초/영점. 총 15분 이내.
- δ=0.03 고정.
- N<5 (t<33) 제외: 밀도 근사 불안정.
- ξ'/ξ 계산: #33과 동일한 해석적 공식 (log(ξ)' = 1/s + 1/(s-1) - log(π)/2 + ψ(s/2)/2 + ζ'/ζ(s)).
- ζ'/ζ: h=1e-6 중앙 차분 (mpmath.diff 사용 금지).
- **partial correlation 사용**: res vs NNS에서 t를 통제하지 않아도 됨 (res는 이미 log(t) 제거 후 잔차).

**성공 기준**:
- **강한 양성**: res-NNS |r| > 0.5, p < 0.01이고, 2변수 R² > 0.92
- **양성**: |r| > 0.3, p < 0.05이고, 2변수 R² > 0.88
- **음성**: |r| < 0.2, p > 0.1 → 잔차는 NNS와 무관 (다른 원인 탐색 필요)
