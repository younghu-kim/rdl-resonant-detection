**다음 작업**: **C-375 — 교차 가족 A_Λ-gap_min 보편성 검정 (Paper 4 씨앗)**

5개 L-함수 가족에서 ρ(A_Λ, gap_min_GUE)의 보편성을 검정한다.

**실험 설계**:

1. **대상 L-함수** (5개):
   - ζ(s) — degree 1, N=1 (기준선, C-270 재현)
   - χ₋₃ (mod 3) — degree 1, N=3
   - 11a1 타원곡선 — degree 2, N=11
   - Δ (Ramanujan tau) — degree 2, N=1, weight=12
   - sym²(11a1) — degree 3

2. **각 L-함수별 수행**:
   - PARI `lfunzeros()`로 영점 수집. t ∈ [적정 범위] (최소 100개 영점 확보)
   - Hadamard zero-sum으로 A_Λ(t₀) 계산: A_Λ = Im(S₁^Λ)² + 2·H₁^Λ
     여기서 S₁^Λ = lim_{s→ρ} [(s-ρ)Λ'/Λ], H₁^Λ = Σ_{n≠0} 1/(t₀-tₙ)²
   - gap_min = min(t₀-t_{prev}, t_{next}-t₀), GUE 정규화: gap_min / d̄(t₀)
     여기서 d̄ = 평균 영점 간격 = 1/N(T) 역수 (해당 높이에서의 영점 밀도)
   - Spearman ρ(A_Λ, gap_min_GUE) 계산
   - 중앙 60% 트리밍 (양쪽 경계 영점 제외)

3. **핵심 비교**:
   - 5개 ρ값이 모두 < -0.70이면 → 보편성 양성
   - ρ 간 변동 (max - min)이 0.20 이내면 → 강한 보편성
   - 특정 가족에서 ρ > -0.50이면 → 보편성 깨짐 (가장 흥미로운 결과)

4. **참조 코드**: 
   - `scripts/gap_stability_c270.py` — ζ(s) 기준선 (T=2000, Hadamard zero-sum)
   - `scripts/A_gap_correlation_c256.py` — Cauchy 방법 (참고만)
   - `scripts/gl3_A_gap_c264.py` — GL(3) setup

5. **주의사항**:
   - **GUE 정규화 필수**: 각 L-함수의 영점 밀도가 다르므로, gap을 local mean spacing으로 나눌 것
   - **A_Λ 사용** (A_L 아님): C-270에서 A_Λ+gap_min이 t-안정(span=0.16), A_L+gap_min은 불안정(span=0.30)
   - **S₁^Λ 계산**: Λ = completed L-function. PARI `lfun` 객체에서 직접 Λ'/Λ 계산 또는 Hadamard formula 사용
   - **degree 3 (sym²)**: PARI `lfunsympow` 사용. 영점 수 50+ 이상 확보 목표 (적으면 통계 취약)
   - **gap_normalization 주의**: 경험적 d̄=2/(Δt) 사용 금지. 이론적 밀도 공식 사용 (MEMORY.md 참조)
