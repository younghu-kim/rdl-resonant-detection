### 결과 #31 판정: **양성** (강한 양성에서 하향) — κ 기준 교정 후 확립 예정

#### 양성 근거

1. ✅ **mono = 2.0000π 3/3 완벽**: t=1000, 5000, 10000 모두 단일 영점 분리 성공. 고 t에서 영점 밀도 증가(~log(t)/2π)에도 r=0.2 내 단일 영점 유지. 해석적 arg(ξ) 공식의 수치 안정성 확인.
2. ✅ **κ t-안정성**: 1115.9, 1120.3, 1119.6 — t에 무관하게 일관 (변동계수 0.2%). 이것이 핵심 가치.
3. ✅ **수치 인프라 완성**: 해석적 ξ'/ξ 공식이 t=10⁴까지 안정 작동. 향후 모든 고 t 실험의 토대.

#### 확립 보류 사유 — κ 기준값 불일치

검토자가 정확히 포착한 문제: 기존 기준 κ=1030 (bundle_utils.py, h=10⁻²⁰ 수치 미분)과 새 측정 κ≈1116 (해석적 공식)의 8% 차이. 두 가지 가능성:

- (a) 해석적 공식이 정확하고 bundle_utils가 수치 오차를 가짐 → 기준값 교정 필요
- (b) 두 방법의 측정점이 미묘하게 다름 (δ 정의 차이 등)

어느 쪽이든, **동일 방법으로 t<100과 t>1000을 비교해야** "t-스케일링" 주장이 정당화된다. 현재는 사과와 오렌지 비교.

#### peak_σ / FWHM — 이산화 한계 인정

검토자 분석 정확. t별 격자 해상도가 다르므로(201→101→51점) peak_σ의 t-의존 변화(0.497→0.494→0.488)는 물리적 변화가 아닌 격자 아티팩트. FWHM은 전부 "격자 2칸"으로 해상 불가. 이 두 지표는 현재 실험에서 t-스케일링 증거로 사용 불가.

#### Devil's Advocate — #31이 틀린 이유 3가지

1. **κ ≈ 1/δ² 자명성**: 영점 근방에서 |ξ'/ξ|² ≈ 1/|s-ρ|² + cross terms. δ=0.03 오프셋이면 1/0.03²=1111. κ≈1116은 이론적 예측 1111과 <0.5% 차이. "κ가 t-독립"이라는 관측은 사실상 "δ-오프셋이 같으면 leading term이 같다"는 자명한 명제.
   → **반론**: cross terms (ζ'/ζ + ψ(s/2) 기여)가 t에 따라 변할 수 있으나 실제로 <0.5% 기여. 이것 자체가 의미 있는 확인: 고 t에서도 단일 영점이 κ를 지배.
2. **mono=2π는 "영점이 있다"의 동어반복**: winding number = 내부 영점 수. r=0.2 내 단일 영점이면 자동 2π.
   → **반론**: 고 t에서 r=0.2 내에 복수 영점이 들어갈 확률 증가(평균 간격 ~2π/log(t)). t=10000에서 평균 간격≈0.68이므로 r=0.2는 안전 마진 충분. 하지만 이것은 영점 분포의 성질이지 프레임워크의 성질이 아님.
3. **실용적 가치 미약**: "프레임워크가 고 t에서도 작동한다"는 좋지만, 새로운 수학적 발견이 아님. 기존 결과의 범위 확장일 뿐.
   → **인정**: 맞음. 논문에서 "robustness" 절로 기술하되, 독립 결과로 과대 포장하지 말 것.

### 알려진 함정 (누적)

- 모노드로미: eps 차분 금지 → 폐곡선 적분(radius, 64단계)
- Dirichlet character `a` 파라미터: χ(-1)=+1→a=0, χ(-1)=-1→a=1
- κ 측정: 영점 위 직접 측정 금지 → δ=0.03~0.05 오프셋
- findroot: 단일 시작점만, 튜플 전달 금지
- numpy 2.0: np.trapezoid (trapz 아님)
- **bundle_utils.py h=10^{-20}**: dps ≥ 100 또는 해석적 공식 사용.
- **중간점 Hadamard 상쇄**: m=(γₙ+γₙ₊₁)/2에서 두 최근접 영점의 기여 = 0.
- **ξ(s) underflow**: log(ξ)' 해석적 공식으로 영구 해결.
- **mpmath.diff(xi, s)**: t>300에서 실패. 사용 금지.
- **Dirichlet κ 계산**: bare L'/L 사용 금지. Λ'/Λ 해석적 공식 필수.
- **윤곽 적분 시 영점 회피**: t₂는 반드시 영점 사이 중간점.
- **홀수 디리클레 지표 영점 탐색**: Re(L(1/2+it))=0 ≠ L=0. |L| 최소화 필수.
- **짝수 디리클레 지표**: a=0, Γ(s/2) 사용. Re(L)=0 단독 불충분.
- **영점 탐색 보편적 방법**: |L(1/2+it)| 최소화가 Re(L)=0보다 엄밀히 우월.
- **F₂ 면적분 정의**: 격자 플라켓(arg ξ 와동 밀도) 사용.
- **σ-국소화 표현**: "RH의 기하학적 동치" 금지 → "RH와 일관적인 기하학적 증거".
- **Number variance unfolding**: L은 unfolded 좌표.
- **Number variance sub-GUE**: t<600에서 GUE 수렴 기대 금지.
- **Epstein zeta 격자합**: Richardson 외삽 또는 Chowla-Selberg. cutoff N≥200.
- **DH 함수**: χ(2)=i, χ(3)=-i, χ(4)=-1. 홀수 지표(a=1). 오일러 곱 없음.
- **DH mono=4π 오염**: 고 t에서 r=0.5 내 복수 영점. r=0.1~0.15.
- **FWHM 윈도우 아티팩트**: 표준: σ₀±0.3, 201점, 간격 0.003.
- **반사 영점 오염**: 함수방정식 대칭 함수에서 스캔 범위 내 반사 영점 주의.
