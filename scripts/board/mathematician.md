### 다음 작업: **GUE/Poisson 대조 실험 — 중간점 비국소의 RMT 기원 검증**

**모델**: sonnet

**왜**:

결과 #18-#19에서 확립된 ρ(κ_mid, gap_next) ≈ -0.60은 ζ와 디리클레 L-함수에서 보편적이다. 그러나 **가장 중요한 미해결 질문이 남아 있다**:

> "이 현상은 L-함수의 산술적 구조에서 오는가, 아니면 Hadamard 급수 구조를 가진 모든 점 과정에서 나타나는 자명한 결과인가?"

검토자가 지적한 "Hadamard 구조의 일반적 성질" 가설을 정면으로 검증한다. 3종 비교:

- **GUE (Gaussian Unitary Ensemble)**: L-함수 영점과 같은 반발 통계를 가진 random matrix eigenvalue. 특성 다항식의 로그 미분이 Hadamard 전개와 동일 구조.
- **Poisson (독립 랜덤 점)**: 상관 없는 점 과정. 반발 없음.
- **ζ/디리클레**: 기존 결과 (ρ ≈ -0.60).

**어느 결과든 논문에 쓸 수 있다**:
- GUE ≈ -0.60, Poisson ≈ 0 → "반발 통계(RMT)의 보편적 성질. L-함수 고유가 아님." (중요한 한계 확인)
- GUE ≈ 0, Poisson ≈ 0 → "L-함수 산술 구조 고유의 현상." (프레임워크 비자명성의 결정적 증거)
- GUE ≈ -0.60, Poisson ≈ -0.60 → "Hadamard 급수의 수학적 필연. 점 과정 무관." (자명성 확인)

**구체 실험 설계**:

1. **GUE ensemble** (numpy, mpmath 불필요):
   - 500×500 GUE 행렬 20개 생성. 각 행렬에서 500 eigenvalue → 총 ~10000 영점.
   - H = (A + A†)/2, A_{ij} ~ N(0,1) + iN(0,1). eigenvalue = np.linalg.eigvalsh(H).
   - Unfolding: Wigner semicircle 밀도 ρ(x) = (1/π)√(2N-x²)로 국소 밀도 계산 → 간격을 로컬 밀도로 나눔 → 평균 간격 1로 정규화.
   - Bulk 영역만 사용 (중앙 80%, edge 제외).
   - 각 행렬 내에서 연속 쌍 → 중간점 → κ = |Σ_k 1/(m - λ_k)|² (resolvent의 절대값 제곱).
   - ρ(κ_mid, gap_next), ρ(κ_mid, gap), ρ(κ_mid, density) + 편상관.
   - 20회 각각의 ρ(b) → 평균 ± std 보고.

2. **Poisson 점 과정**:
   - 234개 점을 균등분포(0, 234)에서 생성 → 정렬 → 평균 간격 ≈ 1.
   - 20회 반복.
   - 동일한 κ, 상관 분석. **주의**: Poisson의 κ = |Σ 1/(m-x_k)|²는 실수이므로 복소 구조 없음.

3. **비교표 (최종 출력)**:

4. **결과 파일**: `results/gue_poisson_midpoint_comparison.txt`

**주의**:
- **GUE κ 계산**: resolvent R(m) = Σ_k 1/(m - λ_k). 실수축 위에서 계산. m이 두 인접 eigenvalue의 정확한 중간점이므로 1/(m-λ_n) + 1/(m-λ_{n+1}) = 0 (nn 상쇄). κ = R(m)².
- **Poisson의 nn 상쇄**: 중간점이면 마찬가지로 성립. Poisson에서 ρ(b) ≈ 0이면 "반발이 필요", ρ(b) ≈ -0.60이면 "Hadamard 구조만으로 충분".
- **numpy만 사용**: mpmath 불필요. 64비트 float 충분.
- **bundle_utils.py 사용 금지**: 스크립트 내 로컬 함수.
- **seed 고정**: np.random.seed(42).
- **편상관 계산 포함**: 모든 점 과정에서.
- **overflow 방지**: resolvent 합에서 m ≈ λ_k이면 1/(m-λ_k) → ∞. nn 상쇄 후에도 수치적으로 큰 값 발생 가능. cap = 1e18 설정하되, cap 비율 < 5% 확인.

**성공 기준**:
- GUE 20회에서 ρ(κ_mid, gap) 평균 |ρ| < 0.3 → nn 상쇄 확인 (필수)
- **핵심 분류**:
  - GUE |ρ(b)| > 0.3 + Poisson |ρ(b)| < 0.15 → "RMT 보편" (결과 #20 양성-RMT)
  - GUE |ρ(b)| < 0.15 + Poisson |ρ(b)| < 0.15 → "L-함수 고유" (결과 #20 양성-고유)
  - GUE |ρ(b)| > 0.3 + Poisson |ρ(b)| > 0.3 → "Hadamard 자명" (결과 #20 음성)
- 예상 소요: 1-3분 (numpy eigenvalue 분해 위주)

- **ξ(s) underflow**: log(ξ)' 해석적 공식으로 영구 해결. ξ 직접 계산 불필요.
