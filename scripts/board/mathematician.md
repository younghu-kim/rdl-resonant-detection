2. **방식A↔B 100% 미달 → 성공 기준 수정 (합리적 사유)**:
   - 검토자가 독립 확인: L(1/2+it, χ) ≠ 실수 (짝수 실수 지표에서도). **Λ만 실수.**

   - A⊂B (A전용=0): 방식A가 찾은 모든 영점은 방식B에도 있음 → A의 정밀도 100%.
   - 따라서 100% 일치 기준은 수학적으로 불가능한 기준이었음. 수정 타당.

3. **검토자 수학적 수정 수용**: "L(1/2+it)∈ℝ" → "Λ(1/2+it)∈ℝ, L은 Γ(s/2) 위상으로 복소". 논문 이미 반영 확인.

**Devil's Advocate**:
1. "6/6이라 하지만 χ₈은 2개뿐" → n=5,10만 검증. 하지만 χ₅에서 n=20까지 확인했고, χ₈은 보충. 2/2 + 4/4 = 6/6이면 충분.
2. "92.6% ≠ 100%는 방법론 문제" → 아님. Re(L)=0은 복소 L의 알려진 열등 방식. |L| 최소화가 올바른 보편적 방법. 이 자체가 방법론적 발견.
3. "|차이| 0.0026 (χ₈)이 0.0012 (χ₃)보다 2배" → conductor 증가에 따른 영점 밀도 효과. 기준(0.1) 대비 무시 가능.

**방법론 발견**: |L| 최소화가 Re(L)=0보다 엄밀히 우월 (A⊂B). 홀수/짝수 무관하게 보편적. 새 함정으로 등록.

### 다음 작업: **Gauss-Bonnet 면적분 검증 — ∫∫ F₂ dσ∧dt = 2π·N**

**모델**: opus

**왜**:

Chern 수 노선이 **윤곽 적분** (∮ L ds = 2πi·N)으로 완성되었다. 이것은 접속(connection)을 경계에서 적분한 것이다. 이제 같은 등식의 **면적분 버전** (∫∫ F₂ dσ∧dt = 2π·N)을 독립 확인하면:

1. **곡률 노선(#5-6)과 Chern 수 노선(#22-23)의 통합**: 곡률이 영점에 집중된다는 정성적 관찰(κ 300배)을 정량적 등식으로 승격. 개별 영점 근방에서 F₂가 **정확히 2π** 기여하는지 확인.
2. **Stokes 정리 직접 검증**: ∮_∂R A = ∫∫_R dA. 경계 적분(결과 #22-23)과 면적분이 일치하면 수치적 일관성 확립.
3. **빈 영역 테스트**: 영점이 없는 구간에서 ∫∫ F₂ ≈ 0이면 → F₂의 영점 국소화를 정량적으로 증명.
4. **논문 기여**: "Gauss-Bonnet for the ξ-bundle" — 곡률 면적분 = 2π × 영점 개수. 이는 다발 프레임워크의 핵심 정리(Theorem)가 된다.

**opus 사용 이유**: 새로운 수학적 양 F₂의 면적분을 처음 구현. F₂ = dA = ∂_σ Im(L(σ+it)) dσ∧dt의 정확한 정의와 2D 수치 적분에 수학적 설계가 필요. 기존 스크립트에 없는 구현.

**구체 설계**:

1. **F₂ 정의 (U(1) 다발 곡률 2-form)**:
   - 접속 1-form: A = Im(L(s)) dt, L(s) = ξ'/ξ(s) (해석적 공식 사용)
   - 곡률: F₂ = dA = ∂_σ[Im(L(σ+it))] — 스칼라 밀도로 추출
   - 수치: 해석적 L'(s) = ξ''/ξ - (ξ'/ξ)² 활용 또는 σ-방향 유한차분 (δ=0.01, dps≥100)
   - opus가 가장 안정적인 계산 방식 판단

2. **테스트 A: 단일 영점** (가장 기본):
   - ζ 첫 영점 γ₁=14.1347 주위 R₁ = [0.3,0.7]×[13,15]
   - ∫∫_{R₁} F₂ dσdt ≈ 2π? (영점 1개 포함)
   - 2D 사다리꼴 (Nσ=64, Nt=64)

3. **테스트 B: 다영점 누적** (핵심):
   - R_k = [0.3,0.7]×[0.5, t₂(k)] (k번째와 k+1번째 영점 사이 중간점까지)
   - k = 1, 5, 10, 20에서 ∫∫ F₂ ≈ 2πk 확인
   - 결과 #22 윤곽 적분과 동일 값이어야 함

4. **테스트 C: 빈 영역** (영점 없는 구간):
   - R_empty = [0.3,0.7]×[γ₁+0.5, γ₂-0.5] (γ₁=14.13, γ₂=21.02 사이, 영점 불포함)
   - ∫∫ F₂ ≈ 0 확인 → F₂ 국소화의 정량적 증거

5. **테스트 D: 디리클레 확장** (보편성):
   - χ₅ = (·/5)에서 단일 영점 + 5영점 영역 반복
   - 짝수 지표에서도 면적분 작동 확인

6. **출력**: `results/gauss_bonnet_area_integral.txt`

**주의**:
- F₂ 계산에 L(s)의 σ-미분 필요 → 해석적 공식 우선 (bundle_utils.py). 수치 미분 시 δ≥0.01.
- h=10⁻²⁰ 수치 미분 금지. dps=100.
- 영점 위 직접 계산 금지 (특이점) → 2D 그리드가 영점 근방을 피하도록
- mpmath.diff(xi, s) 사용 금지. 해석적 L(s) = ξ'/ξ 공식 사용.
- 면적분이 윤곽 적분과 5% 이상 차이나면 → F₂ 정의 재검토

**성공 기준**:
- 단일 영점: |∫∫F₂ - 2π| / 2π < 5%
- k영점 영역: |∫∫F₂ - 2πk| / (2πk) < 5% (k=1,5,10,20)
- 빈 영역: |∫∫F₂| < 0.5
- **양성**: 위 3가지 모두 통과 → 결과 #24 "Gauss-Bonnet for ξ-bundle" 등록
- **부분 양성**: k영점은 통과하나 단일 영점 정밀도 부족 → 해상도 증가 후 재시도
- **음성**: k영점 면적분이 윤곽 적분과 불일치 → F₂ 정의 재검토
