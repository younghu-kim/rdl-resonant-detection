### 결과 #29 판정: 조건부 양성

**양성 근거**:
- Epstein zeta Z_Q(s)에서도 임계선 영점의 다발 서명(κ, mono, FWHM)이 리만 제타와 정량적으로 일치
- 프레임워크의 L-함수 계열 확장성 입증 (ζ → Dirichlet L → Epstein Z)
- λ=1과 λ=5의 서명 편차 <1% → 이차형식 무관 보편성

**미달성**:
- 1차 목표(σ≠1/2 영점 발견 → 서명 차이 정량화)는 탐색 범위 한계로 미달성
- 이것만으로는 falsifiability 테스트 불완전

**Devil's Advocate — 이것이 가치 없는 이유 3가지**:
1. 탐색 범위가 좁아서 off-critical zeros 없는 것은 부재 증거가 아님
2. on-critical 서명 동일성은 δ=0.03 오프셋 때문에 당연 (모든 영점이 비슷한 κ≈1/δ²)
3. λ=5도 탐색 범위 내에서는 RH 만족할 수 있어서 진짜 반례가 아님

**반론**: Devil's Advocate 포인트 2에 대해 — κ의 절대값이 1/δ²로 결정되는 것은 맞지만, on-critical 영점 간의 κ 상대 분산이 λ=1(σ=7.2%)과 λ=5(σ=4.7%)에서 작고 안정적이라는 점은 의미 있음. 또한 mono=2π가 정확히 유지되는 것은 위상적 불변량의 안정성을 보여줌.

**최종 판정**: 조건부 양성 — on-critical 확장성은 유효하나, falsifiability는 별도 실험(#30)으로 완성해야 함. 논문에는 "Epstein zeta 확장 + 한계 명시"로 반영.

## 다음 작업: 결과 #30 — Davenport-Heilbronn 함수 적대적 검증

**모델**: opus

**왜**: Davenport-Heilbronn (DH) 함수는 **off-critical zeros 존재가 수학적으로 보장**된 유일한 실용적 반례. Epstein에서 실패한 1차 목표를 여기서 달성해야 한다. 이것이 프레임워크 falsifiability의 최종 퍼즐 조각. 심사자가 "RH 위반 함수에서는 어떻게 되나?"라고 물었을 때, 이 결과 하나로 답변 가능.

**수학적 배경**:
DH 함수는 다음과 같이 정의:
```
f(s) = [(1-iκ)/2] · L(s,χ) + [(1+iκ)/2] · L(s,χ̄)
```
여기서 χ는 mod 5 원시 지표 (차수 4), κ = (√(10-2√5)-2) / (√5-1).
이 f(s)는 함수방정식 f(s) = ε·(5/π)^{1/2-s}·Γ((1-s+a)/2)/Γ((s+a)/2)·f̄(1-s̄)를 만족하지만, σ>1/2에 무한히 많은 영점이 존재 (Davenport & Heilbronn, 1936).

**구체 설계**:

1. **DH 함수 구현**:
   - χ mod 5 (차수 4): χ(1)=1, χ(2)=i, χ(3)=-i, χ(4)=-1, χ(0)=0
   - κ 계산: κ = (√(10-2√5)-2)/(√5-1)
   - f(s) = [(1-iκ)/2]·L(s,χ) + [(1+iκ)/2]·L(s,χ̄)
   - 검증: 함수방정식 수치 확인 (f(s) vs transform of f(1-s̄))

2. **영점 탐색**:
   - 임계선 영점: σ=1/2, t∈[1,60] 스캔 (Δt=0.1)
   - **Off-critical 영점**: σ∈[0.5,1.0], t∈[1,60] 밀도 높은 격자 (n_sigma≥30, n_t≥200)
   - |f(σ+it)|<threshold 점에서 Newton/findroot 정련 → |f|<10⁻²⁰
   - 문헌 참조: σ≈0.8 근처, t≈6-30 구간에 off-critical zeros 보고

3. **다발 진단** (핵심):
   - 접속: L_f = f'/f (대수 미분)
   - **곡률**: κ = |f'/f|² at δ=0.03 오프셋
   - **모노드로미**: ∮ d(arg f) 폐곡선 적분 (radius=0.5, 64단계)
   - **σ-국소화**: κ(σ) 프로파일 → FWHM, peak_σ

4. **핵심 비교 테이블**:

| 메트릭 | on-critical (σ=1/2) 예상 | off-critical (σ≠1/2) 예상 |
|--------|--------------------------|---------------------------|
| κ | ~1/δ² (기존과 동일) | 유사하거나 다를 수 있음 |
| mono/π | 2.00 | 2.00 (위상 보존이면) |
| FWHM | ~0.04 (좁음) | **넓어져야 함** (σ=1/2 집중 깨짐) |
| peak_σ | 0.50 | **σ₀ ≠ 0.5** (영점 위치에 피킹) |

가장 민감한 판별 지표는 **peak_σ**와 **FWHM**. off-critical 영점의 κ 프로파일이 σ=1/2가 아닌 실제 영점 위치에서 피킹하면 강한 양성.

**주의사항**:
- DH 함수는 오일러 곱이 없으므로 L-함수가 아님 — 논문에 명시
- f(s)는 일반 복소수 값 — |f| 최소화로 영점 탐색
- χ mod 5: 홀수 지표 (a=1, χ(-1)=χ(4)=-1)
- κ 측정 시 δ=0.03 오프셋 유지
- 모노드로미는 폐곡선 적분 (eps 차분 금지)
- 격자가 성기면 off-critical zeros를 놓침 — σ 방향 30+, t 방향 200+
- mpmath dps≥80 유지

**성공 기준**:
- off-critical 영점 1개+ 발견 AND 다발 서명(특히 peak_σ, FWHM)이 on-critical과 정량적으로 다름 → **강한 양성** (falsifiability 확립)
- off-critical 영점 발견했으나 서명이 같음 → **음성** (프레임워크가 구분 못 함 — 한계로 기재)
- off-critical 영점 못 찾음 → 구현 검증 후 범위 확대 재시도

- findroot: 단일 시작점만, 튜플 전달 금지
- numpy 2.0: np.trapezoid (trapz 아님)
- **bundle_utils.py h=10^{-20}**: dps ≥ 100 또는 해석적 공식 사용.
- **중간점 Hadamard 상쇄**: m=(γₙ+γₙ₊₁)/2에서 두 최근접 영점의 기여 = 0.
- **ξ(s) underflow**: log(ξ)' 해석적 공식으로 영구 해결.
- **mpmath.diff(xi, s)**: t>300에서 실패. 사용 금지.
- **Dirichlet κ 계산**: bare L'/L 사용 금지. Λ'/Λ 해석적 공식 필수.
- **윤곽 적분 시 영점 회피**: t₂는 반드시 영점 사이 중간점.
- **홀수 디리클레 지표 영점 탐색**: Re(L(1/2+it))=0 ≠ L=0. |L| 최소화 필수.
- **짝수 디리클레 지표**: a=0, Γ(s/2) 사용. L(1/2+it) ≠ 실수 (Λ만 실수). Re(L)=0 단독 불충분.
- **영점 탐색 보편적 방법**: |L(1/2+it)| 최소화가 Re(L)=0보다 엄밀히 우월 (A⊂B). 패리티 무관.
- **F₂ 면적분 정의**: ∂_σ Im(L)은 대칭 상쇄 → 격자 플라켓(arg ξ 와동 밀도) 사용. Stokes 등가.
- **σ-국소화 표현**: "RH의 기하학적 동치" 금지 → "RH와 일관적인 기하학적 증거" 수준으로. FWHM=0 → "FWHM < δσ".
- **다항식 log(0) 회피**: 격자점이 영점을 정확히 지나지 않도록 범위 조절.
- **Number variance unfolding**: L은 unfolded 좌표(평균 밀도로 정규화). raw t-간격 사용 금지.
- **Number variance sub-GUE**: t<600에서 GUE 수렴 기대 금지. Berry(1988) arithmetic correction 레짐. "GUE 재현" 표현 금지.
- **Epstein zeta 격자합**: 수렴 느림 — Richardson 외삽 또는 Chowla-Selberg 공식 사용 권장. cutoff N≥200.
- **DH 함수 χ mod 5**: χ(2)=i, χ(3)=-i, χ(4)=-1. 홀수 지표(a=1). off-critical zeros는 σ>0.5, t≈6-30 구간 보고.
