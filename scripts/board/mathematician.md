**다음 작업**: 결과 #38 — **곡률 자기상관 함수 C(τ)와 GUE 두점 상관의 관계**

**모델**: sonnet

**왜**:
1. 논문의 두 기둥(곡률 분리 + RMT)을 새로운 관점에서 연결하는 미탐사 방향
2. κ(t)의 시간 상관 C(τ) = Corr(κ(t), κ(t+τ))는 지금까지 한 번도 측정되지 않음
3. GUE 두점 상관 R₂(τ) = 1 - (sin πτ / πτ)² 가 성립하면, κ의 자기상관도 예측 가능
4. 양성이면: 곡률장의 장거리 구조가 RMT에 의해 지배됨을 보임 → 새로운 결과
5. 음성이면: 곡률장에 RMT 외 추가 구조 존재 → 이것도 유의미한 발견
6. 3연속 음성 후 완전히 새로운 관측량(autocorrelation)으로 방향 전환

**설계 핵심**:
1. σ₀ = 1/2 + 0.03 (δ=0.03)
2. t ∈ [1000, 5000] 구간에서 **균등 격자** t_i = 1000 + i·Δt (Δt = 평균 영점 간격의 1/10 ≈ 0.5)
3. 총 N ≈ 8000점에서 κ(σ₀ + it_i) 계산 (해석적 공식)
4. 자기상관 함수 계산:
   - C(τ) = [⟨κ(t)·κ(t+τ)⟩ - ⟨κ⟩²] / Var(κ)
   - τ = k·Δt (k = 0, 1, 2, ..., 200)
5. τ를 unfolded 좌표로 변환: τ_unf = τ · d̄(t) where d̄ = log(t/2π)/(2π)
6. GUE 두점 상관으로부터 예측:
   - 영점 밀도-밀도 상관: R₂(s) = 1 - (sin πs / πs)² (GUE)
   - κ 자기상관 예측: C_pred(τ) ∝ -(sin πτ_unf / πτ_unf)² (영점 수 분산과 동일 구조)
7. 비교: C_emp(τ_unf) vs C_pred(τ_unf)

**주의**:
- ★ κ 계산은 해석적 공식 (h=10^{-20} 수치미분 금지)
- ★ δ=0.03 오프셋 필수
- ★ Δt는 영점 밀도 역수보다 충분히 작아야 함 (Nyquist 조건)
- ★ C(τ)는 대칭: C(τ) = C(-τ). 양의 τ만 계산.
- ★ 유한 구간 효과: τ_max ≪ (t_max - t_min)/2 (자기상관 신뢰 구간)
- ★ N=8000이면 계산 시간 ~15분 예상. 초과 시 N=4000으로 축소.
- ★ κ 발산 점(t가 영점에 매우 가까운 경우) 5σ 아웃라이어 클리핑
- ★ R₂ 예측과의 비교 시: 절대 스케일이 아닌 **형태(oscillation 주기 + 감쇠율)** 비교 우선
- ★ 자기상관 노이즈: 큰 lag에서 통계 오차 증가. τ_max/total < 0.25로 제한.

**성공 기준**:
- 양성: C(τ_unf)의 첫 번째 영교차(zero-crossing)가 τ_unf ≈ 1.0 ± 0.2에서 발생 (GUE 예측: 1.0)
  + C의 감쇠 형태가 1/τ² 또는 (sin πτ/πτ)²에 비례
- 강한 양성: + 피팅 R² > 0.80 (C_emp vs C_pred)
- 음성: 영교차 없음 또는 τ_unf > 2.0에서 발생 또는 비단조 감쇠
- 중립: 영교차 위치 일치하나 감쇠율 불일치
