### 다음 작업: **합성 함수 대조 실험 (Synthetic Off-Critical Control)**

**모델**: sonnet

**왜**:

26개 결과 모두 양성 대조군이다. **음성 대조군이 하나도 없다.** 프레임워크가 ξ에서 σ=1/2 δ-집중을 보이지만, "영점이 σ≠1/2에 있는 함수에서도 같은 결과가 나오지 않는가?" 확인 없이는 방법론적 인공물 가능성을 배제할 수 없다.

이것은 **프레임워크 전체의 신뢰성**에 관한 실험이므로, 추가 확장(N=500, 새 L-함수)보다 우선한다.

**구체 설계**:

1. **합성 함수 정의** (단순 다항식, 영점 위치 정확히 알려짐):

   ```python
   # f₁: 모든 영점이 σ=1/2 위 (RH-유사)
   f1(s) = (s - (0.5+14i))(s - (0.5+21i))(s - (0.5+25i))
   
   # f₂: 일부 영점이 σ≠1/2 (RH-위반)
   f2(s) = (s - (0.3+14i))(s - (0.7+14i))(s - (0.5+21i))
   
   # f₃: 모든 영점이 σ≠1/2 (완전 RH-위반)
   f3(s) = (s - (0.3+16i))(s - (0.7+16i))(s - (0.6+22i))(s - (0.4+22i))
   ```

   NOTE: 함수 방정식 대칭 불필요. 다항식의 위상 구조(arg 와동)만 이용. 격자 플라켓은 arg(f)의 와동만 감지하므로 다항식이어도 정확히 작동.

2. **테스트 매트릭스 (11종)**:

   | 테스트 | 함수 | σ-범위 | t-범위 | 기대 N | 핵심 검증 |
   |--------|------|--------|--------|--------|-----------|
   | §A-1 | f₁ | [0.3, 0.7] | [10, 28] | 3 | 전체 영점 검출 |
   | §A-2 | f₁ σ-프로파일 | 200점 | [10, 28] | δ(σ=0.5) | RH-유사 패턴 |
   | §A-3 | f₁ | [0.6, 0.9] | [10, 28] | 0 | off-critical 빈 |
   | §B-1 | f₂ | [0.2, 0.8] | [10, 24] | 3 | 전체 영점 검출 |
   | §B-2 | f₂ σ-프로파일 | 200점 | [10, 24] | 3피크 | σ=0.3, 0.5, 0.7 분리 |
   | §B-3 | f₂ | [0.49, 0.51] | [10, 24] | 1 | σ=0.5 영점만 |
   | §B-4 | f₂ | [0.6, 0.8] | [10, 24] | 1 | σ=0.7 영점만 |
   | §B-5 | f₂ | [0.2, 0.4] | [10, 24] | 1 | σ=0.3 영점만 |
   | §C-1 | f₃ | [0.2, 0.8] | [12, 25] | 4 | 전체 영점 검출 |
   | §C-2 | f₃ σ-프로파일 | 200점 | [12, 25] | 2피크 | σ=0.3/0.7 + σ=0.4/0.6 |
   | §C-3 | f₃ | [0.49, 0.51] | [12, 25] | **0** | **임계선 빈 ← 핵심!** |

3. **구현**:
   - 기존 offcritical_gauss_bonnet.py의 격자 플라켓 코드 복사
   - `log_xi_imag(s)` 대신 `float(im(log(f(s))))` 사용 (mpmath.log + im)
   - 격자 해상도: Nσ=64, Nt=200 (다항식이므로 작은 격자 충분)
   - dps=30 (다항식은 수치 안정, 높은 정밀도 불필요)

4. **계산량 추산**: 다항식 연산 ~1000배 빠름. **총 ~5-10분**.

5. **출력**: `results/synthetic_offcritical_control.txt`

**sonnet 사용 이유**: 기존 플라켓 코드 복사 + 함수만 다항식으로 교체. 새 수학/알고리즘 불필요.

**주의**:
- 격자점이 영점을 정확히 지나면 log(0) 발생. t-범위를 영점의 Im 부분과 0.5 이상 간격 두도록 설정. 위 설계에서 이미 반영됨 (f₁ 영점 t=14,21,25 vs t-범위 [10,28]).
- 복소 로그 branch cut: mpmath.log의 기본 branch cut은 음의 실수축. 다항식 영점 근방에서 wrap이 올바르게 작동 확인 필요 → 플라켓 방법은 자동으로 wrap하므로 문제 없음.
- σ-프로파일: 각 σ값에서 t-방향 1D GB를 계산 (기존 offcritical_gauss_bonnet.py §B와 동일).

**성공 기준**:
- **완전 양성**: 11/11 ✅ — f₁은 δ(σ=0.5), f₂는 3개 σ 분리 검출, f₃은 임계선 빈
- **부분 양성**: f₁ OK + f₂/f₃에서 수치 오차 → 격자 해상도 증가 후 재시도
- **음성**: f₂/f₃에서 σ=1/2 δ-집중 → **프레임워크 신뢰성 심각 문제** (사실상 불가능 — 방법론은 Cauchy's argument principle이므로)

**논문 가치**: "σ-국소화 관측이 ξ의 고유 성질이지 방법론적 인공물이 아님"을 보증하는 **필수 대조 실험**. 특히 §C-3(f₃, 임계선 N=0)이 가장 결정적.

### 알려진 함정 (누적)

- 모노드로미: eps 차분 금지 → 폐곡선 적분(radius=0.5, 64단계)

- findroot: 단일 시작점만, 튜플 전달 금지
- numpy 2.0: np.trapezoid (trapz 아님)
- **bundle_utils.py h=10^{-20}**: dps ≥ 100 또는 해석적 공식 사용.
- **중간점 Hadamard 상쇄**: m=(γₙ+γₙ₊₁)/2에서 두 최근접 영점의 기여 = 0.
- **ξ(s) underflow**: log(ξ)' 해석적 공식으로 영구 해결.
- **mpmath.diff(xi, s)**: t>300에서 실패. 사용 금지.
- **Dirichlet κ 계산**: bare L'/L 사용 금지. Λ'/Λ 해석적 공식 필수.
- **윤곽 적분 시 영점 회피**: t₂는 반드시 영점 사이 중간점.
- **홀수 디리클레 지표 영점 탐색**: Re(L(1/2+it))=0 ≠ L=0. |L| 최소화 필수.
- **짝수 디리클레 지표**: a=0, Γ(s/2) 사용. L(1/2+it) ≠ 실수 (Λ만 실수). Re(L)=0 단독 불충분.
