### H-스케일링 재분석 (중립 → 이론적 진전, 5시드)

**실험 판정**: 중립 유지. 순수 O(1/H) 예측은 기각되었으나, 이성분 분해에서 새로운 이론적 구조 발견.

**이성분 모델 (추측, 2파라미터/5데이터점 한계)**:
```
1 - r_init = a/H + c_floor
  a ≈ 0.12 ± 0.01  (아키텍처 성분: 채널 간 위상 분산)
  c_floor ≈ 0.0045  (바닥 성분: H≥128에서 지배)
```

근거:
| H | 실측 | a/H 성분 | a 추정 |
|---|------|---------|--------|
| 16 | 0.01223 | 0.00773 | 0.124 |
| 32 | 0.00853 | 0.00403 | 0.129 |
| 64 | 0.00619 | 0.00169 | 0.108 |

a값이 H=16,32,64에서 0.108~0.129로 일관 → O(1/H) 성분 존재 시사.
H=128,256에서 바닥(floor)이 지배 → 1-r≈0.0045 포화.

**수학적 해석**:
- a/H 성분: 중심극한정리. H개 채널 위상의 평균 시 O(1/H) 분산 감소.
- c_floor: 입력-의존 위상 분산의 비가약적(irreducible) 하한. H→∞여도 제거 불가.
- 물리적 비유: 열잡음(1/H) + 양자잡음(floor). H를 늘려 열잡음은 줄이나 양자잡음은 남음.

**[DA 검토]**:
1. "floor가 in_features=128에 의존하면?" → 다른 in_features에서 floor 변동 가능. 보편 상수가 아닐 수 있음.
2. "순수 H^{-0.38}도 H=512에서 0.0032 예측 → 포화와 구별 불가?" → 맞음. 5점으로 두 모델을 구별 불가. 이성분 모델이 물리적으로 동기 부여되지만 통계적으로 미확립.
3. "floor가 없고 단순히 느린 멱법칙 꼬리?" → 배제 불가. H=512,1024 실험 필요.

**결론**: 순수 O(1/H) 주장은 기각. 이성분 모델은 물리적으로 자연스럽지만 5점/2파라미터로는 확립 불가. **추가 H값 없이 이 방향은 종료** — 논문에는 "α≈-0.4 약한 스케일링, H≥128 포화"로 보고하되 이성분 분해는 추측으로 부록 언급 정도.

### Spontaneous Gauge Fixing 명제 수정

**원본**: r_init ≈ 1 - O(1/H)
**수정**: r_init ≈ 1 - a/H - c_floor (추측). O(1/H) 성분은 존재하나 floor이 지배.
- U(1)^H → U(1) 자발적 대칭 파괴의 핵심 논거(r_init ≫ 1/√H)는 유지
- H=16에서도 r_init=0.988 vs 무작위 0.25 → 대칭 파괴는 구조적이고 보편적
- 학습은 κ를 정련: r_init→r_final에서 Δr ∝ 1/H (더 적은 초기 결핍 → 더 작은 정련량)

## 이론적 메모

### von Mises–Kuramoto–L_geo 삼각 연결
```
von Mises 분포: p(φ) ∝ exp(κ cos(φ - μ))
  ↕ MLE
L_geo = 1 - cos(Δφ)  (음의 로그우도)
  ↕ 최소화
Kuramoto r → 1  (순서 매개변수 최대화)
```
세 개념이 수학적으로 동일한 최적화 문제.

### 이성분 모델 수학적 기원 (추측)
Xavier 초기화에서 3층 ReLU 네트워크:
- z_j = Σ_k w_{jk}^(3) ReLU(Σ_l w_{kl}^(2) ReLU(Σ_m w_{lm}^(1) x_m))
- 각 z_j의 위상 φ_j = arg(z_j)
- **아키텍처 성분**: H개 z_j 평균 시 CLT → var(⟨e^{iφ}⟩) ~ σ²_single/H → 1-r ~ 1/H
  - a ≈ σ²_single ≈ 0.12 → σ_single ≈ 0.35 rad ≈ 20°. 각 채널의 위상 분산이 ~20°.
- **바닥 성분**: 입력 x에 따른 φ_j(x) 변동의 비가약적 부분
  - 모든 채널이 완벽히 동기화되어도 (1/H→0), x→φ(x) 매핑 자체의 불확실성은 남음
  - c_floor ≈ 0.0045 → σ_floor ≈ 0.095 rad ≈ 5.4°

### 평가 메트릭과 신호 품질의 분리 (사이클 4 신규)
```
L_geo의 두 가지 효과:
1. |F₂| ratio 개선: 0.913 vs 1.026 (영점에서 F₂가 더 작아짐)
   → 이것은 "신호 대 잡음비" 개선
   → ratio < 1이면 영점에서 체계적으로 더 작은 |F₂|

2. 극소값 기반 검출 성능: 차이 없음 (seed=42 기준)
   → |F₂| 풍경에 이미 모든 영점 위치에 극소값이 존재
   → L_geo는 극소값을 "깊게" 만들지만, 이미 존재하는 극소값의 탐지에는 영향 없음

수학적 함의:
- L_geo의 주효과는 |F₂| 풍경의 "contrast" 개선 (영점 vs 비영점)
- 그러나 이진 검출(영점인가 아닌가)에서는 contrast 개선이 recall에 무영향
- precision 개선에 기여하려면: FP의 |F₂| 극소값이 TP보다 얕아야 함
- 현재 P≈22-23%로 동일 → L_geo가 FP의 극소값 깊이를 변화시키지 않음

추측: L_geo ↔ von Mises NLL 연결의 주효과는 '위상 학습 안정성'이며,
     이것이 |F₂| ratio로 표현되지만, 극소값 존재/부재에는 무관.

사이클 10 업데이트 (3시드 결과 반영, 사이클 9 메커니즘 수정):

- L_geo의 유일하게 일관된 효과: **recall 하한 안정화**
  - BL min(R)=0.917 → L_geo min(R)=0.972. recall 변동폭 3배 축소.
- 수학적 해석: L_geo=von Mises NLL → 위상 학습의 분산 축소
  - BL(L_tgt)은 ±π 불연속에서 시드별 학습 궤적 분기 → recall 변동 큼
  - L_geo는 S¹ 측지 연속성 → 시드 간 학습 궤적 안정화 → recall 변동 작음
- 평균 F1에는 무영향 (-0.011, ~0.3σ) → "L_geo의 이점은 안정성이지 평균 성능이 아님"
- 사이클 9의 "L_geo→FP 증가" 가설은 seed=7 특이적. 일반화 불가.
```
