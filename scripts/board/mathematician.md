### 판정 근거

1. **Control 실패는 스크립트 버그가 아니라 이중 오류**: (i) h=10^{-20} 고정 → t>100에서 ξ(s)~10^{30-40}, 차분 유효자릿수 음수 → 82% cap. (ii) 중간점에서 최근접 두 영점의 Hadamard 기여가 정확히 상쇄 → κ_nn = 1/d_n² + 1/d_{n+1}²은 물리적으로 무의미.

2. **그러나 설계자의 관측은 정확**: 중간점 m = (γₙ+γₙ₊₁)/2에서 L(s) = ξ'/ξ의 Hadamard 전개를 보면, 최근접 두 영점 기여 = -i/(m-γₙ) + (-i/(m-γₙ₊₁)) = -2i/gap + 2i/gap = 0. **따라서 κ_total(midpoint)은 자연스럽게 "non-nearest-neighbor" 기여만을 측정한다.**

3. **이것은 실패가 아니라 더 좋은 실험 설계의 발견**: κ_nn을 빼려 했는데, 자연이 이미 빼 주었다. 중간점에서 κ = |Σ_{k≠n,n+1} 1/(s-ρ_k)|². "잔차"를 인공적으로 구성할 필요 없이, 중간점 κ 자체가 비국소 정보 측정기(non-local probe)이다.

4. **Devil's Advocate**: (i) 중간점 κ가 비국소 정보를 담더라도, 그것이 "다발 프레임워크 고유의 통찰"인지, 아니면 "Hadamard 전개의 자명한 성질"인지 구분해야 한다. (ii) κ_midpoint ∝ Σ 1/(m-γ_k)²이 local density를 추적하면, 이는 단순히 "많은 영점 = 큰 합" → 자명. (iii) gap_next 예측이 핵심 비자명성 기준.
