**구체적 실험 설계**:

H₀는 ξ'/ξ(s)의 Laurent 전개에서 0차항:
  ξ'/ξ(s) = 1/(s−ρ) + H₀ + H₁(s−ρ) + ...

ζ(s)의 경우 ξ'/ξ를 분해하면:
  ξ'/ξ(s) = 1/s + 1/(s−1) − ½log(π) + ½ψ(s/2) + (ζ'/ζ)(s)

Γ-인자 기여 (s=ρ에서 극 없음):
  H₀^Γ = 1/ρ + 1/(ρ−1) − ½log(π) + ½ψ(ρ/2)

산술 기여 (극 제거 필요):
  H₀^L = lim_{s→ρ} [(ζ'/ζ)(s) − 1/(s−ρ)]

전체: H₀ = H₀^Γ + H₀^L

**실험 단계**:
1. ζ(s) 12영점 각각에서 H₀^Γ 해석적 계산 (mpmath digamma)
2. H₀ 수치 계산: ξ'/ξ(ρ+ε) − 1/ε, ε = 10^{-10}~10^{-14} Richardson 외삽
3. H₀^L = H₀ − H₀^Γ 추출
4. 이론 A_Γ = Im(H₀^Γ)², A_L = A_measured − A_Γ 비교
5. χ₅²(even, q=5) 동일 분해 반복:
   H₀^Γ(χ₅²) = −½log(5/π) + ½ψ(ρ/2) (even이므로 μ=0, 감마 = Γ(s/2))

6. H₀^L(χ₅²) 추출 → #76의 "log(q/π)/2 = 13%" 불일치 원인 정량화
