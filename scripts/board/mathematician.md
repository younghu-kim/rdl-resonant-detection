**다음 작업**: **κ 잔차 분석 — 곡률의 비국소 영점 정보 검증**

**모델**: sonnet

**왜**:

기존 결과 #16에서 ρ(g_norm, κ_peak)=0.835를 확인했으나, 검토자가 정확히 지적했듯 이는 "Hadamard 곱의 자명한 귀결"일 수 있다. κ = |ξ'/ξ|² ≈ Σ 1/(t-γₙ)² 이므로, 최근접 영점이 κ를 지배하면 κ는 단순히 "가장 가까운 영점까지의 거리의 역제곱"에 불과하다.

**핵심 질문**: κ에서 최근접 영점 기여를 제거한 "잔차 곡률" κ_residual이 비국소적(non-local) 영점 정보를 인코딩하는가?

만약 κ_residual이 다음 영점 간격이나 국소 밀도를 예측하면 → **κ는 nearest-neighbor 이상의 정보를 담고 있다** (다발 프레임워크의 비자명한 기여). 만약 아니면 → **κ ≈ 1/(t-γ_nearest)²** (자명한 귀결, 그러나 이것도 유용한 음성 결과).

이 실험은 결과 #16에 대한 검토자의 "Hadamard 자명성" 비판을 정면으로 검증한다. 어느 결과가 나오든 논문에 쓸 수 있다.

**구체 실험 설계**:

1. **영점 수집**: mpmath.zetazero(k)로 k=1..235 (t∈[0,450]). 연속 영점 쌍 234개.

2. **중간점 측정**: 각 쌍 (γₙ, γₙ₊₁)의 중간점 m = (γₙ + γₙ₊₁)/2에서:
   - κ_total(½+im) = |ξ'(½+im)/ξ(½+im)|² (mpmath로 직접 계산, dps=50)
   - κ_nn = 1/(m-γₙ)² + 1/(m-γₙ₊₁)² (Hadamard 최근접 2항)
   - κ_residual = κ_total - κ_nn

3. **상관 분석**:
   - (a) **Control**: Spearman ρ(κ_total, gap). 예상 ρ ≈ -0.9 이상 (κ_total ∝ 1/gap² 자명). 이것이 통과해야 실험 신뢰.
   - (b) **Partial**: Spearman ρ(κ_residual, gap). κ_residual도 현재 간격에 상관이 있는가?
   - (c) **핵심**: Spearman ρ(κ_residual, gap_next). gap_next = γₙ₊₂ - γₙ₊₁. 잔차가 **다음** 간격을 예측하는가? (비국소 정보 검증)
   - (d) **밀도**: Spearman ρ(κ_residual, local_density). local_density = #{γ : |γ-m| < 5}. 잔차가 국소 밀도를 추적하는가?
   - (e) **지배도**: κ_nn/κ_total 비율의 평균, 표준편차, 분포. 최근접 항이 κ를 얼마나 지배하는가?

4. **추가**: κ_nn/κ_total < 0.8인 "비국소 지배" 점의 특성 분석 (있다면)

5. 결과 파일: `results/curvature_residual_nonlocal.txt`

**주의**:
- 중간점 m은 모든 영점에서 gap/2 이상 떨어져 있으므로 κ cap 문제 없음
- κ_total 계산: ξ'/ξ = d/ds log ξ(s). mpmath에서 ξ(s) = 0.5·s·(s-1)·π^(-s/2)·gamma(s/2)·zeta(s). 또는 직접 meijerg 사용. **bundle_utils.py의 기존 κ 계산 함수가 있으면 그것을 사용할 것.**
- Hadamard 전개: κ_nn에서 γₙ, γₙ₊₁만 사용 (γ̄ₙ = -γₙ 항은 t >> 1에서 무시 가능)
- dps=50 충분 (t < 450)
- 234개 데이터 포인트로 Spearman 검정 충분
- gap_next 계산 시 마지막 쌍은 제외 (n=234)

**성공 기준**:
- (a) |ρ(κ_total, gap)| > 0.8 → **control 통과** (필수)
- (c) |ρ(κ_residual, gap_next)| > 0.3, p < 0.01 → **양성** (κ에 비국소 정보 존재)
- (d) |ρ(κ_residual, density)| > 0.4 → **보조 양성**
- (e) κ_nn/κ_total 평균 > 0.95 → **nn 지배 확인** (음성이지만 정량적 결과)
- 기준 (c) 충족 → 결과 #18 양성 등록 ("κ residual encodes non-local zero structure")
- 기준 (c) 미충족 + (e) 확인 → 결과 #18 음성 등록 ("κ ≈ nearest-neighbor Hadamard, 95%+ nn-dominated")
- 기준 (a) 미충족 → **스크립트 버그**, 재실행
