**구체적 과제**:

Observation 3 (A-gap 보편적 음상관, ★★★★)의 이론적 메커니즘을 해석적으로 도출한다.

**증명할 핵심 부등식**:

Hadamard 분해에서 (Prop 11 기반):
```
A(γ₀) = S₁² + 2H₁
S₁ = -Σ_{k≠0} 1/(γ₀-γₖ) + Γ-terms   (부호 교대, NN 지배)
H₁ = Σ_{k≠0} 1/(γ₀-γₖ)²  + Γ'-terms  (항상 양)
```

수치 관측:
- S₁ + 2/Δ_R > 0이 GL(1)~GL(3) 모두에서 70-90%
- H₁이 A의 75-78% 기여 (H₁/A ≈ 0.7~0.8)
- A와 gap 사이 음상관 ρ ≈ -0.57

**해석적 도출 전략**:
1. **NN 지배 분석**: S₁ ≈ -1/Δ_L + 1/Δ_R + (smooth background). gap 축소 → |S₁| 증가 → S₁² 증가 → A 증가.
2. **H₁ 하한**: H₁ ≥ 1/Δ_R² + 1/Δ_L² (명백). gap 축소 → H₁ 폭증. 이것이 A-gap 음상관의 주된 원인.
3. **GUE 조건부 기대값**: E[A | gap=g]의 g-의존성을 pair correlation R₂(x)로 도출. Dyson kernel → 닫힌 형태 가능?
4. **ρ ≈ -0.57 보편성**: GUE spacing 분포와 Hadamard 급수의 결합에서 ρ값이 보편적으로 정해지는가?
