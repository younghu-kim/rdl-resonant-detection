### 구체적 실험 사양

**스크립트**: `scripts/gue_local_agap_c283.py` (C-277 기반 수정)

**핵심 변경** (C-277 → C-283):
1. A_bare 계산 시 **전체 고유값이 아닌 ±N_MAX 이웃만** 사용

3. Edge exclusion 대신 **trim 20%** (양쪽 20% 제거, 중앙 60%)
4. N = [200, 500, 1000, 2000], 앙상블 = {200:100, 500:50, 1000:30, 2000:20}
5. 비교용으로 A_bare_full(전체)과 A_bare_local(±300) **모두** 출력

**측정할 양**:
```
ρ_S(A_bare_full, gap_min)   # C-277 재현 (검증)
ρ_S(A_bare_local, gap_min)  # ←핵심: ±300 이웃 제한
```

**코드 구조 (의사코드)**:
```python
for each eigenvalue i (trim된 중앙 60%):
    # full (C-277 방식)
    S1_full = sum(1/(λ_i - λ_k) for k != i)
    H1_full = sum(1/(λ_i - λ_k)**2 for k != i)
    A_full = S1_full**2 + 2*H1_full
    
    # local (C-282b ζ(s) 방식)
    lo = max(0, i - N_MAX)
    hi = min(N-1, i + N_MAX)
    S1_local = sum(1/(λ_i - λ_k) for k in [lo,hi] if k != i)
    H1_local = sum(1/(λ_i - λ_k)**2 for k in [lo,hi] if k != i)
    A_local = S1_local**2 + 2*H1_local
    
    # gap_min (GUE 정규화)
    gap_raw = λ_{i+1} - λ_i
    gap_min = gap_raw / mean_spacing
```

**모델**: **sonnet** (C-277 코드 수정, 파라미터 변경 + local sum 추가. 수학적 판단 불필요.)

**왜**: 이 실험이 **전체 해석의 기반**:
- ρ_local ≈ -0.93 이면: GUE local ≈ ζ(s) local → "ζ(s) > GUE" 소멸, 국소성의 효과
- ρ_local ≈ -0.86 이면: GUE에서 국소 제한은 영향 미미 → ζ(s) 진짜 GUE 초과
- ρ_local ≈ -0.90 이면: 국소 효과 존재하지만 ζ(s)가 여전히 약간 강함

**주의**:

- N=200일 때 ±300이면 사실상 전체 → N≥500부터가 의미 있는 데이터
- gap_min은 unfolded spacing 사용 (C-277과 동일)
