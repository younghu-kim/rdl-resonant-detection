## 다음 작업: 결과 #35 — δ-sweep: κ 차선도항의 δ-의존성 검증

**모델**: sonnet

**왜**: 이것이 지금 가장 가치있는 이유:

1. **#32-34의 치명적 약점 해소**: 모든 결과가 δ=0.03 단일값. δ-sweep은 Lorentzian² 모델의 보편성을 직접 테스트.
2. **4사이클 arc 완결**: #32→#33→#34로 구조 해명. δ-sweep으로 보편성 확인하면 arc를 닫고 새 방향 전진 가능.
3. **이론 예측이 명시적**: f₁=δ²/(δ²+g²)²에서 δ는 명시적 파라미터. δ를 바꾸면 f₁ 형태 변경 → R² 유지 여부가 직접 테스트.
4. **논문 완결성**: δ-독립성 없이는 "차선도항 구조" 절의 설득력 부족.

**구체 설계**:

`scripts/curvature_delta_sweep.py` 신규 작성:

### 파트 A: 고정 영점 집합에서 δ-sweep

#34의 99개 영점 인덱스 그대로 사용. δ = {0.01, 0.02, 0.03, 0.05, 0.1} (5개).

각 (영점, δ) 쌍에서:
- s = 1/2 + δ + it_n
- κ(δ) = |ξ'/ξ(s)|² (해석적 공식: log(ξ)' = 1/s + 1/(s-1) - log(π)/2 + ψ(s/2)/2 + ζ'/ζ(s))
- Δκ(δ) = κ(δ) - 1/δ²
- NNS_min은 δ와 무관 (영점 간격 고정)

### 파트 B: δ별 1변수 회귀

각 δ에서: Δκ ~ a(δ)·log(t/2π) + b(δ)
보고: a(δ), R²_1var(δ)
**이론 예측**: Δκ ≈ (2Re[R]/δ) + Im[R]² 에서 Re[R]은 δ-독립 (Stirling 성분). 따라서 a(δ) ∝ 1/δ.
**검증**: a(δ)·δ가 상수(=약 0.048)인지 점검.

### 파트 C: δ별 2변수 회귀 (Lorentzian²)

각 δ에서: Δκ ~ a·log(t/2π) + b + c·δ²/(δ²+NNS_min²)²
보고: R²_2var(δ), c(δ)
**핵심 질문**: R²_2var(δ)가 모든 δ에서 > 0.88인가?

### 파트 D: 통합 회귀 (5δ × 99영점 = 495 데이터)

전체 495점에서 단일 회귀:
Δκ ~ (a/δ)·log(t/2π) + b/δ² + c·δ²/(δ²+NNS_min²)²
자유 파라미터 3개(a, b, c). **성공 기준**: 통합 R² > 0.90.

### 파트 E: δ→0 극한 진단

δ=0.01에서 수치 정밀도 이슈 진단:
- 1/δ²=10000. Δκ = κ - 10000 → 유효 자릿수 감소 위험.
- dps=80으로 상향. R²가 급락하면 수치 아티팩트 보고.

**주의사항**:
- 영점 좌표 + NNS는 한 번만 계산 (99회 zetazero). δ별 κ만 재계산: 99×5=495회.
- dps = max(50, 30+t/200). δ=0.01은 dps=80.
- ξ'/ξ 해석적 공식: #33/#34와 동일. ζ'/ζ: h=1e-6 중앙 차분.
- 총 시간 추산: ~25-35분 (고 t 영점이 느림).
- N<5(t<33) 제외.

**성공 기준**:
- **강한 양성**: 5개 δ 전부 R²_2var > 0.88, 통합 R² > 0.90, a(δ)·δ 변동 <20%
- **양성**: 4/5 δ에서 R²_2var > 0.80, 통합 R² > 0.85
- **음성**: 3개 이상 δ에서 R²_2var < 0.70 → δ-특이적 아티팩트
