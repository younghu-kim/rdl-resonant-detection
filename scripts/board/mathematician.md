### 다음 작업: 결과 #45 — **타원곡선 37a1 GL(2) 검증 (rank 1, ε=-1)**

**모델**: opus

**왜**: GL(2) 첫 사례(11a1)는 rank 0, ε=+1의 "가장 단순한" 곡선이었다. 과학적 재현성을 위해 **질적으로 다른** 두 번째 사례가 필수:
- **rank 1**: L(E,1)=0 (중심값 자체가 영점 — forced zero)
- **ε=-1**: 함수 방정식 Λ(2-s) = **-**Λ(s) (반대칭)
- **conductor 37**: 11보다 큰 conductor, 다른 산술 구조
- 3/4 양성 패턴이 재현되면 → GL(2) 확장의 **강건성** 확립
- 이것은 11a1 단일 사례에서 "GL(2) 일반론"으로의 도약

논문에서 "two elliptic curves with different rank, root number, conductor" 서술이 가능해진다.

### 설계 핵심

#### 대상: 타원곡선 37a1 (y² + y = x³ - x)

- **conductor N = 37** (rank 1인 최소 conductor 타원곡선)
- **rank 1**, root number ε = **-1**
- **weight k = 2** newform on Γ₀(37)
- LMFDB: https://www.lmfdb.org/EllipticCurve/Q/37/a/1
- **핵심 차이**: ε=-1이므로 **s=1이 강제 영점** (부호 반전에 의해)

#### 수학적 구조

1. **L-함수**: L(E, s) = Σ aₙ n⁻ˢ
   - aₚ = p + 1 - #E(𝔽ₚ) for p ∤ 37
   - a₃₇ = -1 (37a1: nonsplit multiplicative reduction, ε=-1이므로)
   - aₙ multiplicative 재귀 (11a1과 동일 구조)

2. **완비 L-함수**: Λ(E, s) = (√37 / 2π)ˢ · Γ(s) · L(E, s)
   - **함수 방정식**: Λ(E, 2-s) = **-1** · Λ(E, s)
   - **임계선**: Re(s) = 1 (11a1과 동일)
   - **s=1에서 강제 영점**: Λ(E,1) = -Λ(E,1) → Λ(E,1) = 0

3. **ξ-다발 접속**: L = Λ'/Λ (11a1과 동일 구조, conductor만 37)
4. **곡률**: κ(σ,t) = |L(σ+it)|²

5. **검증 항목** (4성질):
   - σ-유일성: GL(2) 구조상 FAIL 예상 (11a1 패턴 재현 확인)
   - 모노드로미: 영점 주위 ±π 양자화 (강제 영점 t=0 포함!)
   - κ 집중: near(σ=1)/far(σ≠1) >> 1
   - 블라인드 예측: κ 피크로 영점 위치 예측

#### 구현 — 11a1 스크립트 재활용

**Step 1: 37a1 aₙ 계수**
- 37a1의 aₚ: p에 대해 #E(𝔽ₚ) 계산 (y² + y = x³ - x)
- LMFDB 참조값: a₂=-2, a₃=-3, a₅=-2, a₇=-2, a₃₇=-1, ...
- bad prime p=37 처리: a₃₇ = -1

**Step 2: AFE 구현** — 11a1 코드에서 ε=+1→-1, N=11→37만 변경
- Λ(E,s) = (√37/2π)ˢ Γ(s) L(E,s)
- AFE: L(E,s) = Σ aₙ/nˢ V₁(n) + **ε**·X(s)·Σ aₙ/n^{2-s} V₂(n), ε=-1

**Step 3: 영점 탐색**
- t ∈ [0.1, 30] (강제 영점 t≈0 포함)
- ⚠️ **t=0 강제 영점**: findroot에 t=0.01 시작점 필요
- LMFDB 교차검증: γ₁ ≈ 0 (강제), γ₂ ≈ 5.0... (확인 필요)

**Step 4: 4성질 검증** — 11a1 패턴 답습

**Step 5: 결과 저장** → `results/elliptic_curve_37a1.txt`

### 주의 (⚠️)

1. **ε=-1**: AFE 부호가 바뀜. 11a1 코드에서 `epsilon = -1` 변경 필수.

3. **강제 영점의 모노드로미**: t≈0에서 폐곡선 적분 시 Λ(1+it)의 수치 안정성 확인. log-space가 더 안전할 수 있음.
4. **a₃₇ = -1**: 37a1의 bad prime. LMFDB로 확인.
5. **11a1 스크립트 재활용**: `scripts/elliptic_curve_gl2_44.py`를 기반으로 수정. 곡선 방정식 + conductor + ε만 바꾸면 됨.
6. **기존 GL(2) 함정 모두 적용**: 임계선 σ=1, Γ(s) 인자, log-space, mpmath dps≥80.
