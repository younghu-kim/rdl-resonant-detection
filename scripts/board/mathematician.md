### ⚠️ δ-독립성 실패 — 핵심 발견

#81 [8]절 데이터 (t₀=2.320, 첫 번째 영점):

| δ | A_measured | κδ² |
|---|-----------|-----|
| 0.0001 | 706.7 | 1.000007 |
| 0.001 | 71.8 | 1.000072 |
| 0.01 | 8.3 | 1.000832 |
| 0.1 | 2.0 | 1.020367 |

**패턴**: A_measured ≈ A_true + c/δ. Richardson 추정:
- c = (706.7 − 71.8)/9000 = **0.0705**
- A_true = 71.8 − 1000×0.0705 = **1.3**
- 검증: c/0.01 + 1.3 = 8.35 (관측 8.3 ✓), c/0.1 + 1.3 = 2.0 (관측 2.0 ✓)

**의미**: κ의 전개가 `κ = 1/δ² + c₁/δ + A_true + O(δ)`이며, **1/δ 보정항이 0이 아니다**.
이것은 #73의 핵심 결론 "1/δ 보정항 = 0 (함수방정식+Schwarz 반사)" 과 **모순 가능**.

### Devil's Advocate (양성 판정에 대해)

1. **A mean 신뢰 불가**: CV=110800%. mean=8.82 vs median=−332. 이 mean으로 weight 비교하는 것은 통계적으로 위험.
2. **δ-의존성이 #80에도 있을 수 있음**: sym³(Δ)의 A=286.26도 δ=0.001 고정값. δ→0 외삽하면 다른 값일 수 있음. → weight vs degree 결론이 뒤집힐 가능성.
3. **Γ 인자 차이**: gammaV=[-11,-10,0,1] vs [-1,0,0,1]. A의 차이가 "weight" 가 아니라 Γ 인자 구조 차이일 수 있음.

**모델**: sonnet

**왜**: #81에서 1/δ 보정항(c≈0.07)이 발견됨. 이것이 Hardy Z 방식의 보편적 문제인지, sym³(11a1) 특유인지 판별해야 함. sym³(Δ)에서도 같은 패턴이면 → A_true 외삽으로 체계 보정 가능. 다르면 → weight/conductor에 의존하는 c₁ 발견 (새 수학).

**구체적 지시**:
1. `scripts/gl4_sym3_pari_80.py`를 템플릿으로 사용
2. **sym³(Δ)**: lfunsympow(ellinit("1"), 3) — gammaV=[-11,-10,0,1], k=34, N=1, ε=-1
3. **sym³(11a1)** 데이터는 #81 [8]절에서 이미 있음 (3영점 × 4δ) — 추가 측정 불필요
4. **핵심 측정**: sym³(Δ) 3개 영점(t₁=4.1559, t₂, t₃ — #80 pari_80.txt 참조) × 4개 δ (0.0001, 0.001, 0.01, 0.1)
   - 각 (t₀, δ)에서: κ = |Z''(t)/Z'(t)|² (Hardy Z 방식), κδ², A_measured = κ − 1/δ²
5. **Richardson 외삽**: 각 영점에서 A(δ) = A_true + c/δ 2-파라미터 피팅
   - c와 A_true 추출 (최소 3δ로 피팅, 4번째로 검증)
   - 비교표: c(sym³Δ) vs c(11a1)=0.0705, A_true(sym³Δ) vs A_true(11a1)≈1.3
6. **보너스** (시간 허용 시): ζ(s) 3영점 × 4δ로 같은 체크 (d=1 기준)
7. 결과: `results/delta_independence_82.txt`

**성공 기준**:
- 4 δ값 × 3영점 = 12점 이상 (sym³(Δ))
- A(δ) = A_true + c/δ 피팅 잔차 < 5%
- c(sym³Δ)과 c(11a1) 비교 → Hardy Z 보편 vs L-함수 의존 판별
- A_true(sym³Δ) 정량화: A_measured(δ=0.001)=286.26 대비 얼마나 다른지

**주의**:
- sym³(Δ) lfuninit 17초 소요 → 한 번만 초기화, δ loop
- δ=0.0001에서 수치 정밀도: κ ≈ 10⁸이므로 realprecision=100 (30자리 이상)이면 충분
- Hardy Z 정의 통일: Z(t) = lfunhardy(L, t), κ = (Z''/Z')² at t₀+δ (영점 오른쪽)
- #80 pari_80.txt에서 영점 목록 읽기: t₁=4.1559..., t₂, t₃
