### 핵심 발견: κ_near = 1/δ² + O(1/δ) 이론

영점 s₀=1/2+it₀에서 δ=0.03 떨어진 점에서:
```
conn = Λ'(s)/Λ(s) ≈ 1/(s-s₀) + 2γ'(s₀)/γ(s₀) + L''(s₀)/L'(s₀)
κ = |conn|² ≈ 1/δ² + O(1/δ) + O(1)
```

1/δ² = 1/(0.03²) = **1111.11** — 모든 κ_near 값의 지배항!
