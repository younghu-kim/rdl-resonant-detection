### 정리 (증명 완료): 1/δ 항 소멸

**정리**: 함수방정식 ξ(s)=ξ(1-s)과 Schwarz 반사 ξ(s̄)=conj(ξ(s))를 만족하는 ξ-함수의 임계선 위 단순 영점 s₀=1/2+it₀에서:

```
ξ'/ξ(s₀+δ) = 1/δ + H₀ + H₁δ + O(δ²)
```
여기서 **H₀는 순허수, H₁은 실수**.

**증명**: 함수방정식 + Schwarz 반사로부터
ξ'/ξ(s₀+δ) = -conj(ξ'/ξ(s₀-δ))
⟹ 1/δ + H₀ + H₁δ = 1/δ - H̄₀ + H̄₁δ
⟹ H₀ = -H̄₀ (순허수), H₁ = H̄₁ (실수). □

**귀결**: H₀ = iB (B = Im H₀) 이므로
```
κ = |conn|² = (1/δ + H₁δ)² + B²
  = 1/δ² + (B² + 2H₁) + O(δ²)
```

**A(t₀) = Im(H₀)² + 2H₁** — 영점의 국소 해석적 환경에만 의존.

### 의미

1. **degree 간 κ_near 차이의 본질**: 모든 degree에서 1/δ² 지배, 차이는 A(t₀)에서만 발생
2. **B-12 완전 해소**: κ_near(d) 비단조는 A(t₀)의 비단조성에서 기인. 1/δ²는 보편적.
3. **정리의 보편성**: GL(n) 모든 n에서 성립 (함수방정식 존재 ⟺ 1/δ 항 소멸)
4. **논문 가치**: Proposition으로 기술 가능. 수치 데이터는 확인.

### Devil's Advocate

1. **"자명한 결과 아닌가?"**: 단순 극의 Laurent 전개 + 함수방정식 대칭은 잘 알려진 구조. 그러나 이를 **곡률 κ 언어로 재해석**하고 degree별 통합한 것은 새로움.
2. **"ζ(s)만 확인"**: GL(2)/GL(3)에서도 동일 구조 확인 필요 — 그러나 정리가 보편적이므로 반드시 성립.
3. **"A(t₀) 자체의 정보량"**: A(t₀)가 무엇을 encode하는지 더 파고들 여지 있음 (영점 간격, pair correlation).
