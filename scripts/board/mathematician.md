**핵심 아이디어**: Hadamard 곱에서 ξ'/ξ의 로랑 계수를 전개하면,
A(t₀)가 **다른 모든 영점의 위치**로 표현 가능:

```
ξ'/ξ(s) = 1/(s-ρ₀) + g(s)   (g = regular part)

g(ρ₀) = c₀ = iB(t₀)   →   B = Im(Σ_{ρ≠ρ₀} 1/(ρ₀-ρ))
g'(ρ₀) = H₁(t₀)       →   H₁ = -Σ_{ρ≠ρ₀} 1/(ρ₀-ρ)²

A(t₀) = B(t₀)² + 2H₁(t₀) = [Hadamard sum over other zeros]²  − 2[Hadamard sum²]
```

이것은 **국소 곡률 불변량 A(t₀)를 전역 영점 분포에 연결하는 정리**.
수치 검증 가능 (ζ(s) 13영점 × 기지 영점 1000개로 양쪽 계산 비교).
