**다음 작업**: **C-374 — Monodromy Theorem 엄밀 증명 구성**

ξ-다발 프레임워크에서 monodromy = 2π가 "왜" 정확히 2π인지를 엄밀하게 증명하라.

**증명 전략**:

1. **설정**: L-함수 ξ(s)가 σ=1/2 위에 단순 영점 ρ₀ = 1/2 + it₀을 가짐.
   접속 L(s) = ξ'(s)/ξ(s). 곡률 κ(s) = |L(s)|².

2. **핵심 관찰**: ρ₀ 근방에서 Laurent 전개
   L(s) = 1/(s - ρ₀) + a₀ + a₁(s-ρ₀) + ...
   
3. **Monodromy 정의**: σ=1/2 위에서 연속 영점 ρₖ, ρₖ₊₁ 사이의 위상 변화
   Δφ = ∫_{tₖ}^{tₖ₊₁} Im[L(1/2 + it)] dt
   
4. **증명 목표**: Δφ = π (반주기) 또는 전체 순환 = 2π를 보이기.
   - Argument principle: (1/2πi) ∮_C L(s) ds = (ξ 내부 영점 수)
   - σ=1/2 위의 적분은 이 윤곽의 한 변
   - FE 대칭으로 반대편 기여 동일 → 합하면 2π per zero

5. **확인할 사항**:
   - 이것이 자명한 재서술(tautology)인지 vs 비자명 정리인지 반드시 판정
   - 단순 영점 가정이 필수인지, 중복 영점에서 어떻게 되는지
   - 기존 문헌(Iwaniec-Kowalski, Titchmarsh)에 동치 진술이 있는지

6. **산출물**:
   - 정리(Theorem) + 증명(Proof) 문장을 LaTeX 형식으로 작성
   - 자명성 판정 결과 보고
   - 비자명이면 → 논문 Proposition으로 승격 위치 결정

**모델**: opus (수학적 증명 구성, 자명성 판정 필요)

**왜**: 세 논문이 모두 "계산 관측"에 기반함. 하나의 엄밀한 정리가 추가되면 수학 저널 수용 가능성이 극적으로 증가. Monodromy = 2π는 (a) 완벽하게 관측됨 (편차 0), (b) argument principle에서 따를 가능성 높음, (c) 비자명이면 프레임워크의 수학적 정당성을 확립.

**주의**:
- 자명성 함정 경고: "단순 영점이면 winding=1" 이 argument principle 그 자체의 재서술이면 가치 없음. 반드시 **무엇이 새로운지** 명시할 것.
- 비자명성의 원천: ξ-다발의 접속이 L=ξ'/ξ로 정의되고, 곡률이 |L|²이라는 **기하학적 해석**이 새로운 것. 순수 해석학적으로는 자명해도, 미분기하학적 프레임워크에서의 정리로서 가치가 있을 수 있음.
- `scripts/skill_library/` 내 기존 코드 참조 (특히 ξ'/ξ Laurent 전개 관련)
- 실패 시: "이 정리는 argument principle의 직접 귀결이므로 Remark로 기술" 결론도 유효한 산출물

**성공 기준**:
1. Monodromy = 2π의 엄밀한 증명 문장 (LaTeX)
2. 자명/비자명 판정 + 근거
3. 논문 내 위치 결정 (Theorem/Proposition/Remark)
