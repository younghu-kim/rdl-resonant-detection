### B-01: σ-유일성 FAIL = GL(n) 지문 ⭐⭐⭐ 핵심 발견 (원인 규명)
- **현상**: GL(2), GL(3) 모든 사례에서 σ-유일성 FAIL
- **FAIL 패턴 비교**:
  - GL(1): σ=0.5 극대 (PASS)
  - GL(2): 평탄 (σ 무관, FAIL)
  - GL(3): σ 단조 증가 {0.3:25, 0.5:23, 0.7:25, 0.9:27} (FAIL)
  - 세 패턴 전부 다름 → **L-함수 분류기** 확인
- **⭐ 원인 규명 (#66b, 2026-04-17)**:
  - 5-Level 역보정 실험: Λ, Λ/|γ|, Λ/γ, L(s), Phase 전부 FAIL
  - **감마 인자가 원인이 아님** ("렌즈 효과" 가설 기각)
  - **순수 L(s)의 부호변환**: σ=0.3→29, σ=0.5→19, σ=0.7→1
  - **진짜 원인**: Dirichlet 급수 Σc(n)/n^s 의 수렴 특성
    - σ 낮을수록 급수 느리게 수렴 → 진동 많음 → 부호변환 증가
    - 이는 L-함수의 **산술적 구조**(계수 분포)에서 비롯
  - → "고칠" 대상이 아니라 **읽어야 할 정보**
- **파생**: Γ-인자 메커니즘 소절 (#47) + 급수 수렴 메커니즘 추가 필요

### B-05: 공명 지표 가설 → 급수 수렴 지표 가설로 재해석 ⭐⭐
- **원래 가설**: R(n) = f_gamma(n) / f_zero_density, 감마 인자 진동이 원인
- **#66b에 의한 수정**: 감마 인자가 원인이 아님. L(s) 급수 자체의 수렴 특성이 원인.
- **새 가설**: GL(n)의 σ-유일성 FAIL 정도는 Dirichlet 급수 degree와 계수 분포에 의존
  - degree 1 (GL(1)): 급수 수렴 빠름 → σ 의존성 약함 → σ=0.5 특이성 유지 (PASS)
  - degree 2 (GL(2)): 급수 수렴 중간 → σ 의존성 상쇄 → 평탄 (FAIL)
  - degree 3 (GL(3)): 급수 수렴 느림 → σ 의존성 강함 → 단조 증가 (FAIL)
- **정량화 과제**: degree d L-함수의 부호변환 수 S(σ,d)의 σ-의존성 공식 유도
  - S(σ,d) ~ d·f(σ) + g(계수 분포) 형태?
- **검증 로드맵 (degree 1→5)**:
  - degree 1 (GL(1)): ✅ PASS 확립
  - degree 2 (GL(2)): ✅ 평탄 FAIL 확립. Maass form (#69)으로 비정칙 확인 예정
  - degree 3 (GL(3)): ✅ 단조 증가 FAIL 확립 (#63-#68)
  - degree 4 (GL(4)): ✅ #80 sym³(Δ) — Hardy Z 기반, 31영점, A(d=4)=286.26
  - degree 5 (GL(5)): ⏳ #72 sym⁴ L-함수로 측정 예정
  - **#73**: 5점 데이터로 S(σ,d) 정량 공식 유도 — degree d의 함수로 σ-FAIL 패턴 기술
- **수렴속도 메커니즘 규명 (#73)**:
  - Dirichlet 급수 부분합 S_N(σ) = Σ_{n≤N} c(n)/n^σ 의 잔차를 degree별로 측정
  - 계수 성장률: |c(n)| ≤ d(n)·n^((d-1)/2) (Ramanujan bound) → degree ↑ → 계수 폭발 → 수렴 지연
  - σ-의존 수렴 잔차 R(N,σ,d) 스케일링 법칙 추출
  - 부호변환 수 S(σ) 와 R(N,σ,d) 사이의 인과 다리 구축
  - **이것이 빠지면 #74 공식이 경험적 피팅에 그침 — 수학적 설명력의 핵심**
- **5점 확보 + 수렴속도 규명 시 기대**: degree에 따른 체계적 법칙 + 원인 메커니즘 → L-함수 분류론의 새 불변량 제안
- **가치**: 별도 논문급. "왜 FAIL하는가"에 대한 정량적 답 + 인과적 설명 + GL(n) 보편 분류 체계.

### B-08: RH ↔ BSD 구조적 연결 ⭐⭐⭐ (가설)
- **출발**: RH (영점이 **어디**에 있는가) → ξ-다발 프레임워크 구축
- **발견**: σ-FAIL 패턴 = L-함수의 해석적 행동이 산술적 구조(degree, 계수 성장)를 인코딩
- **평행**: BSD (영점이 **몇 개**인가) = L-함수의 해석적 행동이 산술적 구조(rank)를 인코딩
- **RDL의 위치**: "영점 주변 행동이 **어떤 모양**인가" = σ-프로파일 S(σ,d)
  - RH: 영점의 위치 (Re(s)=1/2)
  - BSD: 영점의 개수 (ord_{s=1} L(E,s) = rank)
  - RDL: 영점 주변의 σ-의존 구조 (S(σ,d) = degree의 함수)
- **공통 본질**: L-함수가 산술을 인코딩하는 방식의 서로 다른 단면
- **검증**: #71~#74에서 S(σ,d)가 체계적 법칙으로 나오면 RH↔BSD를 잇는 다리 가능성
- **주의**: 현재는 가설 단계. 데이터 5점(degree 1~5) 확보 후 판정

### B-23: Hardy Z κ vs ξ-bundle κ 정체 판명 (신규 #82, 강화 #83, 확립 사이클 #149)
- **현상**: ξ-bundle κ (σ-방향)와 Hardy Z κ (t-방향)는 물리적으로 다른 양.
- **차이**:
  - ξ-bundle (σ-방향): κ = 1/δ² + A(t₀) + O(δ²), **c₁=0 증명됨** (#73)
  - Hardy Z (t-방향): κ = 1/δ² − (Z''/Z')·(1/δ) + ..., **c₁ = −Z''/Z' ≠ 0**
- **수치 확인**: Hardy Z A값과 ξ-bundle A값 완전 역전
  - sym³(Δ): Hardy Z A=286 vs ξ-bundle A=2.77
  - sym³(11a1): Hardy Z A=8.82 vs ξ-bundle A=10.66
- **상태**: ★★ 확립. #84에서 논문 반영 완료.

### B-24: Hardy Z κδ² 정밀도 degree-역상관 (신규, 2026-04-18 #85)
- **현상**: Hardy Z κδ² 개별 편차가 degree 증가와 함께 악화.
