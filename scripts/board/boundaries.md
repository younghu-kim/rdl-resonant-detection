# 경계 추적 보드 (Boundary Tracker)

> 프레임워크가 "작동하는 영역"과 "깨지는 영역"의 경계 목록.
> 각 경계 = 잠재적 후속 논문 씨앗. 경계에서 멈추지 말고 파고들 것.

## 제1논문에서 발견된 경계

### B-01: σ-유일성 FAIL = GL(n) 지문 ⭐⭐⭐ 핵심 발견 (원인 규명)
- **현상**: GL(2), GL(3) 모든 사례에서 σ-유일성 FAIL
- **FAIL 패턴 비교**:
  - GL(1): σ=0.5 극대 (PASS)
  - GL(2): 평탄 (σ 무관, FAIL)
  - GL(3): σ 단조 증가 {0.3:25, 0.5:23, 0.7:25, 0.9:27} (FAIL)
  - 세 패턴 전부 다름 → **L-함수 분류기** 확인
- **⭐ 원인 규명 (#66b, 2026-04-17)**:
  - 5-Level 역보정 실험: Λ, Λ/|γ|, Λ/γ, L(s), Phase 전부 FAIL
  - **감마 인자가 원인이 아님** ("렌즈 효과" 가설 기각)
  - **순수 L(s)의 부호변환**: σ=0.3→29, σ=0.5→19, σ=0.7→1
  - **진짜 원인**: Dirichlet 급수 Σc(n)/n^s 의 수렴 특성
    - σ 낮을수록 급수 느리게 수렴 → 진동 많음 → 부호변환 증가
    - 이는 L-함수의 **산술적 구조**(계수 분포)에서 비롯
  - → "고칠" 대상이 아니라 **읽어야 할 정보**
- **파생**: Γ-인자 메커니즘 소절 (#47) + 급수 수렴 메커니즘 추가 필요

### B-02: Epstein zeta 조건부 양성 (#29)
- **현상**: 함수방정식 없는 L-함수에서 프레임워크 부분 작동
- **분류**: 새 수학 필요 (함수방정식 의존성의 정확한 역할)
- **후속 가능성**: "언제 작동하고 언제 안 되는가"의 필요충분조건

### B-03: GUE 연결 실패 (#36-38)
- **현상**: spacing ratio, 곡률-GUE 분포, κ 자기상관 모두 음성
- **분류**: 구조적 한계 또는 접근법 오류
- **후속 가능성**: 다발 기하학 ↔ 랜덤 행렬 이론의 올바른 다리는 무엇인가

### B-04: 저 t 영역 FP (#57 χ mod 7)
- **현상**: t<17에서 5개 FP (off-line zero capture)
- **분류**: 수치적 한계 (conductor 효과)
- **후속 가능성**: 고 conductor에서 저 t FP 소멸 여부 확인

### B-05: 공명 지표 가설 → 급수 수렴 지표 가설로 재해석 ⭐⭐
- **원래 가설**: R(n) = f_gamma(n) / f_zero_density, 감마 인자 진동이 원인
- **#66b에 의한 수정**: 감마 인자가 원인이 아님. L(s) 급수 자체의 수렴 특성이 원인.
- **새 가설**: GL(n)의 σ-유일성 FAIL 정도는 Dirichlet 급수 degree와 계수 분포에 의존
  - degree 1 (GL(1)): 급수 수렴 빠름 → σ 의존성 약함 → σ=0.5 특이성 유지 (PASS)
  - degree 2 (GL(2)): 급수 수렴 중간 → σ 의존성 상쇄 → 평탄 (FAIL)
  - degree 3 (GL(3)): 급수 수렴 느림 → σ 의존성 강함 → 단조 증가 (FAIL)
- **정량화 과제**: degree d L-함수의 부호변환 수 S(σ,d)의 σ-의존성 공식 유도
  - S(σ,d) ~ d·f(σ) + g(계수 분포) 형태?
- **검증 로드맵 (degree 1→5)**:
  - degree 1 (GL(1)): ✅ PASS 확립
  - degree 2 (GL(2)): ✅ 평탄 FAIL 확립. Maass form (#69)으로 비정칙 확인 예정
  - degree 3 (GL(3)): ✅ 단조 증가 FAIL 확립 (#63-#68)
  - degree 4 (GL(4)): ⏳ #71 Rankin-Selberg convolution으로 측정 예정
  - degree 5 (GL(5)): ⏳ #72 sym⁴ L-함수로 측정 예정
  - **#73**: 5점 데이터로 S(σ,d) 정량 공식 유도 — degree d의 함수로 σ-FAIL 패턴 기술
- **수렴속도 메커니즘 규명 (#73)**:
  - Dirichlet 급수 부분합 S_N(σ) = Σ_{n≤N} c(n)/n^σ 의 잔차를 degree별로 측정
  - 계수 성장률: |c(n)| ≤ d(n)·n^((d-1)/2) (Ramanujan bound) → degree ↑ → 계수 폭발 → 수렴 지연
  - σ-의존 수렴 잔차 R(N,σ,d) 스케일링 법칙 추출
  - 부호변환 수 S(σ) 와 R(N,σ,d) 사이의 인과 다리 구축
  - **이것이 빠지면 #74 공식이 경험적 피팅에 그침 — 수학적 설명력의 핵심**
- **5점 확보 + 수렴속도 규명 시 기대**: degree에 따른 체계적 법칙 + 원인 메커니즘 → L-함수 분류론의 새 불변량 제안
- **가치**: 별도 논문급. "왜 FAIL하는가"에 대한 정량적 답 + 인과적 설명 + GL(n) 보편 분류 체계.

### B-08: RH ↔ BSD 구조적 연결 ⭐⭐⭐ (가설)
- **출발**: RH (영점이 **어디**에 있는가) → ξ-다발 프레임워크 구축
- **발견**: σ-FAIL 패턴 = L-함수의 해석적 행동이 산술적 구조(degree, 계수 성장)를 인코딩
- **평행**: BSD (영점이 **몇 개**인가) = L-함수의 해석적 행동이 산술적 구조(rank)를 인코딩
- **RDL의 위치**: "영점 주변 행동이 **어떤 모양**인가" = σ-프로파일 S(σ,d)
  - RH: 영점의 위치 (Re(s)=1/2)
  - BSD: 영점의 개수 (ord_{s=1} L(E,s) = rank)
  - RDL: 영점 주변의 σ-의존 구조 (S(σ,d) = degree의 함수)
- **공통 본질**: L-함수가 산술을 인코딩하는 방식의 서로 다른 단면
- **검증**: #71~#74에서 S(σ,d)가 체계적 법칙으로 나오면 RH↔BSD를 잇는 다리 가능성
- **주의**: 현재는 가설 단계. 데이터 5점(degree 1~5) 확보 후 판정

### B-09: κ_near degree-의존성 ⭐⭐⭐ **확정** (#69+#69b, 2026-04-17)
- **현상**: GL(2) Maass κ_near=1114.9±0.5 (2 forms, n=20 TP) vs GL(3) sym² 1125.16±1.7 (6곡선, n≈60)
- **차이**: 10.3 (0.91%), GL(3) 분포의 ~6σ 밖
- **#69b 판별 결과**: R=13.78(even) κ_near=1115.40, R=9.53(odd) 1114.38 → **차이 0.09%, 재현 확정**
- **결론**: κ_near는 degree(또는 감마 인자 구조)에 의존하는 상수. "보편상수"가 아닌 "분류 불변량"
- **후속**: GL(1) ζ, GL(2) holomorphic(Δ), GL(4) Rankin-Selberg에서 κ_near(d) 추가 측정 → 함수 형태 결정
- **저γ 편향**: γ<6에서 κ_near~1111.6 (0.3% 하방). DPS 한계 또는 감마 인자 근사 오차. 미해결.

### B-10: σ-유일성 PASS/FAIL 분기 조건 ⭐⭐ → **#70에서 판별 예정** (#69b 갱신, 2026-04-17)

### B-14: Rankin-Selberg 비원시성 함정 (교훈, 2026-04-17 #72)
- **현상**: L(s, Δ⊗Δ) = L(s, sym²Δ)·ζ(s). 원시적 degree 4가 아님.
- **결과**: #72 기각. κ_near(d=4)=1125.69 ≈ κ_near(d=3)=1125.16은 degree 3 성분 지배.
- **교훈**: L(s, f⊗f) = L(s, sym²f)·L(s, ∧²f). 원시성 확인 없이 degree 카운트 금지.
- **해결**: sym³(Δ) (Kim-Shahidi, 원시적 GL(4), N=1) 사용으로 전환 (#72v2).

### B-15: degree 4 수렴 한계 (신규, 2026-04-17 #72v2)

### B-16: AFE backward Dirichlet 수렴 조건 (신규, 2026-04-17 #72v3 진단)
- **현상**: LAFE(s)의 backward term Ldir(1-s+wk)에서 Re(1-s+c_shift) > 1 필수.
- **조건**: **Re(s) < c_shift** (정확히는 Re(s) < c_shift + 1, 실용적으로 Re(s) ≤ c_shift - 1)
- **#72v3 오류**: c_shift=4에서 Re(s)=4,5를 테스트 → backward Re=1,0 → Dirichlet 발산/경계
  - Re(s)=4: rel=328 (backward Re=1)
  - Re(s)=5: rel=333,000 (backward Re=0, 발산)
  - Re(s)=3: rel=5.11 (backward Re=2, 느린 수렴)
- **수정**: Re(s)=2~2.5에서 테스트 → backward Re=3~2.5 → 충분한 수렴
- **일반화**: 모든 degree의 AFE FE 검증에 적용. degree 1~3에서는 c_shift=4, Re(s)=1/2~2로 테스트했기 때문에 문제 없었음.
- **상태**: #72v4에서 수정 예정. 통과 확률 80%.

### B-06: GL(3) κ 집중 과소측정 (#63)

### B-17: κ_near = 1/δ² 스케일링 법칙 → ★ 해결됨 (#73, 2026-04-18)
- **정리**: κ(s₀+δ) = 1/δ² + A(t₀) + O(δ²). 1/δ 항 정확히 0.
- **증명**: 함수방정식 ξ(s)=ξ(1-s) + Schwarz 반사 → H₀ 순허수, H₁ 실수.
- **A(t₀)**: Im(H₀)² + 2H₁. 영점 국소 해석적 환경에만 의존. δ 무관.
- **수치 확인**: ζ(s) 13영점 × 5δ = 65점, κ·δ² ∈ [1.00012, 1.01206].
- **의미**: 모든 degree의 κ_near를 통합. B-12 완전 해소. 논문 Proposition 기술.
