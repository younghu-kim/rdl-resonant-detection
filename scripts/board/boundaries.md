# 경계 추적 보드 (Boundary Tracker)

> 프레임워크가 "작동하는 영역"과 "깨지는 영역"의 경계 목록.
> 각 경계 = 잠재적 후속 논문 씨앗. 경계에서 멈추지 말고 파고들 것.

## 제1논문에서 발견된 경계

### B-01: σ-유일성 FAIL = GL(n) 지문 ⭐⭐⭐ 핵심 발견 (원인 규명)
- **현상**: GL(2), GL(3) 모든 사례에서 σ-유일성 FAIL
- **FAIL 패턴 비교**:
  - GL(1): σ=0.5 극대 (PASS)
  - GL(2): 평탄 (σ 무관, FAIL)
  - GL(3): σ 단조 증가 {0.3:25, 0.5:23, 0.7:25, 0.9:27} (FAIL)
  - 세 패턴 전부 다름 → **L-함수 분류기** 확인
- **⭐ 원인 규명 (#66b, 2026-04-17)**:
  - 5-Level 역보정 실험: Λ, Λ/|γ|, Λ/γ, L(s), Phase 전부 FAIL
  - **감마 인자가 원인이 아님** ("렌즈 효과" 가설 기각)
  - **순수 L(s)의 부호변환**: σ=0.3→29, σ=0.5→19, σ=0.7→1
  - **진짜 원인**: Dirichlet 급수 Σc(n)/n^s 의 수렴 특성
    - σ 낮을수록 급수 느리게 수렴 → 진동 많음 → 부호변환 증가
    - 이는 L-함수의 **산술적 구조**(계수 분포)에서 비롯
  - → "고칠" 대상이 아니라 **읽어야 할 정보**
- **파생**: Γ-인자 메커니즘 소절 (#47) + 급수 수렴 메커니즘 추가 필요

### B-02: Epstein zeta 조건부 양성 (#29)
- **현상**: 함수방정식 없는 L-함수에서 프레임워크 부분 작동
- **분류**: 새 수학 필요 (함수방정식 의존성의 정확한 역할)
- **후속 가능성**: "언제 작동하고 언제 안 되는가"의 필요충분조건

### B-03: GUE 연결 실패 (#36-38)
- **현상**: spacing ratio, 곡률-GUE 분포, κ 자기상관 모두 음성
- **분류**: 구조적 한계 또는 접근법 오류
- **후속 가능성**: 다발 기하학 ↔ 랜덤 행렬 이론의 올바른 다리는 무엇인가

### B-04: 저 t 영역 FP (#57 χ mod 7)
- **현상**: t<17에서 5개 FP (off-line zero capture)
- **분류**: 수치적 한계 (conductor 효과)
- **후속 가능성**: 고 conductor에서 저 t FP 소멸 여부 확인

### B-05: 공명 지표 가설 → 급수 수렴 지표 가설로 재해석 ⭐⭐
- **원래 가설**: R(n) = f_gamma(n) / f_zero_density, 감마 인자 진동이 원인
- **#66b에 의한 수정**: 감마 인자가 원인이 아님. L(s) 급수 자체의 수렴 특성이 원인.
- **새 가설**: GL(n)의 σ-유일성 FAIL 정도는 Dirichlet 급수 degree와 계수 분포에 의존
  - degree 1 (GL(1)): 급수 수렴 빠름 → σ 의존성 약함 → σ=0.5 특이성 유지 (PASS)
  - degree 2 (GL(2)): 급수 수렴 중간 → σ 의존성 상쇄 → 평탄 (FAIL)
  - degree 3 (GL(3)): 급수 수렴 느림 → σ 의존성 강함 → 단조 증가 (FAIL)
- **정량화 과제**: degree d L-함수의 부호변환 수 S(σ,d)의 σ-의존성 공식 유도
  - S(σ,d) ~ d·f(σ) + g(계수 분포) 형태?
- **검증**: Maass form (degree 2, 비정칙) + Rankin-Selberg (degree 4) 데이터 필요
- **가치**: 여전히 별도 논문급. "왜 FAIL하는가"에 대한 정량적 답.

## 제2논문 경계 (진행 중)

### B-07: GL(3) Zero-Finding 확립 (#67) ⭐⭐⭐
- **현상**: ξ-다발 κ-피크 방법이 LMFDB 미수록 영점 21개를 독립 발견
- **검증**: AFE 직접계산 21/21 통과 (median |Λ|=2.5e-27)
- **삼중 확인**: 부호변환(#63) + κ-피크(#65) + AFE(#67)
- **의미**: 프레임워크가 "zero-finding machine"으로 작동 — 알려진 영점 재발견이 아닌 새 영점 발견
- **후속**: 더 높은 conductor, 다른 GL(3) L-함수에서 재현 여부

### B-06: GL(3) κ 집중 과소측정 (#63)
- **현상**: κ near/far = 5.3× (임계 10× 미달). far 샘플이 3점뿐.
- **원인**: 63 영점, 평균 간격 0.76. t_far = t_zero ± 2*spacing에서 샘플 부족.
- **해결**: 중간점 샘플링 (영점 사이 중점)으로 far 통계 보강. 별도 스크립트 불필요, v2 수정으로 가능.
- **우선순위**: 낮음 (기법 문제, 본질적 한계 아님)
